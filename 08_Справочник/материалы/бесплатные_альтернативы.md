# Бесплатные альтернативы Cursor (2026)

> Cursor платный — но есть способы кодить с AI бесплатно.
> Источник: @misha_davai_po_novoi

---

## Полностью бесплатные

### Gemini CLI (Google)
- **1000 запросов к Gemini Pro в день** — бесплатно
- CLI в терминале
- Сайт: github.com/google-gemini/gemini-cli

### Cline
- Работает с **любыми** моделями: Claude, GPT, Gemini, DeepSeek
- Бесплатная модель **MiniMax 2.5** — 2000 запросов/день
- По мозгам близка к Claude Opus
- Сайт: cline.bot

### DeepSite (Hugging Face)
- Генерация сайтов прямо **в браузере**
- Китайские модели (DeepSeek, Qwen, Kimi)
- До 3 проектов бесплатно
- Сайт: huggingface.co/deepsite

### Google AI Studio
- **~50 запросов Gemini Pro/день**, безлимит Flash
- Аналог Cursor в браузере
- Сайт: aistudio.google.com

### Antigravity
- Полный аналог Cursor, но на **Google Gemini**
- Форк VS Code — привычный интерфейс
- Бесплатный тариф

### Qwen Coder
- Китайская модель для кодинга
- **2000 бесплатных запросов/день**
- Сайт: qwenlm.github.io/blog/qwen3-coder/

## Дешёвые альтернативы

### GitHub Copilot — $10/мес
- 300 premium запросов
- GPT-4.1 и GPT-4o **бесплатно**
- Встроен в VS Code

### Kilo Code — от $5
- $5 бесплатно при регистрации
- +$20 при пополнении $10
- OpenRouter под капотом
- Сайт: kilocode.ai

### GenSpark — $25/мес
- **Безлимит** GPT-5.2, Claude Opus 4.5 и другие
- Генерация картинок в 4K
- Сайт: genspark.ai
