# Модуль 08. Справочник — нейронки на все случаи жизни

Справочный модуль: AI-инструменты по категориям, горячие клавиши Cursor, бесплатные альтернативы, рынок труда и AI. Открывай когда нужно найти инструмент под задачу.

## Что внутри

- **презентация.html** — визуальный справочник по категориям (24 слайда)
- **материалы/** — подробные справочники и гайды

## Материалы

- `полный_список.md` — все AI-инструменты с ценами и ссылками
- `как_выбрать.md` — алгоритм: задача → инструмент
- `cursor_commands.md` — команды Cursor (/pr, /fix-issue, /review, /docs)
- `бесплатные_альтернативы.md` — Cline, Gemini CLI, DeepSite, AI Studio
- `рынок_труда_и_ai.md` — как AI меняет рынок и что с этим делать

## Как пользоваться

Это не учебный модуль — это справочник. Не нужно проходить его целиком. Открывай когда нужно найти ответ на конкретный вопрос.
