# Список AI-инструментов для масштабирования

## 🖥️ Кодинг и разработка

### Claude Code
- **Что:** AI-агент в терминале. Говоришь текстом (или голосом через Wispr) «сделай сайт-визитку» — он делает. Без Cursor, без интерфейса.
- **Сайт:** claude.ai/code
- **Цена:** Включено в Claude Max ($20/мес) или по API
- **Для кого:** Все, кто хочет создавать проекты одной командой

### Cline
- **Что:** Бесплатное расширение для VS Code. Работает с любой моделью — Claude, GPT, бесплатные. Для тех, кто не хочет платить за Cursor Pro.
- **Сайт:** github.com/cline/cline
- **Цена:** Бесплатно (+ стоимость API выбранной модели, можно $0 с бесплатными)
- **Для кого:** Все, кто хочет экономить или экспериментировать с разными моделями

### Conductor
- **Что:** Открываешь 5 окон, в каждом свой Claude Code агент. Один делает сайт, второй пишет тексты, третий ищет конкурентов — всё параллельно.
- **Сайт:** github.com (ищите conductor claude code)
- **Цена:** Open source (бесплатно)
- **Для кого:** Те, кто хочет делать 5 задач одновременно

## 🎤 Голос и ввод

### Wispr Flow
- **Что:** Голосовой ввод текста в любом приложении
- **Сайт:** wispr.ai
- **Цена:** Бесплатно
- **Для кого:** Все, кто много печатает

## 🔍 Ресерч и аналитика

### GenSpark
- **Что:** AI-поисковик с глубоким анализом
- **Сайт:** genspark.ai
- **Цена:** Бесплатно
- **Для кого:** Маркетологи, аналитики, предприниматели

### Parallel.ai
- **Что:** Параллельный запрос к нескольким AI-моделям
- **Сайт:** parallel.ai
- **Цена:** Бесплатно
- **Для кого:** Все, кто хочет сравнить ответы разных моделей

## 📚 Работа с документами

### NotebookLM
- **Что:** AI-ассистент по загруженным документам (Google)
- **Сайт:** notebooklm.google.com
- **Цена:** Бесплатно
- **Для кого:** Все, кто работает с большим объёмом информации

## 🎨 Дизайн и интерфейсы

### Weavy
- **Что:** Визуальный AI-редактор для создания UI
- **Для кого:** Дизайнеры, продакт-менеджеры, предприниматели

### OpenCreator
- **Что:** Генерация дизайн-компонентов и страниц по описанию
- **Для кого:** Все, кто хочет создавать без знания кода
