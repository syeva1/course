# Дорожная карта: от Cursor до AI-бизнеса

## Этап 1: Cursor (ты уже здесь ✅)

Ты умеешь:
- [ ] Создавать проекты в Cursor с AI
- [ ] Писать промпты, которые дают результат
- [ ] Работать с API и подключать нейросети
- [ ] Создавать Telegram-ботов

**Если не отметил все — вернись к модулям 01–09.**

## Этап 2: Агенты (неделя 1–2)

- [ ] Создай агента-копирайтера для своего бизнеса (модуль 07)
- [ ] Создай агента-ассистента для ответов клиентам
- [ ] Настрой Cursor Rules — пусть Cursor знает твой бизнес
- [ ] Протестируй: дай агенту 10 реальных задач, оцени результат
- [ ] Заведи lessons.md — записывай ошибки агента

**Результат:** 2–3 работающих агента, которые экономят тебе 1–2 часа в день.

## Этап 3: Новые инструменты (месяц 1)

- [ ] Установи Claude Code или Cline
- [ ] Сделай первый проект без Cursor — только через терминал/Cline
- [ ] Попробуй Wispr Flow — диктуй задачи голосом
- [ ] Загрузи документы в NotebookLM — получи AI-эксперта по ним
- [ ] Проведи ресерч в GenSpark — сравни с Google

**Результат:** 5+ инструментов в арсенале вместо одного Cursor.

## Этап 4: Параллельная работа (месяц 2)

- [ ] Попробуй Conductor — запусти 3 агента одновременно
- [ ] Автоматизируй 3 бизнес-процесса (контент, поддержка, аналитика)
- [ ] Измерь: сколько часов экономишь в неделю?
- [ ] Свяжи ботов + агентов в систему

**Результат:** автоматизировано 50%+ рутины. Экономия 10+ часов в неделю.

## Этап 5: AI-бизнес (месяц 3)

Два пути:

### Путь А: Продавай навык
- [ ] Предложи 3 знакомым предпринимателям настройку AI
- [ ] Сделай кейс: «было → стало → сколько сэкономил»
- [ ] Оформи как услугу: «Настройка AI-системы для бизнеса»
- [ ] Цена: от 30 000 ₽ за настройку

### Путь Б: Масштабируй свой бизнес
- [ ] Автоматизируй 80% рутины через агентов
- [ ] Оставь себе только стратегию и клиентов
- [ ] Один + AI = работа команды из 10 человек
