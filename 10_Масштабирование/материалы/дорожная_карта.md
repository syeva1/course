# Дорожная карта: развитие AI-оркестратора

## Месяц 1: Расширение инструментов

### Неделя 1–2: Терминальные агенты
- [ ] Установить Claude Code или Cline
- [ ] Выполнить 5 реальных задач через агента
- [ ] Сравнить с Cursor: что быстрее, что удобнее

### Неделя 3–4: Голос и ресерч
- [ ] Попробовать Wispr Flow для голосового ввода
- [ ] Провести 3 ресерча через GenSpark
- [ ] Загрузить документы в NotebookLM

## Месяц 2: Автоматизация процессов

### Неделя 5–6: Бизнес-процессы
- [ ] Выбрать 3 рутинных процесса для автоматизации
- [ ] Создать AI-ассистентов для каждого
- [ ] Измерить экономию времени

### Неделя 7–8: Масштабирование
- [ ] Попробовать Conductor (параллельные агенты)
- [ ] Создать систему из 2–3 связанных ботов/агентов
- [ ] Документировать результаты

## Месяц 3: AI-оркестратор

### Неделя 9–10: Интеграция
- [ ] Связать все инструменты в единый workflow
- [ ] Автоматизировать 80% рутинных задач
- [ ] Создать дашборд/отчётность

### Неделя 11–12: Экспертиза
- [ ] Написать кейс о своём опыте
- [ ] Помочь 2–3 коллегам освоить AI-инструменты
- [ ] Определить следующие цели на 6 месяцев

## Принципы

1. **Один инструмент в неделю** — не пытайся освоить всё сразу
2. **Реальные задачи** — учись на своих проектах, не на туториалах
3. **Измеряй** — без цифр не поймёшь, работает ли
4. **Делись** — обучая других, учишься сам
