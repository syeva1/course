Claude Code — AI в терминале

Терминальный агент от Anthropic
  • Работает прямо в командной строке — без GUI
  • Читает, пишет и запускает код автономно
  • Понимает весь проект целиком через контекст репозитория
  • Может выполнять цепочки задач без вмешательства
«С Claude Code я делаю 10–30 pull-request'ов в день. Один человек заменяет целый отдел разработки.»
— Борис Черный, экс-Meta, автор Programming TypeScript
АССИСТ+

Cline — бесплатная альтернатива

Открытое расширение для VS Code
  • Поддерживает все модели: Claude, GPT, Gemini, локальные LLM
  • Работает как агент — читает файлы, пишет код, запускает команды
  • Можно использовать со своим API-ключом — платишь только за токены
  • Идеально для тех, кто хочет экономить и экспериментировать

БЕСПЛАТНО Cline + бесплатные модели = $0 на старте

АССИСТ+

Conductor — параллельные Claude Code

Несколько агентов работают одновременно
  • Запуск нескольких Claude Code сессий параллельно
  • Каждый агент работает над своей задачей
  • Ты — дирижёр, агенты — оркестр
  • Подход Бориса Черного: один человек управляет десятком агентов

МОЩЬ Conductor позволяет масштабировать одного человека в команду

АССИСТ+

Wispr Flow — голосовой ввод

Диктуй текст вместо набора
  • Распознаёт речь и превращает в текст в любом приложении
  • Работает на macOS и Windows
  • Ускоряет написание промптов, писем, документов в 3–5 раз
  • Поддерживает русский и английский языки

СКОРОСТЬ Говоришь — Wispr печатает. Без рук.

GenSpark, Parallel.ai — глубокий ресерч

Исследования, на которые раньше уходили дни
  • GenSpark — AI-поисковик, который собирает и анализирует информацию из десятков источников
  • Parallel.ai — запускает несколько AI-моделей параллельно для сравнения ответов
  • Идеально для анализа рынка, конкурентов, трендов
  • Один запрос = отчёт, на который раньше нужен был аналитик
