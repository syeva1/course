# Инструкция к Модулю 10: Масштабирование

## Зачем мне это?

Ты прошёл 9 модулей. Умеешь Cursor, промпты, API, агенты, ботов. Но Cursor — это один инструмент. А мир AI-инструментов огромен. Этот модуль — карта территории: что ещё существует, зачем нужно и как использовать.

**Цель:** после модуля ты знаешь 5+ инструментов и понимаешь свой путь на следующие 3 месяца.

---

## Как проходить модуль

### 1. Презентация (20 мин)

Откройте `презентация.html` в браузере. 16 слайдов — каждый с конкретным инструментом и примером.

### 2. Попробуйте 3 инструмента (30 мин)

Для каждого — 10 минут:

**Claude Code или Cline:**
- Claude Code: `npm install -g @anthropic-ai/claude-code` → `claude` в папке проекта → «создай hello.py»
- Cline: VS Code → Extensions → Cline → вставь API-ключ → дай задачу

Представь: ты говоришь «сделай мне сайт-визитку» — и он делает. Без Cursor, без интерфейса. А если подключить Wispr Flow — можно диктовать голосом.

**NotebookLM:**
- Зайди на notebooklm.google.com
- Загрузи 2–3 своих документа
- Задай вопрос — получи ответ с цитатами

**GenSpark:**
- genspark.ai → «Тренды в [твоя ниша] на 2026 год»
- Сравни с Google — разница заметна сразу

### 3. Составьте дорожную карту (20 мин)

Откройте `материалы/дорожная_карта.md`. Это ваш план:

**Сейчас → агенты → Claude Code → параллельные агенты → бизнес на AI**

Заполните чеклисты, поставьте даты.

### 4. Практика (30 мин)

Откройте `практика.md` — 4 задания + бонус.

## Ключевые инструменты — что и зачем

| Инструмент | Простыми словами | Цена |
|---|---|---|
| **Claude Code** | Говоришь текстом «сделай сайт» — он делает. Без Cursor. | $20/мес |
| **Cline** | Как Cursor, но бесплатно и с любой моделью | $0 + API |
| **Conductor** | 5 окон — 5 агентов — 5 задач одновременно | Бесплатно |
| **Wispr Flow** | Диктуешь — печатает. Голосовой ввод в 5 раз быстрее | Бесплатно |
| **NotebookLM** | AI-эксперт по твоим документам | Бесплатно |
| **GenSpark** | AI-аналитик: ресерч за 2 минуты вместо 2 дней | Бесплатно |
