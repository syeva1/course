# Модуль 05 — Автоматизация и AI-агенты

## Чему научишься
- Автоматизировать рутинные задачи через Cursor
- Выбирать модели и не привязываться к одной
- Создавать AI-агентов для своего бизнеса
- Настраивать Cursor Rules (.mdc файлы)
- Использовать скиллы вместо одноразовых промптов
- Улучшать агентов через lessons.md

## Файлы модуля
- `презентация.html` — 17 слайдов
- `инструкция.md` — пошаговое руководство
- `практика.md` — 3 задания
- `шпаргалка.md` — краткая шпаргалка
- `материалы/выбор_модели.md` — таблица моделей 2026
- `материалы/идеи_автоматизации.md` — примеры задач

## Ключевые идеи
1. Не влюбляйся в модели — они обновляются каждый месяц
2. Проверяй модели на lmarena.ai
3. Агент = Роль + Контекст + Задачи + Правила
4. Скиллы > промпты (переиспользуемые компоненты)
5. Воркфлоу — где предсказуемо, агент — где нет
