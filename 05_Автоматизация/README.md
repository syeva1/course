# Модуль 05 — Личная автоматизация

AI для жизни: автоматизируй рутину, освободи время для важного.

## Содержание

| Файл | Описание |
|------|----------|
| [презентация.html](презентация.html) | Слайды модуля (16 слайдов) |
| [инструкция.md](инструкция.md) | 10 шагов к первой автоматизации |
| [практика.md](практика.md) | 3 задания + чеклист |
| [шпаргалка.md](шпаргалка.md) | 20 идей что автоматизировать |
| [материалы/идеи_автоматизации.md](материалы/идеи_автоматизации.md) | 30 конкретных идей по категориям |

## Ключевые темы

- Зачем автоматизировать личные задачи
- Claude для таблиц и Excel
- Вайбкодинг бытовых скриптов
- AI для обучения, текстов и планирования
- Автоматические отчёты и сводки
- Проекты мечты с нейронками

## Как использовать

1. Откройте `презентация.html` в браузере
2. Изучите слайды (стрелки ← → или кнопки)
3. Выполните задания из `практика.md`
4. Используйте `шпаргалка.md` как ежедневный справочник
