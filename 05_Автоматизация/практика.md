# Практика — Модуль 05

## Задание 1. Автоматизация текста

Выберите текст, который вы пишете регулярно (ответ клиенту, пост, письмо). Создайте промпт в Claude, который генерирует этот текст по вводным данным.

**Что сдать:** скриншот промпта и результата.

## Задание 2. Таблица с формулами

Возьмите любую свою таблицу (расходы, задачи, клиенты). Попросите Claude:
- Предложить формулы для автоматизации подсчётов
- Написать сводку по данным
- Найти закономерности

**Что сдать:** скриншот таблицы до и после + промпт.

## Задание 3. План на неделю

Попросите Claude составить ваш план на неделю:
1. Опишите свои задачи и приоритеты
2. Укажите ограничения (время, энергия)
3. Получите структурированный план с декомпозицией

**Что сдать:** итоговый план.
