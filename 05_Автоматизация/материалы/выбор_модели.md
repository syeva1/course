# 🤖 Какую модель выбрать в Cursor

> Модель можно переключить прямо в чате агента — выпадающее меню сверху.
> Или в настройках: Cursor Settings → Models

---

## Категории задач

### 🎨 Креатив и тексты
**Задача:** Написать пост, письмо, описание, сценарий.
**Лучшая модель:** Claude Sonnet
- Самый «человечный» слог, без клише
- Понимает тон, эмпатию, русский язык
- Хороший для SMM, копирайтинга, email

### 💻 Код и техника
**Задача:** Написать скрипт, HTML, исправить баг, автоматизация.
**Лучшая модель:** Claude Sonnet или GPT-4o
- Claude Sonnet — эталон для Cursor
- GPT-4o — стабильная классика, хорош для простых задач

### 🧠 Анализ и логика
**Задача:** Отчёт, стратегия, разбор данных, поиск ошибок.
**Лучшая модель:** Gemini Pro или GPT-4o
- Gemini Pro — огромный контекст (до 1М токенов), отличная структура
- GPT-4o — надёжный для аналитики

### ⚡ Массовые задачи
**Задача:** Обработать 100 файлов, перевести, черновики.
**Лучшая модель:** Auto (Cursor сам выберет) или Gemini Flash
- Скорость важнее качества
- Дешёвые модели справляются не хуже

---

## Метод «Тройной прогон»

Если задача важная — не доверяй одной модели:

1. Запусти в **Claude Sonnet** — получи креативный вариант
2. Запусти в **GPT-4o** — получи структурный вариант
3. Сравни и возьми лучшее, или попроси третью модель объединить

---

## Практические советы

| Ситуация | Что делать |
|----------|-----------|
| Не знаю, какую модель выбрать | Оставь **Auto** |
| Нужен красивый текст | Переключи на **Claude Sonnet** |
| Модель тупит, не понимает | Попробуй другую модель |
| Задача огромная (много файлов) | Используй **Agent** режим |
| Важное решение | Тройной прогон (3 модели) |

---

## Как переключить модель в Cursor

1. Открой чат агента (Toggle Agents Side Bar)
2. Сверху чата — выпадающее меню с названием модели
3. Кликни → выбери нужную
4. Или: Cursor Settings → Models → выбери модель по умолчанию
