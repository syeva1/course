## Метод «Тройной прогон»

Для важных задач запусти одну и ту же задачу на 3 разных моделях, сравни результаты и возьми лучший. Это дешевле, чем потом переделывать.

## Как переключить модель в Cursor

1. Открой AI Chat (`Cmd+L`)
2. Внизу окна чата — выпадающий список моделей
3. Выбери нужную
4. Для Agent Mode: та же модель используется автоматически

**Совет:** Держи Claude Opus для сложного, Composer 1.5 для рутины.

## Стратегия для тарифа Pro ($20)

На Pro 500 быстрых запросов/месяц. Распределяй так:
- **70%** — Composer 1.5 или Auto (рутина)
- **20%** — Claude Sonnet (средние задачи)
- **10%** — Claude Opus (архитектура, сложный рефакторинг)
