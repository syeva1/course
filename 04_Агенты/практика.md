# Практика — Автоматизация и AI-агенты

## Задание 1. Автоматизация рутинной задачи

**Цель:** увидеть разницу «вручную vs с AI».

1. Выбери задачу, которую делаешь регулярно (отчёт, обработка файлов, конвертация данных)
2. Засеки время выполнения вручную
3. Попроси Cursor автоматизировать: перетащи файл в чат и опиши задачу
4. Засеки время с AI

**Результат:** сравнение времени + готовый скрипт/шаблон для повторного использования.

---

## Задание 2. Создай AI-агента для своего бизнеса

**Цель:** настроить Cursor как сотрудника.

1. Создай папку `.cursor/rules/` в проекте
2. Создай файл `assistant.mdc` по шаблону:

```
---
description: "Описание агента"
globs: ["**/*"]
---

## Роль
Кто этот агент (копирайтер, ассистент, аналитик)

## Контекст
Что он знает о бизнесе (ЦА, услуги, тон)

## Задачи
Что он делает (пишет тексты, анализирует данные)

## Правила
Чего НЕ делает, какой формат ответа
```

3. Протестируй на 5 реальных задачах
4. Запиши ошибки в `lessons.md`
5. Обнови агента с учётом ошибок

**Результат:** работающий AI-агент + файл lessons.md с улучшениями.

---

## Задание 3. Тройной прогон

1. Возьми важную задачу (текст, анализ, код)
2. Выполни её в Claude
3. Переключи модель (внизу чата) — выполни в GPT
4. Сравни результаты
5. Возьми лучшее из обоих

**Результат:** понимание разницы между моделями на реальной задаче.
