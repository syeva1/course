# Модуль 04. Агенты и скиллы — AI-сотрудники для бизнеса

Создаём AI-агентов через .mdc файлы — виртуальных сотрудников со своей ролью, знаниями и правилами. Учимся выбирать модели и не привязываться к одной.

## Чему научишься

- Создавать .mdc файлы — правила для AI-агентов
- Понимать разницу между промптами и скиллами
- Выбирать модели под задачу (Claude, GPT, Gemini)
- Использовать Тройной прогон для важных задач
- Улучшать агентов через lessons.md

## Файлы модуля

- **презентация.html** — слайды (открой в браузере)
- **инструкция.md** — пошаговый гайд
- **практика.md** — 5 заданий от простого к сложному
- **материалы/** — готовые агенты, скиллы, шаблоны

## Порядок прохождения

1. Посмотри презентацию
2. Пройди инструкцию
3. Выполни все 5 заданий из практики

## Время

~2 часа на первое прохождение.

## Материалы

- `88_готовых_агентов.md` — каталог агентов по категориям
- `33_маркетинговых_скилла.md` — скиллы для маркетинга
- `готовые_mdc/` — готовые .mdc файлы (скопируй в проект)
- `gpt5_промптинг_cursor.md` — настройка поведения моделей
- `hooks_автопроверки.md` — автоматические проверки после агента
- `кейс_3_агента_claude_code.md` — реальный кейс мульти-агентной работы
- `65_скиллов_продакт_менеджера.md` — пример системы скиллов
