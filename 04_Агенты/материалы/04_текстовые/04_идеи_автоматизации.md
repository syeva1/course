# 30 идей автоматизации по категориям

## 📋 Работа и продуктивность

1. **Утренний брифинг** — промпт, который собирает задачи дня из заметок и календаря
2. **Декомпозиция проекта** — разбивка большой цели на шаги с дедлайнами
3. **Шаблоны писем** — 5-7 типовых ответов с переменными (имя, дата, сумма)
4. **Протоколы встреч** — вставить заметки → получить структурированный протокол
5. **Еженедельная ретроспектива** — что сделано, что нет, что на следующую неделю

## 📊 Таблицы и данные

6. **Формулы Excel** — описываете задачу словами, Claude пишет формулу
7. **Сводные таблицы** — автоматический анализ данных с выводами
8. **Очистка данных** — удаление дублей, форматирование, стандартизация
9. **Дашборд расходов** — шаблон таблицы с автоподсчётом по категориям
10. **Прогноз на месяц** — AI анализирует тренды в ваших данных

## ✍️ Тексты и контент

11. **Посты для Telegram** — генерация по теме + тон бренда
12. **Описания для резюме** — переформулировка опыта под вакансию
13. **Коммерческие предложения** — шаблон + адаптация под клиента
14. **Рерайт и редактура** — улучшение текста с сохранением смысла
15. **FAQ для клиентов** — генерация из частых вопросов

## 📚 Обучение и развитие

16. **Конспект из книги** — выжимка ключевых идей и действий
17. **Карточки для запоминания** — AI создаёт flashcards по теме
18. **Объяснение «как для пятилетнего»** — сложные концепции простым языком
19. **План обучения** — roadmap по новому навыку с ресурсами
20. **Разбор ошибок** — вставить задачу + свой ответ → получить анализ

## 🏠 Быт и личное

21. **Меню на неделю** — с учётом бюджета и предпочтений
22. **Список покупок** — автоматически из меню
23. **Сравнение товаров** — таблица плюсов/минусов по критериям
24. **Планирование поездки** — маршрут, бюджет, чеклист
25. **Тренировка** — программа под ваш уровень и цели

## 💻 Скрипты и техническое

26. **Ресайз фотографий** — скрипт для пакетной обработки
27. **Переименование файлов** — по шаблону (дата, категория)
28. **Бэкап заметок** — автокопирование важных файлов
29. **Форма для клиентов** — простая HTML-форма для сбора данных
30. **Парсинг данных** — сбор информации с сайтов в таблицу
