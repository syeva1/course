# Cursor Tutorial 0 — Why Leave GitHub Copilot Behind?
Source: https://cursor101.com/tutorial/learn-cursor-0-why-use-cursor

Cursor's main advantages over Copilot:

## 1. Code Modification Capabilities
- Copilot only inserts/appends code
- Cursor can modify existing code: multi-line editing, inline editing (Cmd K), intelligent prediction, Composer (multi-file)

## 2. Contextual Reference (@)
- Type @ in any input box → suggestion list of referenceable context
- @Codebase for global code search (local index, not remote)
- Copilot uses separate @ and # symbols, more complex, less intuitive
