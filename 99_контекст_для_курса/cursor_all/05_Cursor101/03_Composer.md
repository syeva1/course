# Cursor Tutorial — Double Your Speed with Composer
Source: https://cursor101.com/tutorial/learn-cursor-composer

Cmd+I to open. Excels at multi-file editing.

## Common Issues
- Errors increase with number of files modified
- Unintended changes to unrelated files

## Best Practices
1. Break tasks into 2-4 files per subtask
2. Use Composer snapshots (auto-created before each conversation)
3. One Composer session per feature
4. Test each subtask → if fails, restore snapshot

## Workflow
Plan → Break down → New session → Feed subtasks → Test → Restore if needed → Repeat
