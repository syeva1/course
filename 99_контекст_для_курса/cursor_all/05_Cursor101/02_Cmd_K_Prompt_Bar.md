# Cursor Tutorial 2 — Prompt Bar (Cmd K)
Source: https://cursor101.com/tutorial/learn-cursor-cmdk

## Two types of prompt bars:
1. Code editing (Cmd K in editor) — generate/edit code
2. Terminal (Cmd K in terminal) — generate shell commands

## Code Editing Prompt Bar
- No selection → generates new code (Inline Generation)
- With selection → modifies selected code (Inline Editing)
- Context: related files + recently viewed files

## Tips
- Follow-up instructions to refine results
- Option/Alt Enter for Quick Answer (explains code without generating)
- Generate git commit messages via terminal prompt bar
