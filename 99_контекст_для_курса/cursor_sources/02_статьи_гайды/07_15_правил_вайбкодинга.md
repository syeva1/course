# 15 правил для быстрого запуска проектов (вайб-кодинг)

Источник: канал @misha_davai_po_novoi

1. **Не начинайте с нуля** — используйте шаблоны с GitHub
2. **Доверьте рутину AI-агенту** — Cursor agent mode
3. **Ищите идеи через AI** — Perplexity и другие
4. **Одна задача — один чат**
5. **Тестируйте чаще, исправляйте быстрее**
6. **Не стремитесь к идеалу с первой попытки**
7. **Говорите, а не печатайте** — голосовой ввод
8. **Вдохновляйтесь чужим кодом на GitHub**
9. **Копируйте ошибки в чат с AI**
10. **Сохраняйте историю чатов**
11. **Держите ключи и пароли в .env**
12. **Регулярно делайте git commit**
13. **Разворачивайте проект как можно раньше** (Vercel)
14. **Коллекционируйте удачные промпты**
15. **Получайте удовольствие от процесса**
