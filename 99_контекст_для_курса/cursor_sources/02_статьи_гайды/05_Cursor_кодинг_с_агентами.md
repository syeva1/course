## Ключевые тезисы:

1. **Начинайте с плана** (Shift+Tab → Plan Mode)
2. **Управляйте контекстом** — не перегружайте агента файлами
3. **Новый разговор при смене задачи** — длинные чаты теряют фокус
4. **Rules** (.cursor/rules/) — статические правила для проекта
5. **Skills** (SKILL.md) — динамические возможности агента
6. **TDD-подход** — сначала тесты, потом код
7. **Параллельные агенты** — несколько моделей на одну задачу
8. **Облачные агенты** — делегируй и проверяй позже
9. **Debug Mode** — для сложных багов (гипотезы → инструментирование → анализ)
10. **Конкретные промпты** побеждают абстрактные
