## Рекомендации Миши

- **Простые задачи / новичок** → Cursor
- **Приятный интерфейс** → Codex
- **Бесплатно** → Antigravity
- **Серьёзные проекты** → Claude Code (расширение в Cursor/VS Code или терминал)

## Для курса АССИСТ+

Наша аудитория (бизнес-ассистенты, новички) → Cursor — правильный выбор. Самый низкий порог входа (3/5), визуальный интерфейс, Composer для мульти-файлов.
