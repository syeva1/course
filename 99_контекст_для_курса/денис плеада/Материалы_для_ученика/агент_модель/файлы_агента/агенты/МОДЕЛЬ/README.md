## ЧТО ЭТО

**Ресерчер моделей AI** — это AI-агент, который помогает выбрать оптимальную AI-модель для любой задачи на основе актуальных рейтингов, бенчмарков и цен.

**Главная цель:** Дать быстрый и точный ответ (< 1 сек для 95% задач) с рекомендацией модели.


## ЧТО УМЕЕТ

### 1. Быстрая оценка задачи
- Сложность (простая/средняя/сложная/очень сложная)
- Тип (креатив/код/логика/массовка/WebDev/Vision/Search/Enterprise)
- Контекст (объём данных в токенах)
- Многоуровневость (количество этапов)

### 2. Рекомендация оптимальной модели
- Основная рекомендация с обоснованием
- Альтернативные варианты (бюджетный, премиум)
- Сравнение по критериям (цена, скорость, качество)
- Оценка ROI (цена vs качество)

### 3. Активация модели в Cursor
- Попытка автоматической активации (если API доступно)
- Инструкция для ручной активации
- Проверка доступности модели

### 4. Обновление актуальных данных
- Из 9 источников рейтингов (LMSYS, OpenRouter, Digital Applied, и др.)
- Автоматическое или по команде
- С валидацией и версионированием

## ИНТЕГРАЦИЯ С ПЛАНАМИ

### Автоматический вызов:

При создании плана через `@создай_сотрудника`:
1. План формируется
2. → Автоматически вызывается `@модель`
3. Оценивает задачи из плана
4. Рекомендует модель
5. Добавляет секцию "РЕКОМЕНДУЕМАЯ МОДЕЛЬ" в план
