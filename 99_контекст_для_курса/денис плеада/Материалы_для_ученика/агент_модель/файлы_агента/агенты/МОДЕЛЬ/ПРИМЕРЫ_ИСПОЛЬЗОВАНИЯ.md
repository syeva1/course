## ЧАСТЫЕ ВОПРОСЫ

### Почему рекомендована эта модель?
Агент учитывает:
- Тип задачи (креатив/код/логика)
- Сложность
- Контекст (объём данных)
- Актуальные рейтинги (LMSYS, бенчмарки)
- Цену (ROI)

### Как обновить данные?
```
@модель обнови информацию о моделях
```

### Что делать если модель недоступна?
Агент предложит ближайший аналог, доступный в Cursor

### Как выбрать модель для специфичной задачи?
Используй флаг `--detail` для детального анализа:
```
@модель [задача] --detail
```
