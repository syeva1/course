# МЕТОДОЛОГИЯ ОЦЕНКИ ЗАДАЧ

**Назначение:** Критерии для быстрой и точной оценки задачи перед рекомендацией модели

---

## КРИТЕРИИ ОЦЕНКИ

### 1. Сложность задачи

**Простая (Simple):**
- Короткие запросы (< 50 слов)
- Однозначная задача без подзадач
- Шаблонный результат
- Примеры: "Напиши короткий пост", "Исправь опечатку", "Переведи фразу"
- **Рекомендация:** Flash модели (Gemini Flash, DeepSeek)

**Средняя (Medium):**
- Стандартные задачи с понятной структурой
- 1-2 подзадачи
- Требует контекста, но не глубокого анализа
- Примеры: "Напиши статью про AI", "Создай скрипт для парсинга", "Проанализируй отчёт"
- **Рекомендация:** Балансные модели (Gemini Pro, GPT-4o)

**Сложная (Complex):**
- Многоэтапные задачи
- Требует рассуждений и логики
- 3-5 подзадач
- Примеры: "Создай план автоматизации", "Отладь сложный баг", "Стратегия продвижения"
- **Рекомендация:** Топ модели (Claude Sonnet, Gemini Pro)

**Очень сложная (Very Complex):**
- Комплексные проекты
- Стратегические задачи
- Множество взаимосвязанных этапов
- Примеры: "Построй архитектуру системы", "Глубокий анализ с инсайтами", "Комплексная интеграция"
- **Рекомендация:** Премиум модели (Claude Opus, Gemini 3 Pro)

---

### 2. Контекст (объём данных)

**Малый (< 10K токенов):**
- Короткие тексты, небольшие файлы
- Стандартные задачи
- **Рекомендация:** Любая модель подходит

**Средний (10K-100K токенов):**
- Несколько файлов, средние тексты
- Типичный размер для большинства задач
- **Рекомендация:** Модели с хорошим контекстом (Gemini, Claude)

**Большой (100K-500K токенов):**
- Много файлов, большие документы
- Требуется модель с большим контекстным окном
- **Рекомендация:** Gemini 2.5 Pro (1M токенов), Claude Sonnet (200K)

**Очень большой (> 500K токенов):**
- Огромные объёмы данных
- Критично большое контекстное окно
- **Рекомендация:** ТОЛЬКО Gemini 2.5/3 Pro (1M токенов)

---

### 3. Многоуровневость

**Одноуровневая:**
- Одна задача → один результат
- Нет зависимостей
- Примеры: "Напиши пост", "Создай функцию"
- **Рекомендация:** Простые модели

**Двухуровневая:**
- Задача с подзадачами
- Последовательное выполнение
- Примеры: "Проанализируй данные и создай отчёт", "Напиши код и протестируй"
- **Рекомендация:** Средние модели

**Многоуровневая:**
- Комплексный проект
- Множество взаимосвязанных этапов
- Примеры: "Создай систему автоматизации", "Построй архитектуру решения"
- **Рекомендация:** Топ модели с логикой

---

### 4. Тип задачи (критично для выбора!)

**Креатив (Creative):**
- Посты, тексты, контент, реклама
- Требуется: хороший слог, эмпатия, человечность
- **Рекомендация:** Claude 3.5 Sonnet (лучший слог), Gemini 2.5 Flash (дешевле)
- **Ключевые слова:** "пост", "текст", "контент", "написать", "статья", "пись мо"

**Код (Code):**
- Скрипты, функции, программирование
- Требуется: точность синтаксиса, понимание языка
- **Рекомендация:** Claude 3.5 Sonnet (SWE-bench 80.9%), Qwen 2.5 Coder (дешевле)
- **Ключевые слова:** "код", "скрипт", "функция", "программирование", "баг", "refactor"

**Логика (Logic/Reasoning):**
- Анализ, стратегия, рассуждения, проблемы
- Требуется: высокий IQ, логика
- **Рекомендация:** DeepSeek V3.2 (цена/качество), Gemini 2.5 Pro (контекст)
- **Ключевые слова:** "анализ", "стратегия", "рассуждение", "проблема", "решить", "найти ошибку"

**Массовка (Bulk Processing):**
- Обработка больших объёмов данных
- Требуется: скорость, дешевизна
- **Рекомендация:** Gemini 2.5 Flash Lite ($0.10/1M), DeepSeek V3.2 ($0.24/1M)
- **Ключевые слова:** "обработать", "перевести", "summary", "много файлов"

**Документация (Documentation):**
- README, инструкции, описания
- Требуется: структура, ясность
- **Рекомендация:** Gemini 2.5 Flash (структура), Claude Sonnet (качество)
- **Ключевые слова:** "документация", "README", "инструкция", "описание"

**WebDev (Веб-разработка):**
- Создание веб-приложений, HTML, CSS, JavaScript
- Требуется: понимание веб-технологий
- **Рекомендация:** Claude Opus 4.5 Thinking (WebDev ELO 1520)
- **Ключевые слова:** "веб", "сайт", "HTML", "CSS", "JavaScript", "лендинг"
- **Источник:** LMSYS WebDev категория

**Vision (Мультимодальность):**
- Работа с изображениями, анализ визуала
- Требуется: понимание визуального контента
- **Рекомендация:** Gemini 3 Pro (Vision ELO 1309)
- **Ключевые слова:** "изображение", "картинка", "визуал", "фото", "дизайн"
- **Источник:** LMSYS Vision категория

**Search (Поиск):**
- Поиск и извлечение информации
- Требуется: понимание контекста запроса
- **Рекомендация:** Модели из LMSYS Search категории
- **Ключевые слова:** "найди", "поиск", "извлечь", "информация"
- **Источник:** LMSYS Search категория

**Enterprise (Корпоративные задачи):**
- Бизнес-процессы, интеграции, enterprise системы
- Требуется: понимание enterprise потребностей
- **Рекомендация:** Модели из MCP-Universe рейтингов
- **Ключевые слова:** "интеграция", "процесс", "система", "enterprise", "корпоративный"
- **Источник:** Nebuly MCP-Universe

### Эвристики:

**Если задача < 50 слов:**
- Сложность: Простая
- Рекомендация: Flash модель

**Если содержит "код" + "сложный":**
- Сложность: Сложная
- Рекомендация: Claude Sonnet или Opus

**Если содержит "анализ" + "данные" + "отчёт":**
- Сложность: Средняя
- Рекомендация: DeepSeek V3.2 (логика + дешево)

**Если задача > 500 слов:**
- Контекст: Большой
- Проверить объём входных данных
- Если > 100K токенов → Gemini Pro (1M контекст)
