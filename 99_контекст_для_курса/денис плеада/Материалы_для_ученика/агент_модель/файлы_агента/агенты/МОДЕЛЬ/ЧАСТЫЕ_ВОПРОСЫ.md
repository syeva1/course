## ОБЩИЕ ВОПРОСЫ

### Что делает агент @модель?

Агент оценивает твою задачу и рекомендует оптимальную AI-модель на основе:
- Актуальных рейтингов (LMSYS Arena, OpenRouter, Digital Applied)
- Бенчмарков (SWE-bench, AIME, ARC-AGI-2)
- Цен и ROI
- Типа задачи и сложности

**Цель:** Помочь выбрать модель, которая выполнит задачу максимально качественно и экономично.

---

### Как быстро работает агент?

**Очень быстро:**
- 80% задач: < 0.5 сек (Fast Track, Типовые задачи)
- 95% задач: < 1 сек
- 99% задач: < 3 сек

**Секрет:** Многоуровневая система оптимизации (9 уровней проверки).

---

### Откуда берутся рейтинги?

**Tier 1 (Критичные):**
- LMSYS Chatbot Arena — ELO рейтинги на основе пользовательских оценок
- Digital Applied — бенчмарки (SWE-bench, AIME)

**Tier 2 (Актуальность):**
- LLM-Stats.com — ежедневные обновления
- OpenRouter — популярность и цены

**Tier 3 (Дополнительно):**
- Vellum AI, Nebuly Blog, API Performance

**Обновление:** Раз в неделю (автоматически или по команде)

## ВЫБОР МОДЕЛИ

### Почему рекомендована именно эта модель?

Агент учитывает:
1. **Тип задачи** — креатив/код/логика/массовка/WebDev/Vision
2. **Сложность** — простая/средняя/сложная/очень сложная
3. **Контекст** — объём входных данных
4. **Специализированные рейтинги** — WebDev ELO, Vision ELO, SWE-bench
5. **Цену** — ROI (цена vs качество)
6. **Актуальность** — последние данные из рейтингов

**Пример:**
- Задача: "Создать лендинг" → Тип: WebDev → Проверка WebDev ELO → Claude Opus 4.5 Thinking (ELO 1520)

---

### Можно ли выбрать другую модель?

Да! Агент даёт рекомендацию, но выбор за тобой.

**Если выбрал другую:**
1. Укажи какую и почему
2. Агент сохранит в статистику
3. В следующий раз учтёт твой выбор

**Пример:**
```
Использовал GPT-4o вместо Claude Sonnet, потому что привык к ней
```
→ Агент обновит статистику
→ Для похожих задач будет рекомендовать GPT-4o

---

### Как агент выбирает между похожими моделями?

**Приоритет:**
1. **Качество** — LMSYS ELO, бенчмарки
2. **Специализация** — WebDev/Vision/Search рейтинги
3. **Цена** — ROI
4. **Популярность** — OpenRouter статистика

**Пример:**
- Claude 3.5 Sonnet vs Gemini 2.5 Flash для креатива:
  - Claude: лучший слог, но дороже ($3 vs $0.30)
  - Gemini: хорошо для простых текстов, дешевле
  - **Рекомендация:** Gemini (для простых), Claude (для сложных)

### Как часто обновляются рейтинги?

**Автоматически:** Раз в неделю (фоновое обновление)

**Вручную:**
```

```

**Источники:**

- OpenRouter: раз в 3 дня (цены меняются)

---

### Что если данные устарели?

Агент пометит:
```
⚠️ Данные от 17.12.2025 (7 дней назад)
Рекомендую обновить: @модель обнови информацию
```

**Если > 30 дней:**
```
⚠️ КРИТИЧНО: Данные устарели (30+ дней)
Обязательно обновите!
```

---

### Что если источники недоступны?

**Fallback стратегия:**
1. Использовать кэш (последние сохранённые данные)
2. Пометить "данные от [дата]"
3. Если все источники недоступны → базовая таблица

**Агент НИКОГДА не скажет "не знаю"** — всегда есть fallback.

## РАСШИРЕННЫЕ ВОЗМОЖНОСТИ

### Что такое флаг --detail?

Детальный анализ с полным сравнением моделей:
```
@модель [задача] --detail
```

**Что получишь:**
- Полная оценка задачи
- Сравнительная таблица моделей
- Источники данных с датами
- Детальное обоснование

---

### Что такое флаг --full?

Полный анализ всех источников (медленнее, но точнее):
```
@модель [задача] --full
```

**Когда использовать:**
- Очень сложные задачи
- Комплексные проекты
- Когда нужна максимальная точность

---

### Что такое флаг --быстро?

Принудительный блиц-режим (< 0.5 сек):
```
@модель [задача] --быстро
```

**Когда использовать:**
- Когда нужен мгновенный ответ
- Для простых задач
- Когда время критично

## ОБУЧЕНИЕ И УЛУЧШЕНИЕ

### Как агент обучается?

**Автоматически:**
1. Сохраняет статистику выбора моделей
2. Анализирует обратную связь
3. Обновляет эвристики
4. Добавляет типовые задачи

**Пример:**
- Если 3 раза подряд выбрана другая модель
- → Агент обновит рекомендацию
- → В следующий раз порекомендует выбранную модель

---

### Как дать обратную связь?

После каждой рекомендации агент спросит:
```
Оцени результат:
- Отлично
- Хорошо
- Нужны правки
- Плохо
```

**Если выбрал другую модель — укажи какую и почему.**

Агент сохранит и учтёт в будущем.
