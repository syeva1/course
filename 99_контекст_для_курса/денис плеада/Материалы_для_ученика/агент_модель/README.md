# 🤖 АГЕНТ @модель — Автоматический выбор AI-модели

> **Версия:** 1.0  
> **Дата:** Декабрь 2025

---

## 📖 ЧТО ЗДЕСЬ

Это папка с агентом **@модель** для автоматического выбора оптимальных AI-моделей.

**Агент помогает:**
- Оценить сложность задачи
- Выбрать лучшую модель (цена + скорость + качество)
- Получить рекомендацию за < 1 сек
