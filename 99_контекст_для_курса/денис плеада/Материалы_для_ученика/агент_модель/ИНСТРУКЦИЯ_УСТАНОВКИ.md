## 🎯 ЧТО ТАКОЕ АГЕНТ @модель

**Агент @модель** — это AI-помощник, который помогает выбрать оптимальную AI-модель для любой задачи:
- Оценивает сложность задачи
- Анализирует контекст и требования
- Рекомендует лучшую модель (цена + скорость + качество)
- Обновляет информацию из актуальных рейтингов

**Скорость:** 80% задач решаются за < 0.5 сек ⚡

## 📥 СПОСОБ 1: АВТОМАТИЧЕСКАЯ УСТАНОВКА (РЕКОМЕНДУЕТСЯ)

### Шаг 1: Открой терминал в Cursor

**Mac/Linux:**
- `Cmd + J` → выбери "Terminal"

**Windows:**
- `Ctrl + J` → выбери "Terminal"

### Шаг 2: Перейди в папку с материалами

```bash
cd path/to/04_Выбор_Моделей/агент_модель/
```

### Шаг 3: Запусти скрипт установки

```bash
python3 install_agent.py
```

### Что произойдёт:
1. Скрипт найдёт корень твоего проекта (где `.cursor/`)
2. Скопирует файл `.cursor/rules/модель.mdc`
3. Создаст папку `агенты/МОДЕЛЬ/` с файлами агента
4. Проверит корректность установки
5. Покажет инструкции по использованию

### Если корень проекта не найден автоматически:

```bash
python3 install_agent.py --path /path/to/your/project
```

## 🆘 ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК

### Проблема: "Агент не отвечает"

**Решение:**
1. Проверь, что файл `.cursor/rules/модель.mdc` существует
2. Перезапусти Cursor
3. Попробуй снова: `@модель тестовая задача`

### Проблема: "Скрипт не может найти проект"

**Решение:**
Укажи путь явно:
```bash
python3 install_agent.py --path /полный/путь/к/проекту
```

### Проблема: "Permission denied"

**Решение (Mac/Linux):**
```bash
chmod +x install_agent.py
python3 install_agent.py
```

### Проблема: "Файлы уже существуют"

**Решение:**
- Скрипт спросит, хочешь ли перезаписать
- Если выберешь "нет" → создастся backup (модель.mdc.backup)
