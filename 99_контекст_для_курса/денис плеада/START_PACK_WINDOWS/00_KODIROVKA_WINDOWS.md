# 🔧 НАСТРОЙКА КОДИРОВКИ ДЛЯ WINDOWS

**Проблема:** На Windows файлы с кириллицей в названиях могут отображаться как кракозябры (например, `01_ЯЧЕХРЭВЕРЖЕШЕП` вместо `01_ПРЕЗЕНТАЦИЯ`).

**Решение:** Настроить Windows на использование UTF-8 для поддержки кириллицы.

---

## 📋 ШАГ 1: Настройка региональных параметров Windows

1. Нажми `Win + R` (клавиша Windows + R)
2. Введи: `intl.cpl` и нажми Enter
3. Откроется окно "Региональные стандарты"
4. Перейди на вкладку **"Дополнительно"**
5. Нажми кнопку **"Изменить язык системы"**
6. В открывшемся окне поставь галочку:
   - ✅ **"Бета-версия: Использовать Юникод UTF-8 для поддержки языков во всем мире"**
7. Нажми **"ОК"**
8. **Перезагрузи компьютер** (это обязательно!)

---

## 📋 ШАГ 2: Настройка PowerShell

После перезагрузки:

1. Открой **PowerShell от имени администратора**:
   - Правой кнопкой на меню Пуск → "Windows PowerShell (администратор)"
2. Скопируй и выполни эту команду (вставь и нажми Enter):

```powershell
chcp 65001 | Out-Null; [Console]::InputEncoding=[System.Text.UTF8Encoding]::new(); [Console]::OutputEncoding=[System.Text.UTF8Encoding]::new(); $OutputEncoding=[System.Text.UTF8Encoding]::new()
```

**Важно:** После выполнения команды ничего не произойдёт на экране — это нормально!

---

## 📋 ШАГ 3: Настройка Cursor

1. Открой **Cursor**
2. Нажми `Ctrl + Shift + P` (или `F1`)
3. В поисковой строке введи: `Preferences: Open Settings (JSON)`
4. Выбери **User** версию файла настроек
5. Добавь в конфиг (в конец файла, перед закрывающей скобкой):

```json
"terminal.integrated.env.windows": {
    "PYTHONUTF8": "1"
}
```

**Пример полного файла settings.json:**
```json
{
    "editor.fontSize": 14,
    "terminal.integrated.env.windows": {
        "PYTHONUTF8": "1"
    }
}
```

6. Сохрани файл (`Ctrl + S`)
7. **Перезапусти Cursor**

2. Перезапусти Cursor
3. Открой папку с файлами — названия должны отображаться правильно!
