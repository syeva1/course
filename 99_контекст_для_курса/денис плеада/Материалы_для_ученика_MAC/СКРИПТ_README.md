# 📝 ПРОМПТ ДЛЯ СОЗДАНИЯ README (УРОВЕНЬ PRO)

Используйте этот промпт, чтобы создать профессиональное описание папки, которое реально поможет AI ориентироваться.

---

**Промпт:**

```text
@/ (или укажите конкретную папку)

Проанализируй содержимое этой папки.
Создай файл README.md внутри, используя этот стандарт:

# [Название папки]

> [Краткое описание сути папки в одну строку]

---

## 🎯 О папке
Подробно: зачем эта папка, какую задачу бизнеса/проекта она решает.

## 📂 Содержимое
Опиши структуру внутри:
- 📁 **[Подпапка]** — что внутри и зачем
- 📄 **[Важный файл]** — краткая суть

## ⚙️ Как использовать
Инструкция для меня или команды:
- Что здесь искать?
- Что сюда сохранять?
- Какие правила соблюдать?

## 🔗 Связи
Если эта папка связана с другими (например, берет данные из "Входящих"), укажи это.

---

**Требования:**
1. Используй эмодзи для наглядности.
2. Стиль: четкий, деловой, без воды.
3. Если папка пустая — предложи, что в ней должно быть исходя из названия.
```

---

**Почему это работает:**
Мы добавили разделы "Как использовать" и "Связи". Теперь AI понимает не только *что* лежит в папке, но и *как* с этим работать. Это превращает файловую помойку в рабочую систему.
