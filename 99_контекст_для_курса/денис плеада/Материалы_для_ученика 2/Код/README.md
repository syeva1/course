## Быстрый старт

### 1. Скопируй файлы
Скопируй все файлы из этой папки в свой проект.

### 2. Настрой config.py
```bash
# Переименуй
mv config_example.py config.py

# Открой и заполни свои токены
```

### 3. Установи зависимости
```bash
pip install -r requirements.txt
```

### 4. Запусти бота
```bash
python bot.py
```

## Как работать с TODO

1. Открой файл в Cursor
2. Найди комментарий `TODO`
3. Прочитай подсказку под комментарием
4. Если не получается — выдели TODO и нажми `Cmd+K` (или `Ctrl+K`)
5. Cursor поможет дописать код
