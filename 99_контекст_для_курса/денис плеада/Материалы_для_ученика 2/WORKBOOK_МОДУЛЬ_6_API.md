## ЗАДАНИЯ ПО БЛОКАМ

### Блок 2: Telegram Bot API

**Задание 2.1:** Создание бота
1. Откройте Telegram → найдите @BotFather
2. Напишите /newbot
3. Введите название (например: "Cursor Helper")
4. Введите username (например: cursor_helper_yourname_bot)
5. Скопируйте токен

**Результат:** Токен бота сохранён

**Задание 2.2:** Первый запуск
1. Откройте терминал в Cursor
2. Выполните: `pip install -r requirements.txt`
3. Выполните: `python bot.py`
4. Откройте Telegram, напишите боту /start

**Результат:** Бот отвечает на /start

## ДОМАШНЕЕ ЗАДАНИЕ

### Часть 1: Базовая настройка

1. Убедитесь что бот запускается без ошибок
2. Проверьте команды:
   - [ ] /start — бот приветствует
   - [ ] /help — справка отображается
   - [ ] /status — путь к папке показан
   - [ ] /sync — статистика отображается
   - [ ] /log — последние записи показаны

### Часть 2: Тестирование функций

3. Отправьте боту 4 типа контента:
   - [ ] Текстовое сообщение

   - [ ] документы/ — документ сохранён
   - [ ] изображения/ — фото сохранено

5. Откройте транскрипт голосового — текст должен совпадать с тем что вы сказали

11. Выполните команду /log — увидите последние записи

## РЕШЕНИЕ ЧАСТЫХ ПРОБЛЕМ

### Python не найден
```
Проверьте: python --version
Если не работает — установите Python с python.org
```

### pip не найден
```
Попробуйте: python -m pip install -r requirements.txt
```

### Токен неверный
```
Ошибка: Unauthorized
Решение: Проверьте что токен скопирован полностью в config.py
```

### Бот не отвечает
```
1. Проверьте что bot.py запущен в терминале
2. Проверьте что нет ошибок в терминале
3. Проверьте что написали правильному боту в Telegram
```

### AssemblyAI не работает
```
1. Проверьте что API Key правильный в config.py
2. Проверьте что на аккаунте есть бесплатные минуты
3. Проверьте интернет-соединение
```

### Файлы не сохраняются
```
1. Проверьте что папка Распределение существует
2. Проверьте права на запись в папку
3. Проверьте что путь в config.py правильный
```
