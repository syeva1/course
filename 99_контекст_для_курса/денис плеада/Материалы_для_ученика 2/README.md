## БЫСТРЫЙ СТАРТ

### 1. Установи зависимости

```bash
cd Код
pip install -r requirements.txt
```

### 2. Создай config.py

```bash
cp config_example.py config.py
```

### 3. Заполни config.py

Получи токены по инструкциям и вставь в config.py.

### 4. Проверь интеграции

```bash
python main.py test
```

### 5. Запусти бота

```bash
python main.py bot
```

---

## ВАЖНО

- **НЕ ШАРЬ config.py** — там секретные токены
- **НЕ КОММИТЬ credentials.json** — это ключ к Google
- **Храни токены как пароли** — кто их знает, управляет системой
