СКАЧАТЬ

СКАЧАТЬ CURSOR

САЙТ Официально: cursor.sh

  • Windows: скачай файл .exe
  • macOS: скачай файл .dmg
  • ВАЖНО на Windows не начинай проект в папке с русскими буквами
WINDOWS

WINDOWS: УСТАНОВКА

  • ШАГ 1 Открой скачанный .exe
  • ШАГ 2 Нажимай “Далее” → “Установить”
  • ВАЖНО Если антивирус блокирует — временно разреши установку
Куда положить папку проекта: C:\Cursor\CURSOR_START\
macOS

macOS: УСТАНОВКА

  • ШАГ 1 Открой .dmg
  • ШАГ 2 Перетащи Cursor в Applications
  • ШАГ 3 Запусти Cursor
Куда положить папку проекта: ~/Cursor/CURSOR_START/
АККАУНТ + ОПЛАТА

АККАУНТ И ОПЛАТА

  • ШАГ 1 Открой Cursor и нажми “Войти / Зарегистрироваться”
  • ШАГ 2 Войди через Google / GitHub / Email
  • ОПЛАТА Все варианты оплаты и ссылки уже готовы
Открой файл: 01_Методология/01_Интерфейс_и_Среда/Как_оплатить_Cursor.md
ПЕРВАЯ ПАПКА

ПЕРВАЯ ПАПКА И ГОТОВНОСТЬ

  • ШАГ 1 В Cursor нажми "Открыть папку" и выбери CURSOR_START
  • ШАГ 2 Создай папку: 00_Distribution (Windows) или 00_Распределение (Mac)
  • ШАГ 3 Положи файл .cursorrules (берём из 03_Инструменты/START.cursorrules)

ПРОВЕРКА Спроси в чате: "Какие файлы ты видишь?"

Правильный ответ: видит .cursorrules и твою первую папку.
