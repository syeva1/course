# 🏢 ШАБЛОН: СИСТЕМА ПОРЯДКА (REFERENCE)

> **Назначение файла:** Это эталон логики, которую AI использует для создания ТВОЕЙ структуры.
> Тебе не нужно копировать этот файл. AI сам посмотрит сюда, чтобы понять принципы, и применит их к твоему бизнесу.

---

## 🏗️ ПРИНЦИПЫ "СИСТЕМЫ ПАПОК"

1.  **Единая точка входа:** Всё начинается с одной корневой папки (название проекта).
2.  **Inbox (00_Inbox):** Всегда есть папка для входящего хаоса.
3.  **Нумерация:** Папки пронумерованы (01, 02...) для фиксации порядка.
4.  **README:** В каждой папке есть инструкция для AI.
5.  **Системное (99_):** Технические файлы лежат отдельно.

---

## 🌟 ПРИМЕРЫ АДАПТАЦИИ

### Пример 1: Маркетинговое Агентство
```text
MY_AGENCY/
├── 00_Inbox
├── 01_Управление (Стратегия, Юристы)
├── 02_Клиенты (Папки по каждому клиенту)
├── 03_Трафик (FB, Google, VK)
├── 04_Контент (Копирайтинг, Дизайн)
├── 05_Продажи (КП, Договоры)
├── 06_Команда (Обучение, Найм)
└── 99_Системное
```

### Пример 2: Онлайн-Школа
```text
MY_COURSE/
├── 00_Inbox
├── 01_Методология (Программы, Смыслы)
├── 02_Производство (Видео, Монтаж)
├── 03_Платформа (GetCourse, Tech)
├── 04_Маркетинг (Воронки, Запуски)
├── 05_Кураторы (База знаний, Скрипты)
└── 99_Системное
```

### Пример 3: Личный Бренд / Блог
```text
MY_BLOG/
├── 00_Inbox
├── 01_Стратегия (Позиционирование)
├── 02_Контент_План (Темы, Рубрики)
├── 03_Производство (Reels, YouTube, Посты)
├── 04_Сотрудничество (Реклама, Интеграции)
└── 99_Системное
```

## 🧠 КАК ЭТО РАБОТАЕТ
Когда ты просишь AI создать структуру, он:
1.  Смотрит на твой "Профиль Проекта" (кто ты).
2.  Смотрит на эти Примеры (как надо строить).
3.  **Синтезирует** уникальную структуру, которая идеально подходит тебе.
