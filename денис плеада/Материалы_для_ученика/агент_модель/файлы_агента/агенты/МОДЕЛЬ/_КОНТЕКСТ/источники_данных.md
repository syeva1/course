## ВАЖНЫЕ ТРЕНДЫ 2025

### Проблема "slop":
- **Суть:** Модели получают высокие оценки за форматирование (списки, заголовки) вместо качества содержания
- **Решение:** LMSYS ввёл Style Control — корректировку рейтингов
- **Результат:** Anthropic поднялись, OpenAI и xAI снизились
- **Вывод для агента:** Учитывать Style Control при анализе LMSYS рейтингов

### Специализация рейтингов:
- Рейтинги стали динамичными и привязаны к enterprise-потребностям
- Категоризация по типам задач (WebDev, Vision, Search, Reasoning)
- Общий ELO рейтинг менее важен, чем специализированный

### Enterprise-фокус:
- Появление MCP-Universe для enterprise задач
- LiveBench для reasoning
- Function-calling рейтинги
