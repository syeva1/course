# 📚 МАТЕРИАЛЫ ДЛЯ УЧЕНИКА

**Модуль 04: Выбор Моделей**

Как выбирать правильную модель для задачи

---

## 📂 ФАЙЛЫ В ЭТОЙ ПАПКЕ:

- **presentation.html** — Презентация урока (открой в браузере)
- **ТАБЛИЦА_ВЫБОРА_МОДЕЛЕЙ.md** — Таблица-помощник

---

## 🚀 С ЧЕГО НАЧАТЬ:

1. **Прочитай инструкцию** (файл со "Старт" или главный README)
2. **Открой презентацию** в браузере (двойной клик)
3. **Следуй шагам** из инструкции
4. **Используй шаблоны** для своих задач

---

## 💡 ВАЖНО:

- Все файлы открываются в Cursor или браузере
- Презентации (.html) — только в браузере
- Markdown (.md) — в Cursor или любом редакторе
- Не торопись — лучше качественно, чем быстро
