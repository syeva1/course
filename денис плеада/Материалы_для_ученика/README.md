## 🤖 АГЕНТ @модель

**Автоматический выбор оптимальной AI-модели**

### Что это:
Агент помогает выбрать лучшую модель для любой задачи за < 1 сек

### Как установить:
```bash
cd ../агент_модель/
python3 install_agent.py
```

### Как использовать:
```
@модель какая модель для [твоя задача]
```

**Примеры:**
- `@модель какая модель для написания поста`
- `@модель оцени задачу: Python скрипт`
- `@модель создай план --detail`

### Документация:
- `../агент_модель/ИНСТРУКЦИЯ_УСТАНОВКИ.md` — инструкция по установке
- `../агент_модель/README.md` — полная информация
