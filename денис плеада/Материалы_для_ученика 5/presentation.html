MODULE_02

ЛИЧНОЕ: ПОЛНЫЙ КОНТЕКСТ

Быстрые победы + Глубокий профиль

  • Цель: ВАУ за 15 минут.
  • Мы научим ИИ понимать тебя через Троицу: Тело / Разум / Дух.
  • Погнали "за ручку": я делаю — ты повторяешь.
кино и еда психология здоровье решения
PHILOSOPHY

ТРОИЦА: ТЕЛО / РАЗУМ / ДУХ

3 слоя, чтобы ИИ понял "кто ты"

  • ТЕЛО — анализы, чекапы, врачи, рост/вес, сон, энергия, спорт.
  • РАЗУМ — тесты, паттерны решений, заметки, что нравится/бесит.
  • ДУХ — цель, стратегия, счастье, куда идёшь. Эзотерика — по желанию.

Качество ответа ИИ на 90% зависит от Контекста, который ты ему дал.

BASE_SETUP

ШАГ 1: ПАПКА "ЛИЧНОЕ"

Создаём место для твоего профиля

  • Создай в корне проекта папку: 02_Личное.
  • Внутри создай подпапки: 00_Distribution, 01_Профиль, 02_Тесты.
  • Это твой «сейф» для контекста.

Сначала Личное, потом Бизнес. Тебе нужно увидеть магию на себе.

INBOX

РАСПРЕДЕЛЕНИЕ (DISTRIBUTION)

Не думай "куда положить" — кинь в кучу

  • Папка 00_Distribution — это твой Inbox.
  • Кидай туда ВСЁ: заметки, анализы, скрины тестов.
  • Попроси ИИ: "Разложи файлы из Distribution по папкам личного профиля."

ИИ — твой ассистент. Пусть он сам наводит порядок.

QUICK_WIN_1

МАГИЯ #1: КИНО

Хватит тратить час на выбор

  • Открываешь CHAT (Cmd+L).
  • Даёшь вкусы: жанры, темп, что любишь, что НЕ надо.
  • Просишь 5 вариантов + “почему это под меня”.

Результат: Кино_на_вечер.md — сохрани это в папку.

QUICK_WIN_2

МАГИЯ #2: ПУТЕШЕСТВИЕ

План «Выходной за 2 минуты»

  • Включаешь WEB SEARCH в Чате.
  • Говоришь: «Я в Питере. Найди 3 необычных места без туристов».
  • Просишь маршрут по часам + где выпить лучший кофе.

Результат: План_выходных.md.

QUICK_WIN_3

МАГИЯ #3: ЕДА

Личный нутрициолог бесплатно

  • Открываешь COMPOSER (Cmd+I).
  • Просишь меню на 3 дня (белковое) + список покупок.
  • Нажимаешь ACCEPT. Файл готов сам.

Результат: Меню_недели.md.

TECH_INFO

ЗОЛОТЫЕ ФОРМАТЫ

С чем лучше всего работает Cursor

  • .md (Markdown) — Золотой стандарт для мыслей и структуры.
  • .csv — Лучший для таблиц. В 10 раз легче Excel.
  • .txt — Простейший текст. Надёжно как кирпич.

СОВЕТ: Тяжёлые Excel конвертируй в CSV. AI скажет спасибо.

PERSONAL_DATA

ШАГ 2: ВКУСЫ

По 3 штуки — этого хватит

  • 3 сериала
  • 3 фильма
  • 3 книги
  • 3 артиста/группы
  • 3 страны / 3 города
  • 3 любимых бренда
  • 3 “что бесит” (триггеры)

Не нужно умничать. Пиши как есть.

MIND_CORE

КНИГИ (НЕ СПИСОК, А СМЫСЛ)

ИИ поймёт, как ты думаешь

  • 3 книги + почему они твои.
  • 1 мысль из каждой — которая осталась в голове.
  • Не читаешь? Ок. Тогда 3 подкаста или канала.

Это меняет стиль ответов ИИ "под тебя".

BODY_SPORT

ТЕЛО: СПОРТ

Что любишь + что нельзя

  • Спорт, который тебе реально нравится (не "надо", а "хочу").
  • Сколько раз в неделю реально готов? Не вру себе.
  • Что нельзя: травмы, спина, колени — всё честно.

ИИ даст план, который ты не бросишь на второй день.

BODY_HEALTH

ТЕЛО: ЗДОРОВЬЕ

Анализы, чекапы (без паники)

  • Заключения врачей + Анализы (скрины/PDF) → в Distribution.
  • Давление, пульс, сон, вес — текстом, коротко.
  • ИИ найдет связи (например: стресс → анализы → энергия).

ИИ не врач. Это для контекста, не для диагноза.

MIND_PSY

РАЗУМ: ПСИХОЛОГИЯ

Тесты (нормальные, на psytests.org)

  • Пройди: Big Five, 16 типов (MBTI), Характер.
  • Результаты → скрин/PDF → в Distribution.
  • ИИ узнает твой психотип и будет общаться как лучший коуч.

Чем больше он знает о тебе, тем легче с ним работать.

MIND_ENERGY

САМОЧУВСТВИЕ

Тревожность / настроение (скрининги)

  • Найди "PHQ‑9" и "GAD‑7" на psytests.org.
  • Запиши баллы. Это твой «термометр» состояния.
  • Если баллы высокие — это повод сходить к профи.
MIND_DECISION

СТИЛЬ ПРИНЯТИЯ РЕШЕНИЙ

Ты аналитик или интуит?

  • Найди тест "стиль принятия решений" на сайте.
  • Опиши 3 последних решения: что было, что выбрал, почему.
  • ИИ вычислит паттерн: риски, скорость, самосаботаж.
UPLOAD

КАК СКОРМИТЬ ИИ

Простой протокол через @

  • Открой папку "02_Личное" в Cursor.
  • В чат: прикрепи всю папку через @.
  • Напиши: "Собери мой профиль: Тело/Разум/Дух. Коротко и конкретно."

НЕ грузи: паспорт, пароли, адрес. Только контекст.

FINAL_RESULT

РЕЗУЛЬТАТ

Файл PROFILE_FOR_AI.md

Структура твоего Второго Мозга: 1) ТЕЛО: сон/энергия/спорт/ограничения 2) РАЗУМ: характер/стресс/решения/паттерны 3) ДУХ: цель/стратегия/куда идёшь 4) КАК СО МНОЙ: что бесит + что работает ПРОМПТ ДЛЯ ЦЕЛЕЙ (опционально): “Ты мой стратег. Вот мой профиль @Личное. Помоги поставить 3 цели на 3 месяца. Пиши без воды. Задай 3 уточняющих вопроса.”
