## 🎬 ШАГ 2: УРОК 2.1 — КИНО И ВИНО (Тест Чата)

**Цель:** Понять, как работает Chat (`Cmd+L`) и получить первый результат.

**Задача:** Подобрать фильм на вечер, но не банально.

**Практика:**

1. Нажми `Cmd+L` (Mac) или `Ctrl+L` (Win) — откроется Chat
2. Промпт: 
   ```
   Я устал, хочу комедию про бизнес, но умную. Не "Волк с Уолл-стрит". 
   Что посмотреть? Дай 5 вариантов с обоснованием каждого.
   ```
3. Результат: AI дает подборку с обоснованием
4. Усложнение: 
   ```
   А теперь подбери вино к этому фильму. Объясни связь между фильмом и вином.
   ```

**Что ты узнаешь:**
- Как работает Chat (простой диалог с AI)
- Как формулировать запросы для лучших результатов
- Что AI может связать разные темы (кино + вино)

**Результат:** Список фильмов и вина, сохраненный в `Кино_на_вечер.md`

**Что ты узнаешь:**

- Что AI может структурировать информацию (маршрут, время)

**Результат:** Маршрут выходного дня, сохраненный в `Маршрут_выходного_дня.md`

## 📁 ШАГ 4: УРОК 2.3 — ЛИЧНАЯ ЖИЗНЬ (Структура)

**Цель:** Создать структуру папок для личных дел через Composer.

**Задача:** Навести порядок в документах и личных файлах.

**Практика:**

1. Открой файл `ШАБЛОН_СТРУКТУРЫ_ЖИЗНЬ.md` в этой папке
2. Скопируй структуру из файла
3. Нажми `Cmd+I` (Composer) — откроется режим создания
4. Вставь структуру и добавь промпт:
   ```
   Создай эту структуру папок и файлов в текущей директории.
   Для каждой папки создай README.md с описанием, что там хранить.
   ```
5. Нажми `Accept` (Ctrl+Enter)
6. Результат: Папки "Документы", "Здоровье", "Путешествия" и т.д. появляются автоматически

**Что ты узнаешь:**
- Как работает Composer (`Cmd+I`) — создание файлов и папок
- Как AI создает структуру по шаблону
- Что README файлы помогают AI понимать контекст папок

**Результат:** Готовая структура папок для личной жизни

**ВАЖНО:** Если хочешь создать персональную структуру под себя — используй `ШАБЛОН_СОЗДАНИЯ_ПАПКИ_ЛИЧНОЕ.md` в режиме Plan.

## 📄 ШАГ 6: УРОК 2.5 — ТЕХНИЧЕСКИЕ СТАНДАРТЫ (Форматы файлов)

**Цель:** Понять, какие файлы AI понимает лучше всего.

**Смыслы:**

1. **`.md` (Markdown)** — **Золотой стандарт**
   - ИИ обожает заголовки `#` и списки `-`
   - Легкий, структурированный
   - Используй для всех заметок, планов, инструкций

2. **`.csv`** — **В 10 раз лучше Excel**
   - Если в таблице много данных — сохраняй в CSV
   - AI читает его быстрее и точнее
   - Используй для списков покупок, расходов, контактов

3. **`.txt`** — **Простой и надёжный**
   - Если не нужен Markdown, используй обычный текст
   - Безотказно работает всегда

4. **`.pdf` / `.docx`** — **Тяжёлые**
   - Cursor может их читать, но они "тяжёлые"
   - Для долгосрочной памяти лучше переводить их в `.md`

**Действие:** Конвертировать свои списки дел или покупок из Excel в CSV или MD и «скормить» Cursor.

**Пример:**
```
@список_покупок.csv
Создай меню на неделю на основе этих продуктов.
```

## 🔧 РЕЖИМЫ РАБОТЫ CURSOR (КРАТКО)

**Chat (`Cmd+L`):**
- Простой диалог с AI
- Получение информации, советов, ответов
- Не создает файлы автоматически

**Composer (`Cmd+I`):**
- Создание и редактирование файлов
- AI видит структуру проекта
- Может создавать несколько файлов одновременно

**Plan (в Composer):**
- AI создает план перед выполнением
- Задает вопросы для уточнения
- Используй для сложных задач

**Когда что использовать:**
- **Chat** → Быстрые вопросы, советы, информация
- **Composer** → Создание файлов, структуры, кода
- **Plan** → Сложные задачи, требующие уточнений
