## 📋 ШАГ 3: Создай файл `.cursorrules` в корне проекта

### Вариант 1: Через Cursor

1. Открой Cursor
2. Нажми `Cmd + N` (новый файл)
3. Вставь содержимое из `03_START.cursorrules`
4. Сохрани как `.cursorrules` (с точкой в начале!)
5. Положи в **корень своего проекта**

### Вариант 2: Через Терминал

```bash
cd /путь/к/твоему/проекту
nano .cursorrules
# Вставь содержимое
# Cmd+O → Enter → Cmd+X
```
