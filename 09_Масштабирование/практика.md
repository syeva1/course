# Практика: Модуль 10 — Масштабирование

## Задание 1: Попробуй Claude Code или Cline

**Цель:** Запустить AI-агента в терминале или VS Code.

Вариант A — Claude Code:
1. Установите: `npm install -g @anthropic-ai/claude-code`
2. Запустите в папке проекта: `claude`
3. Дайте задачу: «Создай файл hello.py с функцией, которая возвращает текущую дату»
4. Оцените результат

Вариант B — Cline:
1. Установите расширение Cline в VS Code
2. Подключите API-ключ (Claude, OpenAI или бесплатную модель)
3. Дайте аналогичную задачу
4. Сравните опыт с Cursor

**Результат:** Скриншот работающего агента и ваша оценка (удобство, скорость, качество).

## Задание 2: Исследование с GenSpark

**Цель:** Провести мини-ресерч с помощью AI.

1. Зайдите на genspark.ai
2. Задайте вопрос по вашей сфере: «Тренды в [ваша ниша] на 2025 год»
3. Изучите отчёт
4. Сравните с обычным поиском в Google

**Результат:** Краткое резюме — что нашли, насколько полезно, сэкономило ли время.

## Задание 3: NotebookLM

**Цель:** Получить AI-эксперта по своим документам.

1. Зайдите на notebooklm.google.com
2. Загрузите 2–3 документа (PDF, статьи, свои заметки)
3. Задайте 5 вопросов по загруженным материалам
4. Оцените качество ответов

**Результат:** Список вопросов и оценка полезности (1–10).

## Задание 4: Персональная дорожная карта

**Цель:** Составить план развития на 3 месяца.

1. Откройте `материалы/дорожная_карта.md`
2. Выберите 3 инструмента для освоения
3. Определите 3 процесса для автоматизации
4. Запишите конкретные шаги и дедлайны

**Результат:** Заполненная дорожная карта с датами.

## Бонус: Wispr Flow

Если у вас macOS или Windows:
1. Установите Wispr Flow (wispr.ai)
2. Продиктуйте промпт для Claude/ChatGPT голосом
3. Оцените, насколько это быстрее набора текста
