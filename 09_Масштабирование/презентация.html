Claude Code — голосом «сделай сайт»

Представь: говоришь «сделай мне сайт-визитку» — и он делает. Без Cursor, без интерфейса.
  • Что это: AI-агент, который работает в терминале (чёрное окно с текстом)
  • Как работает: пишешь задачу текстом → он сам создаёт файлы, пишет код, запускает
  • Зачем: для сложных задач, где Cursor не справляется — рефакторинг, деплой, автоматизация
  • Пример: «Создай Telegram-бота, который сохраняет голосовые в текст» → он сам напишет весь код
  • С голосом: Wispr Flow + Claude Code = диктуешь задачу голосом → агент выполняет

ЦЕНА $20/мес (Claude Max) или по API. Один человек с Claude Code делает работу отдела из 5 человек.

АССИСТ+

Cline — бесплатно, работает с любой моделью

Для тех, кто не хочет платить $20/мес за Cursor Pro
  • Что это: расширение для VS Code (бесплатный аналог Cursor)
  • Главный плюс: подключаешь ЛЮБУЮ модель — Claude, GPT, Gemini, или бесплатные локальные
  • Как начать: установи VS Code → Extensions → Cline → вставь API-ключ → готово
  • Пример: подключил бесплатную модель Qwen → пишешь «создай лендинг» → работает!
  • Для кого: если не готов платить за Cursor, но хочешь AI-ассистента в редакторе кода

ЭКОНОМИЯ Cline + бесплатная модель = $0. Cline + API Claude = ~$5-10/мес (вместо $20 за Cursor)

Conductor — 5 агентов одновременно

Открыл 5 окон — в каждом свой агент. Один делает сайт, второй пишет тексты, третий ищет конкурентов.
  • Как это выглядит: 5 терминалов на экране, в каждом Claude Code с отдельной задачей
  • Агент 1: «Сделай лендинг для курса» — пишет HTML/CSS
  • Агент 2: «Напиши 10 постов для Telegram» — генерирует контент
  • Агент 3: «Найди 5 конкурентов и сравни цены» — делает ресерч
  • Агент 4: «Создай email-рассылку на 5 писем» — пишет цепочку
  • Агент 5: «Проверь сайт на ошибки» — делает аудит

РЕЗУЛЬТАТ За 1 час ты получаешь работу, которую команда из 5 человек делала бы неделю

Wispr Flow — голосовой ввод

Диктуй текст вместо набора
  • Распознаёт речь и превращает в текст в любом приложении
  • Работает на macOS и Windows
  • Ускоряет написание промптов, писем, документов в 3–5 раз
  • Поддерживает русский и английский языки

СКОРОСТЬ Говоришь — Wispr печатает. Без рук.

GenSpark, Parallel.ai — глубокий ресерч

Исследования, на которые раньше уходили дни
  • GenSpark — AI-поисковик, который собирает и анализирует информацию из десятков источников
  • Parallel.ai — запускает несколько AI-моделей параллельно для сравнения ответов
  • Идеально для анализа рынка, конкурентов, трендов
  • Один запрос = отчёт, на который раньше нужен был аналитик
АССИСТ+

Weavy, OpenCreator — визуальные холсты

Создавай интерфейсы и прототипы голосом и текстом
  • Weavy — визуальный редактор, который генерирует UI по описанию
  • OpenCreator — создаёт дизайн-компоненты и целые страницы
  • Не нужно знать CSS или Figma — описываешь словами
  • От идеи до прототипа за минуты, а не дни

NO-CODE Визуальные холсты = следующий уровень после Cursor

NotebookLM — ответы по своим документам

Google AI, который читает твои файлы

БЕСПЛАТНО notebooklm.google.com — загрузи и спрашивай

Дорожная карта: пошагово

Cursor → агенты → мульти-агенты → свой бизнес на AI
  • 1 Сейчас: ты уже в Cursor, умеешь создавать проекты с AI. Это база ✅
