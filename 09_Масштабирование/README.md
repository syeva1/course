# Модуль 10: Масштабирование — Следующий уровень

## О модуле

Финальный модуль курса АССИСТ+. Вы прошли путь от основ до мульти-агентных систем. Теперь — время выйти за рамки Cursor и освоить экосистему AI-инструментов, которые превращают одного человека в команду.

## Что внутри

- **презентация.html** — 16 слайдов: инструменты, философия, дорожная карта
- **инструкция.md** — пошаговый план прохождения модуля
- **практика.md** — практические задания
- **шпаргалка.md** — краткая выжимка ключевых тезисов
- **материалы/список_инструментов.md** — полный каталог инструментов
- **материалы/дорожная_карта.md** — план развития на 3–6 месяцев

## Ключевые темы

1. Claude Code и терминальные AI-агенты
2. Cline, Conductor — альтернативы и масштабирование
3. Wispr Flow, GenSpark, NotebookLM — специализированные инструменты
4. Философия дженералистов — почему широкие знания побеждают узкую специализацию
5. Оркестраторы AI (Марк Андриссен)
6. Рынок труда и трансформация профессий
7. Дорожная карта развития

## Результат

После модуля вы понимаете, куда расти дальше, какие инструменты освоить и почему AI-оркестратор — самая перспективная роль.
