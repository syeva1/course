## Кейсы новичков

### Парень, который не мог разобраться в нейронках
Долго использовал AI только для бытовых вопросов («как починить стиралку»). В декабре попробовал Cursor — за 5 минут собрал отчёт. «А чё, так можно было?» За январь сделал: веб-приложение, расширение Chrome, Telegram-бот.

### Автор накодила скрипт для работы
Журналист. Каждую неделю сжимает и нумерует 50+ картинок перед сдачей статей. За вечер с Cursor написала скрипт, который делает это автоматически. Экономия: 2 часа каждую неделю.

### Проверка файлов для клиентов
Менеджер. Клиенты присылают файлы для печати. Раньше вручную проверял формат, размер, цветовой профиль. Теперь — сайт-форма: клиент загружает → скрипт проверяет → отчёт автоматически.

## Кейсы бизнеса

### Парсер новостей для Telegram-канала
Миша Попов. Бот каждые 15 минут проверяет Reddit, блоги, документации. Внутренний скоринг важности. Нейронка пишет текст на русском. Присылает в Telegram с кнопками «аппрув» / «перегенерить». 10-15 часов на разработку.

### Разработчик пересобрал SaaS за $1000/мес
Платил $1000/мес за чужой софт. За неделю с Claude Code пересобрал аналог для своего бизнеса. Теперь бесплатно и полный контроль.

### 5 агентов параллельно
Предприниматель запускает 5 агентов Claude Code одновременно на разные задачи. Сам только ревьюит результаты. Продуктивность x5.

### Академик с 35-летним стажем
Профессор за один сеанс с AI сделал проекты, к которым не мог подступиться годами. «Это изменило мою работу больше, чем интернет».

### Строительный план → цветная карта
Загрузил PDF строительного плана → AI за 2 минуты создал цветную интерактивную карту помещений.

- 10-30 пулл-реквестов в день с помощью Claude Code
- Продуктивность инженеров Anthropic выросла на 200%
- «Создавайте продукт для модели через 6 месяцев, не для сегодняшней»

## Что объединяет все кейсы

1. **Начали с малого** — не гигантский проект, а одна конкретная проблема
2. **Описали задачу словами** — не пытались программировать
3. **Итерировали** — первая версия кривая → исправили → ещё раз → работает
4. **Не были программистами** — журналист, менеджер, профессор, предприниматель
