# Все команды Cursor — шпаргалка

## Палитра команд (Cmd+Shift+P / Ctrl+Shift+P)

### AI
- `Cursor: New AI Chat` — новый чат с AI
- `Cursor: Toggle Agent` — включить/выключить агента
- `Cursor: Apply All` — применить все предложения AI

### Файлы
- `File: New File` — новый файл
- `File: Save All` — сохранить все
- `Preferences: Open User Settings (JSON)` — настройки JSON

### Git
- `Git: Clone` — клонировать репо
- `Git: Commit` — закоммитить
- `Git: Push` — отправить на GitHub
- `Git: Pull` — получить с GitHub

### Терминал
- `Terminal: Create New Terminal` — новый терминал
- `Terminal: Clear` — очистить

### Вид
- `View: Toggle Side Bar` — скрыть/показать файлы
- `View: Toggle Terminal` — скрыть/показать терминал
- `View: Zen Mode` — режим фокуса

## Полезные команды терминала

```bash
# Создать проект
mkdir мой_проект && cd мой_проект

# Инициализировать git
git init

# Python виртуальное окружение
python -m venv .venv
source .venv/bin/activate  # Mac/Linux
.venv\Scripts\activate     # Windows

# Установить зависимости
pip install -r requirements.txt

# Сохранить зависимости
pip freeze > requirements.txt
```
