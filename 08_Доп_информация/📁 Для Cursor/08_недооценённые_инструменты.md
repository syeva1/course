# Недооценённые AI-инструменты 2026

## Голос и текст

### Wispr Flow
**Что:** Голосовой ввод текста в любом приложении. Говоришь → текст появляется.
**Фишка:** Убирает слова-паразиты, форматирует на лету.
**Цена:** $10/мес
**Сайт:** wispr.com

### Granola
**Что:** AI-блокнот для встреч. Без ботов в звонке (не пугает собеседников).
**Фишка:** Записывает встречу → выжимка ключевых решений → задачи.
**Цена:** Бесплатно (базовый)
**Сайт:** granola.so

## Исследования

### NotebookLM (Google)
**Что:** Загружаешь свои документы → AI отвечает ТОЛЬКО по ним.
**Фишка:** Не галлюцинирует (ответы только из твоих данных). Создаёт подкасты из документов.
**Цена:** Бесплатно
**Сайт:** notebooklm.google.com

### Parallel.ai
**Что:** Глубокий AI-ресерч. Ставит подзадачи, идёт вглубь, проверяет факты.
**Фишка:** 500 источников за одно исследование. Оценка уверенности для каждого факта.
**Цена:** $25/мес ($20 бонус при регистрации)
**Сайт:** parallel.ai

## Разработка

### Conductor
**Что:** Запускает несколько копий Claude Code параллельно на Mac.
**Фишка:** 5 задач одновременно, ты только ревьюишь.
**Сайт:** conductor.dev

### Stitch (Google)
**Что:** Генератор интерфейсов. Описал текстом → получил макет → код.
**Фишка:** Быстрее Figma для прототипов.
**Сайт:** stitch.withgoogle.com

### DeepSite (Hugging Face)
**Что:** Генерация сайтов прямо в браузере. Бесплатно.
**Фишка:** Описал → сайт готов. Без установки.
**Сайт:** huggingface.co/deepsite

### Cline
**Что:** Бесплатный CLI для вайбкодинга. Работает с любыми моделями.
**Фишка:** Claude, GPT, Gemini, DeepSeek, локальные через Ollama. Бесплатная модель MiniMax 2.5.
**Сайт:** cline.bot

## Автоматизация

### GenSpark
**Что:** AI-посредник — $25/мес за безлимит GPT-5.2, Claude Opus 4.5 и другие.
**Фишка:** Все топовые модели за $25. Генерация картинок в 4K.
**Сайт:** genspark.ai

### Weavy / OpenCreator
**Что:** Визуальные конструкторы воркфлоу в виде блоков.
**Фишка:** Один раз настроил → используешь многократно. 20 роликов автоматически.
**Сайты:** weavy.ai, opencreator.io
