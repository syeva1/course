# 🗺 Как проходить курс АССИСТ+

---

## Структура курса

Курс состоит из **8 модулей**. Проходи по порядку — каждый следующий опирается на предыдущий.

```
01_Старт          — Установка Cursor, первый проект
02_Контекст       — Настройка AI под себя, .cursorrules, контекст
03_Инструменты    — Терминал, GitHub, расширения, агенты
04_Агенты         — .mdc файлы, мульти-агенты, модели
05_Telegram-бот   — Создание ботов с AI
06_API            — Подключение внешних сервисов, MCP
07_Вайб-кодинг    — Создание реальных продуктов с нуля
08_Справочник     — Команды, альтернативы, кейсы
```

---

## Что лежит в каждом модуле

```
XX_Название/
├── README.md                    ← Описание модуля (начни с него)
├── XX_Название_презентация.html ← Главная презентация (открой в браузере)
├── инструкция.md                ← Пошаговая инструкция (делай по шагам)
├── XX_практика.html             ← Практические задания (HTML)
└── материалы/
    ├── README.md                ← Что в этих папках
    ├── XX_практика.md           ← Практика (текст)
    ├── 📖 Для изучения/         ← HTML-презентации по каждой теме
    └── 📁 Для Cursor/           ← MD-файлы для загрузки в Cursor
```

---

## Как проходить модуль

### Шаг 1. Прочитай README.md
Откроешь — увидишь: что за модуль, чему научишься, сколько времени займёт.

### Шаг 2. Посмотри презентацию
Открой файл `XX_Название_презентация.html` в браузере. Это слайды — листай кнопками внизу.

### Шаг 3. Пройди инструкцию
Открой `инструкция.md` — там пошаговые действия. Делай прямо в Cursor, шаг за шагом. После каждого шага проверяй результат.

### Шаг 4. Изучи материалы
Зайди в `материалы/`:
- **📖 Для изучения** — открывай HTML-файлы в браузере, читай как презентации
- **📁 Для Cursor** — перетащи нужный .md файл в чат Cursor, он будет использовать его как контекст

### Шаг 5. Сделай практику
Открой `XX_практика.html` (или `материалы/XX_практика.md`). Там 5 заданий:
- 2 задания — повтори по инструкции
- 2 задания — сделай сам (похожее, но своё)
- 1 задание — творческое (придумай сам)

## Папка «📁 Для Cursor» — зачем она

Это те же самые материалы, но в формате `.md` (Markdown). Cursor умеет их читать.

**Как использовать:**
1. Открой Cursor
2. Перетащи файл `.md` прямо в чат агента
3. Скажи: «Используй этот файл как базу знаний»
4. Теперь агент знает эту тему и может помогать по ней

**Пример:** Перетащил `01_шаблон_cursorrules.md` → сказал «Настрой .cursorrules для моего проекта по этому шаблону» → агент сделает.

---

## Папка «📖 Для изучения» — зачем она

Те же материалы, но в красивом HTML. Открываешь в браузере — листаешь как слайды. Удобно читать с телефона или компьютера.

---

## Советы

- **Не пропускай модули** — они идут от простого к сложному
- **Делай практику** — без неё знания не закрепятся
- **Загружай файлы в Cursor** — пусть AI знает то же, что и ты
- **Модуль 08 (Справочник)** — возвращайся к нему, когда нужно что-то найти
