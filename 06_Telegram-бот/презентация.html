@BotFather — создание бота

Главный бот Telegram для управления ботами
  • 1 Открой Telegram, найди @BotFather
  • 2 Отправь команду /newbot
  • 3 Придумай имя бота (например, «Ассистент Антона»)
  • 4 Придумай username (например, anton_assist_bot)
  • 5 Получи токен — скопируй и сохрани!

Токен — что это и как хранить

Ключ доступа к вашему боту
# Пример токена (НЕ настоящий!) 7234567890:AAH1234abcDEF_ghijklMNOPqrstuvWXYZ
  • Важно Токен = пароль от бота. Не публикуйте его!
  • Совет Храните в .env файле или переменных окружения
  • Утечка? Перевыпустите через @BotFather → /revoke

Первый бот — Echo

Отвечает тем же текстом, что вы отправили
from telegram import Update from telegram.ext import ( ApplicationBuilder, MessageHandler, filters, ContextTypes ) TOKEN = "ВАШ_ТОКЕН" async def echo(update: Update, ctx: ContextTypes.DEFAULT_TYPE): await update.message.reply_text(update.message.text) app = ApplicationBuilder().token(TOKEN).build() app.add_handler(MessageHandler(filters.TEXT, echo)) app.run_polling()

Частые ошибки

И как их избежать
  • Ошибка Токен в коде → утечка через GitHub
  • Ошибка Нет обработки исключений → бот падает молча
  • Ошибка Бот отвечает в группе на каждое сообщение
  • Ошибка Забыли await → бот не отвечает
  • Ошибка Нет rate-limit → огромные счета за API
  • Решение Используйте .env, try/except, фильтры
