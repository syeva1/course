# Шпаргалка — Telegram-бот

## Установка

```bash
pip install python-telegram-bot openai python-dotenv
```

## Импорты

```python
from telegram import Update, InlineKeyboardButton, InlineKeyboardMarkup
from telegram.ext import (
    ApplicationBuilder, CommandHandler, MessageHandler,
    CallbackQueryHandler, filters, ContextTypes
)
from openai import OpenAI
```

## Базовая структура бота

```python
app = ApplicationBuilder().token("ТОКЕН").build()
app.add_handler(CommandHandler("start", start_func))
app.add_handler(MessageHandler(filters.TEXT, message_func))
app.add_handler(CallbackQueryHandler(callback_func))
app.run_polling()
```

## Команды BotFather

| Команда | Что делает |
|---------|-----------|
| `/newbot` | Создать нового бота |
| `/setdescription` | Описание бота |
| `/setcommands` | Меню команд |
| `/revoke` | Перевыпустить токен |
| `/deletebot` | Удалить бота |

## Обработчик сообщений

```python
async def handle(update: Update, ctx: ContextTypes.DEFAULT_TYPE):
    text = update.message.text
    await update.message.reply_text(f"Вы написали: {text}")
```

## InlineKeyboard

```python
keyboard = InlineKeyboardMarkup([
    [InlineKeyboardButton("Текст кнопки", callback_data="key")],
])
await update.message.reply_text("Выберите:", reply_markup=keyboard)
```

## Обработка кнопок

```python
async def button_click(update: Update, ctx):
    query = update.callback_query
    await query.answer()
    if query.data == "key":
        await query.edit_message_text("Вы нажали кнопку!")
```

## OpenAI запрос

```python
client = OpenAI(api_key="КЛЮЧ")
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[
        {"role": "system", "content": "Системный промпт"},
        {"role": "user", "content": "Вопрос пользователя"}
    ]
)
answer = response.choices[0].message.content
```

## Полезные фильтры

- `filters.TEXT` — текстовые сообщения
- `filters.PHOTO` — фото
- `filters.COMMAND` — команды (/start, /help)
- `filters.ChatType.PRIVATE` — только личные сообщения
