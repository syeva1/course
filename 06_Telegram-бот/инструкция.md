# Инструкция — Telegram-бот за 10 шагов

## Шаг 1. Создайте бота через @BotFather

Откройте Telegram, найдите @BotFather и отправьте `/newbot`. Придумайте имя и username (должен заканчиваться на `bot`). Скопируйте токен.

## Шаг 2. Установите Python и библиотеки

```bash
pip install python-telegram-bot openai python-dotenv
```

## Шаг 3. Создайте файл .env

```
TELEGRAM_TOKEN=ваш_токен_от_BotFather
OPENAI_API_KEY=ваш_ключ_openai
```

Никогда не публикуйте `.env` — добавьте его в `.gitignore`.

## Шаг 4. Напишите echo-бота

Скопируйте код из `материалы/код_echo_бота.py`. Запустите:

```bash
python код_echo_бота.py
```

Напишите боту в Telegram — он должен ответить тем же текстом.

## Шаг 5. Добавьте команду /start

```python
async def start(update, ctx):
    await update.message.reply_text("Привет! Я бот АССИСТ+. Чем помочь?")

app.add_handler(CommandHandler("start", start))
```

## Шаг 6. Подключите AI

Скопируйте код из `материалы/код_ai_бота.py`. Замените токены на свои. Теперь бот отвечает с помощью нейросети.

## Шаг 7. Напишите системный промпт

Опишите, кто ваш бот, что он знает о бизнесе, как должен отвечать. Чем точнее промпт — тем полезнее бот.

## Шаг 8. Добавьте кнопки (InlineKeyboard)

```python
from telegram import InlineKeyboardButton, InlineKeyboardMarkup

keyboard = InlineKeyboardMarkup([
    [InlineKeyboardButton("Услуги", callback_data="services")],
    [InlineKeyboardButton("Цены", callback_data="prices")],
])
```

Обработайте нажатия через `CallbackQueryHandler`.

## Шаг 9. Добавьте обработку ошибок

```python
import logging
logging.basicConfig(level=logging.INFO)

async def error_handler(update, ctx):
    logging.error(f"Ошибка: {ctx.error}")

app.add_error_handler(error_handler)
```

## Шаг 10. Задеплойте бота

Выберите хостинг (Railway, Render, VPS), загрузите код и `.env`, запустите. Убедитесь, что бот работает после закрытия терминала.

```bash
# На VPS:
nohup python bot.py &
```
