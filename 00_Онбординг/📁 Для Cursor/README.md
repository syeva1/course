# Онбординг — Как пользоваться курсом

Этот README — навигация по курсу АССИСТ+ «Cursor AI для бизнес-ассистентов».

## Структура каждого модуля

Каждый модуль содержит **две папки**:

### 📖 Для изучения
HTML-файлы — открывай в браузере и читай:
- **Презентация** — основной урок модуля
- **Материалы** — дополнительные статьи и гайды
- **Практика** — задания с подробным описанием

### 📁 Для Cursor
MD-файлы — перетаскивай в Cursor как контекст:
- **Инструкция** — пошаговые действия
- **Материалы** — справочники, шаблоны, чеклисты
- **Практика** — задания в формате для AI

### Спецпапки (в некоторых модулях)
- **готовые_mdc** — готовые .mdc агенты (Модуль 04)
- **готовые_боты** — готовые Python-боты (Модуль 05)
- **готовые_автоматизации** — готовые скрипты (Модуль 06)

## Порядок изучения

1. `01_Старт` — установка, настройка, первые шаги
2. `02_Контекст` — персональный контекст, .mdc правила
3. `03_Инструменты` — терминал, Git, расширения
4. `04_Агенты` — создание AI-агентов
5. `05_Telegram-бот` — свой бот с нуля
6. `06_API` — интеграции и автоматизации
7. `07_Вайб-кодинг` — от идеи до деплоя
8. `08_Справочник` — горячие клавиши, ресурсы, кейсы
