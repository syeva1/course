# 📚 Нейронки на все случаи жизни

**Дополнительный модуль курса АССИСТ+**

Это справочник. Открывай, когда нужно найти инструмент под задачу.

## Что внутри

- **презентация.html** — визуальный справочник по категориям (24 слайда)
- **материалы/полный_список.md** — ВСЕ инструменты в одной таблице с ценами и ссылками
- **материалы/как_выбрать.md** — алгоритм: какая задача → какой инструмент

## Как пользоваться

1. Определи, что тебе нужно (текст, картинка, видео, код...)
2. Открой `как_выбрать.md` или пролистай презентацию до нужной категории
3. Выбери инструмент, попробуй

> AI-мир меняется каждый день. Мы обновляем справочник регулярно.
