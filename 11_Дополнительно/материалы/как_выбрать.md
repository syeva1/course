# 🧭 Как выбрать AI-инструмент

Алгоритм простой: определи задачу → выбери инструмент → попробуй.

---

## Задача → Инструмент

### 📝 Написать текст
- **Пост, письмо, статья** → ChatGPT или Claude
- **Длинный документ, анализ** → Claude (лучше держит контекст)
- **Текст + актуальные данные** → Perplexity (даёт источники)

### 🔍 Найти информацию
- **Быстрый ресерч с источниками** → Perplexity
- **Глубокий анализ большого документа** → Gemini (огромный контекст) или NotebookLM
- **Комплексное исследование** → Parallel.ai

### 💻 Написать код или сделать приложение
- **Нужен код, есть проект** → Cursor (основной инструмент курса)
- **Хочу работать в терминале** → Claude Code или Cline
- **Автодополнение в VS Code** → GitHub Copilot
- **Хочу приложение без кода** → Lovable или Bolt
- **Нужна онлайн-IDE** → Replit

### 🎨 Создать картинку
- **Максимальное качество** → Midjourney
- **Быстро и просто** → DALL-E 3 (прямо в ChatGPT)
- **Бесплатно, с полным контролем** → Stable Diffusion
- **Текст на картинке (логотип, постер)** → Ideogram

### 🎬 Создать видео
- **Профессиональная генерация** → Runway
- **Реалистичные длинные клипы** → Kling
- **Быстрый короткий клип** → Pika
- **Говорящий аватар, перевод видео** → HeyGen

### 🎵 Музыка и аудио
- **Сгенерировать песню** → Suno или Udio
- **Озвучить текст, клонировать голос** → ElevenLabs
- **Расшифровать запись в текст** → Whisper

### ⚡ Автоматизировать процессы
- **Простая автоматизация** → Zapier
- **Сложные сценарии, визуально** → Make
- **Хочу у себя на сервере** → n8n

### 🎨 Дизайн
- **Презентация, пост, баннер** → Canva AI
- **Прототип интерфейса** → Figma AI или Stitch

### 📋 Продуктивность
- **Встречи и заметки** → Granola
- **Голосовой ввод** → Wispr Flow
- **Работа со своими документами** → NotebookLM
- **Все модели в одном месте** → GenSpark

---

## Правила выбора

1. **Начни с бесплатного** — почти у всех есть бесплатный тариф
2. **Не покупай всё сразу** — возьми 2-3 инструмента под свои задачи
3. **ChatGPT + Claude** — базовый набор, покрывает 80% потребностей
4. **Пробуй новое** — AI-мир меняется каждую неделю
