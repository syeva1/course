# 📋 ТЗ: Telegram бот-подружка для контекстов

> Этот файл — готовое техническое задание.
> Перетащи его в чат агента Cursor и скажи: **«Создай этого бота»**
> Cursor сам напишет весь код.

---

## Цель

Создать Telegram-бота, который:
- Принимает сообщения, файлы, голосовые, картинки, ссылки из Telegram
- Сохраняет всё в папку «Распределение» внутри проекта Cursor
- Транскрибирует голосовые и видео (через AssemblyAI или Whisper)
- Работает локально (когда компьютер включен)

**Результат:** Любая информация из Telegram автоматически попадает в проект Cursor как контекст.

---

## Что бот умеет

| Тип сообщения | Что делает бот | Куда сохраняет |
|---------------|---------------|----------------|
| Текст | Сохраняет как .md файл | `Распределение/сообщения/` |
| Голосовое | Транскрибирует → сохраняет текст | `Распределение/транскрипты/` |
| Видео | Извлекает аудио → транскрибирует | `Распределение/транскрипты/` |
| Документ (PDF, DOCX) | Сохраняет файл | `Распределение/документы/` |
| Картинка | Сохраняет файл | `Распределение/изображения/` |
| Ссылка | Парсит заголовок, сохраняет | `Распределение/ссылки/` |

---

## Команды бота

| Команда | Что делает |
|---------|-----------|
| `/start` | Приветствие и инструкция |
| `/help` | Справка по командам |
| `/status` | Путь к папке и статистика |
| `/list` | Последние сохранённые файлы |

## Формат сохранения текстовых сообщений

```markdown
# Сообщение из Telegram

**Дата:** 2026-03-09 14:30:15
**От:** Мария (@example_user)

---

Текст сообщения здесь...
```

## Технические требования

- Python 3.10+
- Библиотека: `python-telegram-bot`
- Транскрибация: AssemblyAI (бесплатно первые 5 часов) или OpenAI Whisper
- Токены хранить в `.env` файле (не в коде!)

---

## Безопасность

- Токен бота — в `.env`, не в коде
- `.env` добавить в `.gitignore`
- Ограничить бота по ADMIN_ID (только ты можешь писать)

---

## Как использовать это ТЗ

1. Создай папку `telegram-bot` в Cursor
2. Перетащи этот файл в чат агента
3. Напиши: **«Создай бота по этому ТЗ»**
4. Cursor создаст все файлы: bot.py, config.py, requirements.txt
5. Создай бота через @BotFather (инструкция в модуле)
6. Вставь токен в `.env`
7. Запусти: `python bot.py`
