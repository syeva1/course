ТЗ: Бот-подружка для контекстов

Цель

  • Принимает сообщения, файлы, голосовые, картинки, ссылки из Telegram
  • Сохраняет всё в папку «Распределение» внутри проекта Cursor
  • Транскрибирует голосовые и видео (AssemblyAI или Whisper)
  • Работает локально (когда компьютер включен)

Вайбкодим бота с нуля

Шаг 1: Регистрация бота

  • Telegram → найди @BotFather
  • Отправь /newbot
  • Придумай название и username
  • BotFather пришлёт токен — скопируй и сохрани
АССИСТ+
02 / 05

Шаг 2: Создание в Cursor

  • Создай папку проекта (например, my-bot/)
  • Открой в Cursor: File → Open Folder
  • Режим Agent в чате
  • Опиши бота и вставь токен — Cursor создаст файлы
  • Скопируй команду из чата и запусти в терминале
АССИСТ+
03 / 05

Шаг 3: Итерации

  • Первая версия никогда не идеальна — это нормально
  • Описывай проблему в чате: «Бот не пишет, за что замьютил. Доработай.»
  • Добавляй команды: «Добавь /help со списком функций»
  • Цикл: получил → проверил → описал → Cursor исправил → проверил снова
АССИСТ+
04 / 05

Шаг 4: Деплой на сервер

  • Арендуй VPS (Timeweb, Beget — от 300₽/мес, Ubuntu)
  • Попроси Cursor: «Напиши пошаговый план деплоя на Ubuntu»
  • Подключись: ssh root@[IP], перенеси файлы (nano bot.py)
  • Установи: pip install python-telegram-bot python-dotenv → python bot.py
