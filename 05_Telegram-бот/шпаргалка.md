# Шпаргалка — Telegram-бот

## Установка

```bash
pip install python-telegram-bot==21.7 openai python-dotenv aiohttp
```

## Импорты

```python
from telegram import Update, InlineKeyboardButton, InlineKeyboardMarkup
from telegram.ext import (
    ApplicationBuilder, CommandHandler, MessageHandler,
    CallbackQueryHandler, filters, ContextTypes
)
```

## Каркас бота

```python
app = ApplicationBuilder().token("ТОКЕН").build()
app.add_handler(CommandHandler("start", start_func))
app.add_handler(MessageHandler(filters.TEXT, text_handler))
app.add_handler(MessageHandler(filters.VOICE, voice_handler))
app.add_handler(MessageHandler(filters.Document.ALL, doc_handler))
app.add_handler(MessageHandler(filters.PHOTO, photo_handler))
app.add_handler(CallbackQueryHandler(button_handler))
app.run_polling()
```

## Скачивание файлов

```python
# Фото (берём самое большое)
photo = update.message.photo[-1]
file = await photo.get_file()
await file.download_to_drive("path/photo.jpg")

# Документ
doc = update.message.document
file = await doc.get_file()
await file.download_to_drive(f"path/{doc.file_name}")

# Голосовое
voice = update.message.voice
file = await voice.get_file()
await file.download_to_drive("path/voice.ogg")
```

## InlineKeyboard

```python
keyboard = [
    [InlineKeyboardButton("Кнопка 1", callback_data="btn1")],
    [InlineKeyboardButton("Кнопка 2", callback_data="btn2")],
]
markup = InlineKeyboardMarkup(keyboard)
await update.message.reply_text("Выбери:", reply_markup=markup)
```

## Обработка кнопок

```python
async def button_handler(update, context):
    query = update.callback_query
    await query.answer()
    if query.data == "btn1":
        await query.edit_message_text("Ты нажал кнопку 1")
```
