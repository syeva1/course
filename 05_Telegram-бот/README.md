# Модуль 05 — Telegram-бот для бизнеса

> Курс АССИСТ+ | От автоответчика до AI-ассистента, который собирает контекст

## Что внутри

- **презентация.html** — 18 слайдов: 5 типов ботов, архитектура, код, деплой
- **инструкция.md** — пошаговая инструкция (12 шагов)
- **практика.md** — 3 практических задания
- **шпаргалка.md** — быстрый справочник по python-telegram-bot
- **материалы/** — готовые примеры кода
  - `код_контекст_бота.py` — бот-контекст-сборщик (сохраняет всё в папки)
  - `код_ai_бота.py` — бот с AI-консультантом
  - `ТЗ_бота_шаблон.md` — шаблон технического задания для бота

## 5 типов бизнес-ботов

1. **Автоответчик** — отвечает на частые вопросы (цены, услуги, контакты)
2. **AI-консультант** — знает о бизнесе всё, отвечает как менеджер
3. **Контекст-сборщик** — собирает файлы, голосовые, ссылки в папки для Cursor
4. **Модератор** — фильтрует спам, приветствует новичков
5. **Автоматизатор** — парсит новости, рассылает отчёты, интегрируется с CRM

## Технологии

- Python 3.10+
- python-telegram-bot 21.7
- AssemblyAI (транскрибация голосовых)
- OpenAI API (для AI-консультанта)

## Как начать

1. Откройте `презентация.html` в браузере
2. Следуйте `инструкция.md` шаг за шагом
3. Выполните задания из `практика.md`
