# Модуль 06 — Telegram-бот

> Курс АССИСТ+ | От @BotFather до рабочего AI-бота за один модуль

## Что внутри

- **презентация.html** — 16 слайдов: создание, настройка и деплой Telegram-бота
- **инструкция.md** — пошаговая инструкция (10 шагов)
- **практика.md** — 3 практических задания
- **шпаргалка.md** — быстрый справочник по ключевым командам и паттернам
- **материалы/** — готовые примеры кода
  - `код_echo_бота.py` — минимальный echo-бот
  - `код_ai_бота.py` — бот с подключённым AI (OpenAI)

## Для кого

Ассистенты и предприниматели, которые хотят автоматизировать общение с клиентами через Telegram.

## Технологии

- Python 3.10+
- python-telegram-bot
- OpenAI API

## Как начать

1. Откройте `презентация.html` в браузере
2. Следуйте `инструкция.md` шаг за шагом
3. Выполните задания из `практика.md`
