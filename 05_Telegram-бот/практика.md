# Практика — Модуль 05: Telegram-бот

## Задание 1. Контекст-сборщик

**Цель:** создать бота, который собирает всё из Telegram в папку Распределение.

1. Создай бота через @BotFather
2. Возьми за основу `материалы/код_контекст_бота.py`
3. Запусти и протестируй:
   - Отправь текстовое сообщение → проверь `Распределение/сообщения/`
   - Отправь фото → проверь `Распределение/изображения/`
   - Отправь PDF → проверь `Распределение/документы/`
   - Отправь голосовое → проверь `Распределение/транскрипты/`
4. Убедись, что файлы именуются по формату: `YYYY-MM-DD_HH-MM-SS_тип.md`

**Результат:** бот раскладывает любой контент из Telegram по папкам.

---

## Задание 2. AI-консультант для твоего бизнеса

**Цель:** сделать бота, который знает о бизнесе и отвечает клиентам.

1. Возьми за основу `материалы/код_ai_бота.py`
2. Напиши системный промпт:
   - Название компании
   - Услуги с ценами
   - Контакты
   - Правила ответов (вежливость, что НЕ отвечать)
3. Добавь InlineKeyboard с 3-5 кнопками
4. Протестируй: задай боту 10 вопросов о бизнесе
5. Проверь: бот не выдумывает цены? Предлагает связаться, если не знает?

**Результат:** бот-консультант, который отвечает клиентам 24/7.

---

## Задание 3. Придумай своего бота

**Цель:** определить, какой бот нужен именно твоему бизнесу.

1. Выбери тип: автоответчик / консультант / сборщик / модератор / автоматизатор
2. Напиши ТЗ в свободной форме (используй `материалы/ТЗ_бота_шаблон.md`)
3. Открой Cursor, вставь ТЗ в чат Agent Mode
4. Скажи: «Создай Telegram-бота по этому ТЗ»
5. Тестируй → правь → тестируй

**Результат:** рабочий бот, созданный по твоему ТЗ за один вечер.
