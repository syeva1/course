# Практика — Модуль 05. Telegram-бот для бизнеса

## Задание 1. Эхо-бот (повтори по инструкции)

1. Создай бота в @BotFather, получи токен
2. Создай проект в Cursor с файлами: bot.py, config.py, .env, requirements.txt, .gitignore
3. Установи зависимости
4. Через Composer попроси AI написать эхо-бота
5. Запусти и проверь: бот повторяет твои сообщения

Результат: Бот работает, отвечает на /start и повторяет текст.

## Задание 2. Контекст-сборщик (повтори по инструкции)

Добавь к боту из задания 1:

1. Создание папки Распределение с подпапками при запуске
2. Сохранение текста в сообщения/
3. Сохранение документов в документы/

5. Сохранение ссылок в ссылки/

Проверь каждый тип: отправь текст, документ, фото, ссылку. Посмотри что появилось в папках.

Результат: 4 типа контента автоматически раскладываются по папкам.

## Задание 3. Команды управления (сделай сам)

Добавь к боту команды — но НЕ копируй промпт из инструкции. Сформулируй сам:

1. Команда для показа статистики (сколько файлов в каждой папке)
2. Команда для показа последних сохранённых файлов
3. Ограничение: команды работают только для тебя (ADMIN_ID)

Подсказка: используй os.listdir() для подсчёта файлов.

Результат: Команды работают, чужой пользователь получает отказ.

## Задание 4. Кастомный бот под свой бизнес (сделай сам)

Создай нового бота с нуля (новый токен от BotFather). Этот бот — для твоего бизнеса.

Требования:
1. Приветствие /start с описанием что бот умеет
2. InlineKeyboard с 3-4 кнопками (например: Услуги, Цены, Контакты, FAQ)
3. При нажатии кнопки — бот выдаёт соответствующую информацию
4. Свободный текст — бот сохраняет в папку или отвечает через AI

Промпт для Composer напиши сам. Описывай свой бизнес, свои кнопки, свой текст.

Результат: Рабочий бизнес-бот с меню и кнопками. Можешь показать клиентам.
