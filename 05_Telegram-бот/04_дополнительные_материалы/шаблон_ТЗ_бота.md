# Шаблон ТЗ для Telegram-бота

> Скопируй, заполни и отправь в чат Cursor — он напишет бота.

```markdown
# ТЗ: [Название бота]

## Что делает бот
[Опиши в 2-3 предложениях]

## Команды
- /start — [что происходит]
- /help — [что показывает]
- /[команда] — [действие]

## Сценарии
1. Пользователь пишет: [что]
   Бот отвечает: [что]
2. Пользователь нажимает кнопку: [какую]
   Бот делает: [что]

## Кнопки (inline)
- [Текст кнопки 1] → [действие]
- [Текст кнопки 2] → [действие]

## Данные
- Где хранить: [SQLite / JSON / Google Sheets]
- Что хранить: [список полей]

## Внешние API
- [API 1]: для [чего]
- [API 2]: для [чего]

## Технические требования
- Язык: Python 3.11+
- Библиотека: aiogram 3.x
- База: SQLite (aiosqlite)
- Хостинг: [VPS / Railway / ...]
- Токен бота: через .env файл

## Дополнительно
- [Любые пожелания]
```
