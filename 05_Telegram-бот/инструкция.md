# Инструкция — Модуль 05: Telegram-бот для бизнеса

## Шаг 1. Создай бота в BotFather

1. Открой @BotFather в Telegram
2. Отправь `/newbot`
3. Придумай имя (например: `АССИСТ помощник`)
4. Придумай username (например: `assist_helper_bot`)
5. Скопируй токен — длинная строка вида `123456:ABC-DEF...`

## Шаг 2. Создай проект в Cursor

1. Создай папку `telegram-bot`
2. Открой её в Cursor (File → Open Folder)
3. Создай файлы: `bot.py`, `config.py`, `.env`, `requirements.txt`

## Шаг 3. Настрой .env

```
TELEGRAM_BOT_TOKEN=твой_токен_от_BotFather
ASSEMBLYAI_KEY=твой_ключ_assemblyai
ADMIN_ID=твой_telegram_id
```

Добавь `.env` в `.gitignore`!

## Шаг 4. Установи зависимости

```bash
pip install python-telegram-bot==21.7 aiohttp python-dotenv openai
```

## Шаг 5. Напиши config.py

```python
from dotenv import load_dotenv
import os

load_dotenv()

TELEGRAM_BOT_TOKEN = os.getenv("TELEGRAM_BOT_TOKEN")
ASSEMBLYAI_KEY = os.getenv("ASSEMBLYAI_KEY")
DISTRIBUTION_FOLDER = "./Распределение"
ADMIN_ID = int(os.getenv("ADMIN_ID", "0"))
```

## Шаг 6. Создай структуру папок

Бот должен создавать папки автоматически при запуске:

```python
import os
folders = ["сообщения", "транскрипты", "документы", "изображения", "ссылки"]
for folder in folders:
    os.makedirs(f"./Распределение/{folder}", exist_ok=True)
```

## Шаг 7. Напиши обработчики

Каждый тип контента — отдельная функция:
- `save_text()` — текстовые сообщения → `Распределение/сообщения/`
- `save_voice()` — голосовые → скачать → транскрибировать → `Распределение/транскрипты/`
- `save_document()` — документы → `Распределение/документы/`
- `save_photo()` — фото → `Распределение/изображения/`

## Шаг 8. Подключи транскрибацию

1. Зарегистрируйся на assemblyai.com
2. Скопируй API Key
3. В `transcriber.py`: отправляешь аудио → получаешь текст
4. Бесплатно первые 5 часов аудио

## Шаг 9. Добавь команды

| Команда | Что делает |
|---------|-----------|
| `/start` | Приветствие + что бот умеет |
| `/help` | Список команд |
| `/status` | Путь к папке + количество файлов |
| `/sync` | Статистика обработки |
| `/log` | Последние 10 записей |
| `/list` | Последние сохранённые файлы |

## Шаг 10. Протестируй

1. Запусти: `python bot.py`
2. Отправь текст → проверь `Распределение/сообщения/`
3. Отправь голосовое → проверь `Распределение/транскрипты/`
4. Отправь файл → проверь `Распределение/документы/`
5. Отправь фото → проверь `Распределение/изображения/`
