# Делаем Телеграм-бота в Cursor AI без знания кода
Источник: https://habr.com/ru/articles/874220/

## Содержание
1. Проблема и возможное решение
2. Установка Cursor и регистрация
3. Готовимся к первому промпту
4. Пишем первый промпт
5. Тестируем
6. Исправляем ошибки
7. Добавляем весь функционал
8. Деплой

## Проблема
Автор хотел ленту новостей из Telegram-каналов, разбитую по категориям (папкам).

## Шаг 1. Установка
- Скачать Cursor с cursor.com
- 14 дней тестового Pro (250 генераций на Claude)
- Можно регистрироваться с новой почты (но привязка к IP)
- Pro: $20/мес

## Шаг 2. Подготовка
- Показать Cursor документацию Telegram API
- Добавить @Codebase для контекста

## Шаг 3-5. Промпт → Тест → Исправление
- Описать что хочешь → Accept → запустить → скопировать ошибки → отправить AI → повторить
- Это самая долгая фаза: AI часто ошибается

## Шаг 6. Добавление функционала
- Поэтапно: фича → тест → исправление → следующая фича
- "Работает — не трогай"

## Шаг 7. Деплой
- Railway ($5 на счёт, ~40 центов/неделя)
- Интеграция с GitHub для автодеплоя
- Репо: https://github.com/mikhailmurzak/tg-smart-folders-bot

## Итого
~6 часов на всё. Второй проект можно за 2 часа.
