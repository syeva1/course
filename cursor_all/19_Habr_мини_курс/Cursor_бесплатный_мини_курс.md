# Cursor запустил бесплатный мини-курс для разработчиков
Источник: https://habr.com/ru/companies/bothub/news/952022/

Cursor выпустил бесплатный мини-курс для разработчиков, использующих AI.

## Что внутри
- Как устроены современные LLM-модели
- Какие задачи они решают
- Практические приёмы генерации кода
- Примеры промптов
- Методы оптимизации запросов
- Стратегии интеграции AI в рабочие процессы
- Интерактивные задания и тесты

## Формат
- Компактный — можно пройти за один вечер
- Подходит и новичкам, и опытным разработчикам
