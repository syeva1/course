# Cursor for Vibe Coding: A Complete Guide
Source: https://youware.medium.com/cursor-for-vibe-coding-a-complete-guide-b0863d4a2330

## What is Vibe Coding?
AI-assisted programming where you describe a problem in natural language and AI generates code.
Andrej Karpathy: "It's not really coding — I just see stuff, say stuff, run stuff, and copy-paste stuff."

## Cursor Features
- AI-Powered Code Generation (natural language → code)
- Automated Debugging Assistance
- Context-Aware Help (understands your project)
- Documentation and Learning Tools

## Tutorial Steps
1. Download from cursor.com
2. Sign up (free tier available)
3. Cmd+L (chat) or Cmd+K (inline prompt)
4. Describe what you want in plain English
5. Review generated code in diff view
6. Accept/reject, then test

## Best Practices
- Describe intent clearly and specifically
- Break project into smaller tasks
- Review and test often
- Refine prompts based on results
- Use Cursor's guidance features (explain code, suggest improvements)
- Be mindful of limitations — complex tasks may need iteration
- Learn by doing — recognize patterns over time
