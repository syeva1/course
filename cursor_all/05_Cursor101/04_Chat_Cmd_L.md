# Cursor Tutorial 3 — Master Chat (Cmd L)

## When to Use Chat
- Learn concepts, frameworks
- Get code advice and best practices
- Debug and troubleshoot

## Key Features
- Apply: use AI suggestions directly, review in diff view
- @ Context references (same as Cmd K but richer)

## Internal Context (small → large)

## External Context
- @Doc — reference documentation
- @Web — search online
- @URL — specific pages
