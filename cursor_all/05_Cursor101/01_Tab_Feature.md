# Cursor Tutorial 1 — Using the Tab Feature
Source: https://cursor101.com/tutorial/learn-cursor-tab

Three main AI features: Tab, Cmd K (prompt bar), Cmd L (chat)

## Basic Tab Features
- Multi-line Completion: generates code suggestions at cursor position
- Multi-line Editing: suggests edits across multiple lines

## Advanced Tab Features
- Smart Rewrite: intelligently rewrites existing code
- Cursor Prediction: predicts where you want to edit next (e.g., renaming a variable → suggests renaming everywhere)

## Tips
- Tab works in Peek windows (Go to Definition)
- Partial Accepts: Ctrl+→ to accept word by word
