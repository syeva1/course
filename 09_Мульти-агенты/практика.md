# Модуль 09 — Практика

## Задание: создай свою мульти-агентную систему

### Цель

Построить цепочку из 3–5 агентов для решения реальной задачи вашего бизнеса.

### Шаг 1: Выберите задачу

Примеры:
- Создание контента (статьи, посты, рассылки)
- Подбор кандидатов (поиск → скрининг → оценка)
- Анализ конкурентов (сбор → анализ → отчёт)
- Email-маркетинг (сегментация → текст → проверка)

### Шаг 2: Определите агентов

Для каждого агента опишите:
- **Роль** — что он делает
- **Вход** — какой файл читает
- **Выход** — какой файл пишет
- **Промпт** — инструкция агенту

Пример (контент):

| # | Агент | Вход | Выход |
|---|-------|------|-------|
| 1 | Планировщик | тема.txt | plan.md |
| 2 | Проверщик плана | plan.md | plan-review.md |
| 3 | Писатель | plan.md + plan-review.md | draft.md |
| 4 | Редактор | draft.md | text-review.md |
| 5 | Финализатор | draft.md + text-review.md | final.md |

### Шаг 3: Напишите промпты

Для каждого агента создайте файл `prompts/agent-N.md`:

```markdown
# Агент: Планировщик

## Роль
Ты — контент-планировщик. Твоя задача — создать детальный план статьи.

## Вход
Прочитай файл `тема.txt` — там тема и целевая аудитория.

## Выход
Создай файл `plan.md` со структурой:
- Заголовок
- Тезисы (3–5)
- Ключевые сообщения
- CTA
```

### Шаг 4: Запустите и проверьте

1. Запустите первого агента вручную
2. Проверьте выходной файл
3. Передайте следующему агенту
4. Повторите до конца цепочки

### Шаг 5: Автоматизируйте

Когда цепочка работает вручную — автоматизируйте через скрипт или ccpm.

### Критерии успеха

- [ ] Цепочка из 3+ агентов
- [ ] Каждый агент имеет чёткий промпт
- [ ] Файлы-входы и файлы-выходы определены
- [ ] Есть хотя бы один проверщик
- [ ] Цепочка протестирована на реальной задаче
