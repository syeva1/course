# Модуль 09 — Система мульти-агентов

## 🎯 О чём модуль

Как перейти от одного AI-агента к команде агентов, которые работают вместе: планируют, пишут, проверяют и масштабируются.

## 📂 Структура

| Файл | Описание |
|------|----------|
| `презентация.html` | Интерактивная презентация (17 слайдов) |
| `инструкция.md` | Пошаговая инструкция к модулю |
| `практика.md` | Практическое задание |
| `шпаргалка.md` | Краткая шпаргалка по мульти-агентам |
| `материалы/схема_мульти_агентов.md` | Схема архитектуры |
| `материалы/шаблон_цепочки.md` | Шаблон для создания своей цепочки |

## 🔑 Ключевые темы

- CEO-агент и оркестрация
- Content Factory: 5 агентов в цепочке
- Параллельная работа агентов
- ccpm и agtx — инструменты управления
- Промежуточные файлы и логирование
- Масштабирование от 2 до 30 агентов
- Agent Lightning — автоулучшение промптов

## ⏱ Время

~45 минут (презентация + практика)
