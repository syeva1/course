# Модуль 08 — Инструкция

## Как работать с этим модулем

### 1. Посмотри презентацию
Открой `презентация.html` в браузере. Используй стрелки ← → для навигации.

### 2. Получи API-ключи

**Claude API:**
1. Зайди на [console.anthropic.com](https://console.anthropic.com)
2. Зарегистрируйся и подтверди email
3. Перейди в Settings → API Keys → Create Key
4. Скопируй ключ (начинается с `sk-ant-`)

**OpenAI API:**
1. Зайди на [platform.openai.com](https://platform.openai.com)
2. Зарегистрируйся, пополни баланс
3. Перейди в API Keys → Create new secret key

### 3. Настрой окружение

```bash
# Установи Python-библиотеки
pip install anthropic openai python-dotenv requests gspread

# Создай .env файл
echo 'ANTHROPIC_API_KEY=sk-ant-...' > .env
echo 'OPENAI_API_KEY=sk-...' >> .env
```

### 4. Сделай первый запрос
Запусти пример из `материалы/пример_api_запроса.py`.

### 5. Создай Telegram-бота
1. Найди @BotFather в Telegram
2. Отправь `/newbot`
3. Придумай имя и username
4. Получи токен

### 6. Выполни практику
Открой `практика.md` и выполни задания по порядку.
