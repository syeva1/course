Что такое API?

Официант между тобой и кухней

Простая аналогия

  • Ты (клиент) — хочешь блюдо
  • Кухня (сервер) — готовит блюдо
  • Официант (API) — принимает заказ, несёт результат
  • Меню (документация) — что можно заказать

Техническое определение

  • API — Application Programming Interface
  • Интерфейс, через который программы общаются друг с другом
  • Запрос → Ответ (обычно в формате JSON)
  • REST API — самый популярный тип

Claude API — подключаем AI

Получение ключа и первый запрос
  • 1 Зарегистрируйся на console.anthropic.com
  • 2 Перейди в Settings → API Keys → Create Key
  • 3 Скопируй ключ и сохрани в .env файл
  • 4 Сделай первый запрос через Python или curl
import anthropic client = anthropic.Anthropic(api_key="sk-ant-...") message = client.messages.create( model="claude-sonnet-4-20250514", max_tokens=1024, messages=[{"role": "user", "content": "Привет!"}] ) print(message.content[0].text)

OpenAI API — альтернатива

GPT-4, GPT-4o, o1 и другие модели
  • 1 Зарегистрируйся на platform.openai.com
  • 2 Получи API-ключ в разделе API Keys
  • 3 Пополни баланс (pay-as-you-go)
from openai import OpenAI client = OpenAI(api_key="sk-...") response = client.chat.completions.create( model="gpt-4o", messages=[{"role": "user", "content": "Привет!"}] ) print(response.choices[0].message.content)

MCP — Model Context Protocol

Новый стандарт подключения AI к инструментам

Что это?

  • Открытый протокол от Anthropic
  • Позволяет AI-модели подключаться к внешним инструментам
  • Файловая система, БД, API — через единый интерфейс
  • Работает в Cursor, Claude Desktop и других клиентах

Зачем?

  • AI получает доступ к актуальным данным
  • Не нужно вручную копировать контекст
  • Стандартизация — один протокол вместо десятков
  • Безопасность — контролируемый доступ

RAG — AI + твои данные

Retrieval Augmented Generation

Как работает

  • Твои документы разбиваются на части
  • Каждая часть превращается в вектор (эмбеддинг)
  • При вопросе — поиск релевантных частей
  • AI отвечает на основе найденных данных

Применение

  • 📚 База знаний компании
  • 💬 Чат-бот по документации
  • 📋 Ответы по регламентам и инструкциям
  • 🔍 Поиск по внутренним файлам

Webhook

Когда сервис сам стучится к тебе

API vs Webhook

  • API — ты спрашиваешь, сервер отвечает (pull)
  • Webhook — сервер сам присылает данные (push)
  • Webhook = «обратный звонок»

Примеры

  • 💳 Оплата прошла → уведомление
  • 📝 Новая заявка на сайте → данные в CRM
  • 💬 Сообщение в Telegram → обработка ботом
  • 📦 Статус доставки изменился → email клиенту

No-code альтернативы

Zapier, Make, n8n

Когда использовать

  • Нужна быстрая интеграция без кода
  • Простые сценарии: «если X → сделай Y»
  • Прототипирование перед полноценной разработкой

Популярные платформы

  • Zapier — простой, 6000+ интеграций
  • Make (ex-Integromat) — гибкий, визуальный
  • n8n — open-source, можно хостить самому
  • Albato — российская альтернатива

ВАЖНО No-code — отлично для старта, но код даёт полный контроль

Безопасность: токены и .env

Не хардкодь секреты!
  • 🔐 Никогда не храни API-ключи в коде
  • 📁 Используй файл .env для секретов
  • 🚫 Добавь .env в .gitignore
  • 🔄 Регулярно ротируй ключи
  • 👀 Мониторь использование API (лимиты, биллинг)
# .env файл ANTHROPIC_API_KEY=sk-ant-api03-... OPENAI_API_KEY=sk-proj-... TELEGRAM_BOT_TOKEN=123456:ABC-DEF... # Python from dotenv import load_dotenv import os load_dotenv() api_key = os.getenv("ANTHROPIC_API_KEY")
