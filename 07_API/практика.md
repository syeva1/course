## Задание 2: Telegram-бот для уведомлений

**Цель:** Создать бота и отправить себе сообщение.

1. Создай бота через @BotFather
2. Получи свой chat_id (напиши боту, затем проверь через `https://api.telegram.org/bot<TOKEN>/getUpdates`)
3. Напиши скрипт, который отправляет сообщение «🔔 Тестовое уведомление от АССИСТ+»

```python
import requests
import os
from dotenv import load_dotenv

load_dotenv()
TOKEN = os.getenv("TELEGRAM_BOT_TOKEN")
CHAT_ID = os.getenv("TELEGRAM_CHAT_ID")

url = f"https://api.telegram.org/bot{TOKEN}/sendMessage"
requests.post(url, json={"chat_id": CHAT_ID, "text": "🔔 Тестовое уведомление от АССИСТ+!"})
```

**Результат:** Скриншот сообщения от бота в Telegram.

## Задание 3: Google Sheets — запись данных

3. Создай Google Таблицу и расшарь на email сервисного аккаунта
4. Запиши строку данных:

```python
import gspread

gc = gspread.service_account(filename="credentials.json")

ws = sh.sheet1
ws.append_row(["Иван Петров", "Ищу ассистента", "2025-03-09", "Новая"])
```
