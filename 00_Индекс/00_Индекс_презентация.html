XX_Название/

├── XX_Название_презентация.html  ← Главные слайды
├── инструкция.md                  ← Делай по шагам
├── XX_практика.html               ← Задания

README.md — всегда начинай с него. Там описание модуля и чему научишься.

Две папки в материалах

📖 Для изучения

HTML-презентации по каждой теме. Открой в браузере — листай как слайды. Удобно читать с телефона или компьютера. Это для твоего обучения.

📁 Для Cursor

Те же материалы, но в формате .md (Markdown). Это для загрузки в Cursor.

  • Перетащи файл прямо в чат агента
  • Или положи в папку проекта
  • Cursor прочитает и будет использовать как базу знаний

Пример: Перетащил шаблон_cursorrules.md → сказал «Настрой правила для моего проекта» → агент сделает

Как проходить модуль

1

Прочитай README.md — описание модуля, чему научишься, сколько времени

2

Посмотри презентацию — открой HTML-файл в браузере, листай слайды

3

Пройди инструкцию — открой инструкция.md, делай шаг за шагом прямо в Cursor

4

Изучи материалы — 📖 читай в браузере, 📁 загружай в Cursor

5

Сделай практику — 5 заданий: 2 по инструкции, 2 сам, 1 творческое

В каком порядке проходить

Строго по порядку — каждый модуль опирается на предыдущий.

  • 01–02 — фундамент: установка + настройка под себя
  • 03–04 — инструменты + агенты (основная работа)
  • 05–06 — практика: боты + API (реальные проекты)
  • 07 — вайб-кодинг (создание продуктов с нуля)
  • 08 — справочник (возвращайся когда нужно)

Модуль 08 — не обязательно проходить подряд. Это шпаргалка, к которой возвращаешься по необходимости.

Советы перед стартом

  • Не пропускай модули — от простого к сложному
  • Делай практику — без неё знания не закрепятся
  • Загружай файлы в Cursor — пусть AI знает то же, что и ты
  • Не бойся ошибок — Cursor всегда поможет исправить
  • Экспериментируй — меняй промпты, пробуй разные подходы
