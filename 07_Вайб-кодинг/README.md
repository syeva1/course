> Курс АССИСТ+ | От идеи до рабочего продукта за один вечер

## Что внутри

- **презентация.html** — 18 слайдов: типы продуктов, процесс, примеры, деплой
- **инструкция.md** — пошаговая инструкция
- **практика.md** — 3 практических задания
- **шпаргалка.md** — быстрый справочник
- **материалы/** — шаблоны и примеры
  - `шаблон_лендинга.md` — описание для Cursor
  - `шаблон_readme.md` — README для нового проекта
  - `деплой_github_pages.md` — пошаговый деплой

## Что такое вайб-кодинг

Ты описываешь что хочешь — AI пишет код. Ты проверяешь — просишь доработать. Никакого программирования в классическом смысле.

## Что можно создать

- Лендинги и сайты
- Калькуляторы и квизы
- Telegram-боты
- Chrome-расширения
- Парсеры и автоматизации
- Бизнес-дашборды

## Как начать

1. Откройте `презентация.html` в браузере
2. Следуйте `инструкция.md`
3. Выполните задания из `практика.md`
