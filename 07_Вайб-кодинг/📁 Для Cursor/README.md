# Модуль 07. Вайб-кодинг — создаём продукты

Создаём реальные продукты с помощью AI: лендинги, калькуляторы, парсеры, Chrome-расширения, дашборды. От идеи до деплоя за один вечер.

## Чему научишься

- Превращать идею в работающий продукт через Cursor
- Использовать паттерн README → Plan → Agent → проверка → commit
- Делать адаптив для мобилки
- Деплоить на GitHub Pages, Vercel, VPS
- Декомпозировать большие проекты

## Файлы модуля

- **презентация.html** — слайды (открой в браузере)
- **инструкция.md** — пошаговый гайд
- **практика.md** — 5 заданий от простого к сложному
- **материалы/** — шаблоны, гайды по деплою

## Порядок прохождения

1. Посмотри презентацию
2. Пройди инструкцию — создай свой первый продукт
3. Выполни задания из практики

## Время

~3 часа на первое прохождение.

## Материалы

- `шаблон_лендинга.md` — описание для генерации лендинга
- `шаблон_readme.md` — README для нового проекта
- `деплой_github_pages.md` — пошаговый деплой на GitHub Pages
- `деплой_vps.md` — деплой на VPS с systemd
- `большие_проекты_декомпозиция.md` — как разбивать большие задачи
