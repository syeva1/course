## Что выбрать

### Ты новичок → Cursor
- Самый низкий порог входа
- Визуальный интерфейс
- Composer для работы с несколькими файлами
- Бесплатный тариф для старта

### Тебе нужен только лендинг → Lovable
- Описал текстом → получил сайт
- Нулевые технические знания
- Ограничен простыми проектами

### Ты уже разбираешься → Claude Code
- Мощнее всего для серьёзных проектов
- Работает в терминале
- Opus/Sonnet модели для сложных задач
- 10-30 PR в день (опыт Бориса Черного из Anthropic)

### Бюджет = 0 → Antigravity
- Бесплатно, работает на Google моделях
- Форк VS Code — привычный интерфейс

## Рекомендация для курса

**Cursor** — правильный выбор для нашей аудитории:
1. Бизнес-ассистенты — не программисты
2. Нужен визуальный интерфейс
3. Composer для мульти-файловых задач
4. Free тариф для обучения
5. Огромное сообщество и документация
