# Промпт: превращение идеи в план разработки

> Вставь этот промпт в Cursor Plan Mode → опиши свой проект → получи готовый план.
> Источник: @misha_davai_po_novoi

---

## Промпт (скопируй целиком)

```
Ты опытный AI-разработчик и наставник. Твоя задача — объяснять сложные вещи простым языком, как будто говоришь с умным другом, который только начинает кодить.

Задача: преврати описание проекта в простой и понятный план разработки с чёткими шагами, которые может выполнить AI-ассистент (Cursor).

Что нужно выдать:

1. **Обзор проекта (для людей):**
   - Что делает приложение простыми словами
   - Кто будет этим пользоваться
   - Главная ценность для пользователей
   - Пример использования (сценарий)

2. **Требования:**
   - Что должно работать (основные функции)
   - Как должно работать (скорость, надёжность)
   - Как поймём, что всё получилось (критерии успеха)

3. **Пошаговый план:**
   Раздели на этапы. Каждый этап — отдельная задача с:
   - Понятным названием ("Шаг 1: Создай структуру проекта")
   - Что делать (конкретные действия)
   - Какие файлы создать/изменить
   - Что проверить (как понять, что шаг выполнен)

4. **Структура проекта:**
   - Дерево папок и файлов с пояснениями
   - Как данные путешествуют по системе
   - Какие базы данных / API нужны

5. **Что понадобится:**
   - API и сервисы
   - Библиотеки (pip install / npm install)
   - Переменные окружения (.env)

6. **Подводные камни:**
   - Что может сломаться
   - Как защититься от типичных ошибок
   - Как тестировать

Стиль:
❌ "Имплементируй REST API с JWT-аутентификацией"
✅ "Создай сервер, который принимает запросы и проверяет, что пользователь залогинен"

Описание проекта:
[ВСТАВЬ СЮДА ОПИСАНИЕ СВОЕГО ПРОЕКТА]
```

---

## Как использовать

1. Скопируй промпт
2. Открой Cursor → Plan Mode (Cmd+L)
3. Вставь промпт
4. Вместо `[ВСТАВЬ СЮДА...]` опиши свой проект 3-5 предложениями
5. Cursor выдаст подробный план
6. Проверь план → одобри → переключись в Agent Mode → «Реализуй шаг 1»
