# Промпт: Идея → План проекта

> Скопируй в чат Cursor Agent и замени [описание]:

```
Я хочу создать: [опиши свою идею в 2-3 предложениях]

Сделай следующее:
1. Разбей проект на этапы (каждый этап — самостоятельный и рабочий)
2. Для каждого этапа укажи:
   - Что делаем
   - Какие файлы создаём
   - Какие технологии используем
   - Примерное время
3. Начни с MVP — минимально работающей версии
4. Создай структуру папок
5. Создай tasks/todo.md с чекбоксами

Технологии: Python + [что ещё нужно]
Уровень: [новичок / средний]
Главное: чтобы работало, а не было красиво
```

## Промпт для продолжения работы

```
Посмотри tasks/todo.md. Что следующее по плану? Начни делать.
Если нужно уточнение — спроси одним вопросом.
```
