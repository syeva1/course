# Инструкция — Модуль 07: Вайб-кодинг

## Шаг 1. Выбери продукт

Определись, что хочешь создать:
- Лендинг для бизнеса / услуги
- Калькулятор стоимости
- Парсер данных
- Chrome-расширение
- Дашборд с метриками

## Шаг 2. Напиши README.md

Создай папку проекта и напиши README:
- Что это за продукт (1-2 предложения)
- Для кого
- Какие функции
- Какие технологии (HTML/CSS/JS, Python, и т.д.)
- Как должен выглядеть (цвета, стиль)

Используй шаблон из `материалы/шаблон_readme.md`.

## Шаг 3. Открой в Cursor

1. File → Open Folder → твоя папка
2. Cursor прочитает README и поймёт контекст

## Шаг 4. Plan Mode

1. Открой Agent Side Bar (левая панель)
2. Включи Plan Mode (иконка плана)
3. Напиши: «Составь план реализации проекта по README»
4. Cursor предложит структуру файлов и порядок работы
5. Проверь план — если ОК, одобри

## Шаг 5. Agent Mode — первый шаг

1. Переключись в Agent Mode
2. Попроси реализовать ПЕРВЫЙ шаг плана
3. Например: «Создай базовую HTML-страницу с шапкой и героем»
4. Cursor создаст файлы

## Шаг 6. Проверь

1. Открой результат в браузере (для HTML) или запусти (для Python)
2. Работает? → Следующий шаг
3. Баг? → Скажи Cursor что не так

## Шаг 7. Следующий шаг

Повтори шаги 5-6 для каждой функции:
- «Добавь секцию услуг с 3 карточками»
- «Добавь форму обратной связи»
- «Сделай адаптив для мобилки»

**Правило: один запрос = одна функция.**

## Шаг 8. Коммить

После каждого рабочего шага:
```bash
git add .
git commit -m "Добавил секцию услуг"
```

Если сломаешь — можно откатиться.

## Шаг 9. Финальная проверка

- Открой на телефоне (адаптив)
- Проверь все кнопки и ссылки
- Попроси кого-то протестировать

## Шаг 10. Деплой

Выбери платформу:
- **GitHub Pages** — бесплатно, для HTML/CSS/JS
- **Vercel** — бесплатно, для React/Next.js
- **VPS** — от 300 р/мес, для ботов и серверного кода

Пошаговый гайд в `материалы/деплой_github_pages.md`.
