# Практика — Модуль 07: Вайб-кодинг

## Задание 1. Лендинг для бизнеса

**Цель:** создать одностраничный сайт за один вечер.

1. Выбери бизнес (свой или придуманный)
2. Напиши README: название, услуги, стиль, цвета
3. Открой в Cursor → Plan Mode → Agent Mode
4. Создай 5 секций:
   - Герой-блок с заголовком и кнопкой
   - 3 карточки услуг
   - Отзывы клиентов (можно придуманные)
   - Форма обратной связи
   - Контакты и соцсети
5. Проверь на мобилке
6. Задеплой на GitHub Pages

**Результат:** рабочий лендинг по ссылке username.github.io/project.

---

## Задание 2. Калькулятор стоимости

**Цель:** сделать интерактивный калькулятор для клиентов.

1. Определи, что считаешь (стоимость услуги, доставки, подписки)
2. Выбери 3-5 параметров (тип, количество, срочность и т.д.)
3. Придумай формулу расчёта
4. Напиши README → Cursor создаст калькулятор
5. Добавь кнопку «Оставить заявку» → ссылка на Telegram
6. Встрой на лендинг (из задания 1)

**Результат:** калькулятор, который считает стоимость и собирает заявки.

---

## Задание 3. Свой продукт

**Цель:** создать продукт, который решает реальную проблему.

1. Подумай: что ты делаешь руками каждый день?
   - Собираешь данные вручную? → парсер
   - Отвечаешь на одни вопросы? → бот
   - Сверяешь таблицы? → автоматизация
2. Напиши ТЗ в свободной форме
3. Создай README → открой в Cursor
4. Plan Mode → Agent Mode → тестируй
5. Задеплой и используй

**Результат:** рабочий инструмент, который экономит тебе время.
