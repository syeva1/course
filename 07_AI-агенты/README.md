# Модуль 07 — AI-агенты

> Цифровая рабочая сила для вашего бизнеса

## О модуле

В этом модуле вы узнаете, как создавать AI-агентов — цифровых сотрудников, которые работают автономно, помнят контекст и улучшаются с каждой итерацией.

## Чему научитесь

- Разница между промптом и агентом
- Как создать агента через Cursor Rules и CLAUDE.md
- Скиллы как переиспользуемые компоненты (тренд 2026)
- Типы агентов: ассистент, аналитик, копирайтер
- Цикл улучшения агентов через lessons.md
- Библиотека из 61 готового агента
