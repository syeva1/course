## Задание 2. Telegram-уведомление из Python

**Цель:** отправить себе сообщение через Telegram Bot API.

1. Создай бота через @BotFather
2. Получи свой CHAT_ID:
   - Напиши боту любое сообщение
   - Открой `https://api.telegram.org/bot{ТОКЕН}/getUpdates`
   - Найди `"chat":{"id":ЧИСЛА}` — это CHAT_ID
3. Напиши скрипт:

```python
import requests, os
from dotenv import load_dotenv

load_dotenv()
TOKEN = os.getenv("TELEGRAM_BOT_TOKEN")
CHAT_ID = os.getenv("TELEGRAM_CHAT_ID")

url = f"https://api.telegram.org/bot{TOKEN}/sendMessage"
requests.post(url, json={
    "chat_id": CHAT_ID,
    "text": "<b>Тест!</b>\nAPI работает 🎉",
    "parse_mode": "HTML"
})
```

4. Запусти → проверь Telegram
5. Модифицируй: добавь текущую дату и время в сообщение

**Результат:** скриншот сообщения от бота в Telegram.

## Задание 3. Google Sheets — запись данных

**Цель:** записать данные в Google Таблицу из Python.

1. Создай сервисный аккаунт в Google Cloud Console (см. инструкцию, шаг 9)
2. Скачай `credentials.json`
3. Создай Google Таблицу «Заявки» с колонками: Имя | Телефон | Запрос | Дата | Статус
4. Расшарь таблицу на email сервисного аккаунта
5. Запиши строку данных:

```python
import gspread
from datetime import datetime

gc = gspread.service_account(filename="credentials.json")
sh = gc.open("Заявки")
ws = sh.sheet1

ws.append_row([
    "Иван Петров",
    "+7 999 123-45-67",
    "Ищу бизнес-ассистента",
    datetime.now().strftime("%Y-%m-%d %H:%M"),
    "Новая"
])
print("Строка добавлена!")
```

6. Проверь таблицу — строка должна появиться

**Бонус:** прочитай все строки из таблицы и выведи на экран:
```python
records = ws.get_all_records()
for r in records:
    print(f"{r['Имя']} — {r['Статус']}")
```

**Результат:** скриншот таблицы с добавленной строкой.
