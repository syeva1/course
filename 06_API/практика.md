## Задание 3. AI-обработка текста (сделай сам)

Создай скрипт, который:
1. Принимает текст из файла `input.txt`
2. Отправляет его в Claude API с промптом: «Сделай краткое резюме этого текста в 3 пунктах»
3. Записывает результат в `output.txt`
4. Выводит в консоль: «Готово! Результат в output.txt»

Подготовь `input.txt` — вставь туда любой длинный текст (статью, описание, документ).

Промпт для Cursor напиши сам. Не копируй.

Результат: Скрипт обрабатывает текст через AI и сохраняет результат.

## Задание 4. Связка двух сервисов (сделай сам)

Создай скрипт, который связывает 2 любых сервиса. Варианты:

**Вариант A: AI → Telegram**
- Скрипт генерирует через Claude 3 идеи для поста
- Отправляет их тебе в Telegram

**Вариант B: Файл → AI → Файл**
- Скрипт читает CSV с отзывами клиентов
- AI классифицирует каждый отзыв (позитивный/негативный/нейтральный)
- Результат записывает в новый CSV

**Вариант C: AI → Google Sheets**
- Скрипт генерирует через AI контент-план на неделю
- Записывает в Google Таблицу (день | тема | формат)

Выбери один вариант и реализуй.

Результат: Работающая связка из двух сервисов.
