# Модуль 06 — API и интеграции

> Курс АССИСТ+ | Подключаем AI, таблицы, мессенджеры и платежи через API

## Что внутри

- **презентация.html** — 23 слайда: от основ API до связки Telegram + AI + Google Sheets
- **инструкция.md** — пошаговая инструкция с получением ключей
- **практика.md** — 4 практических задания (от простого к связке сервисов)
- **шпаргалка.md** — справочник: коды, примеры, чеклисты
- **материалы/** — готовый код и справочники
  - `пример_api_запроса.py` — первый запрос к Claude API
  - `пример_связки.py` — Telegram + AI + Google Sheets
  - `список_api.md` — каталог полезных API для бизнеса
  - `гайд_MCP.md` — подключение MCP-серверов в Cursor

## Ключевые темы

- Что такое API, JSON, HTTP-методы
- API-ключи, .env, безопасность
- Claude API и OpenAI API — реальный код
- Telegram API — уведомления из Python
- Google Sheets API — чтение и запись данных
- Webhooks — сервис сам тебе пишет
- MCP — удлинитель для Cursor (горячая тема 2026)
- RAG — AI, который знает твой бизнес
- No-code: Zapier, Make, n8n, Albato
- Стоимость использования API

## Как начать

1. Откройте `презентация.html` в браузере
2. Следуйте `инструкция.md` для получения API-ключей
3. Запустите примеры из `материалы/`
4. Выполните задания из `практика.md`
