# Модуль 08 — API и интеграции

## Описание

Этот модуль посвящён работе с API — как подключать AI-модели, сервисы и инструменты друг к другу. Вы научитесь делать запросы к Claude и OpenAI API, отправлять уведомления в Telegram, автоматизировать Google Таблицы и строить связки сервисов.

## Содержание

| Файл | Описание |
|------|----------|
| `презентация.html` | Интерактивная презентация (15 слайдов) |
| `инструкция.md` | Пошаговая инструкция к модулю |
| `практика.md` | Практические задания |
| `шпаргалка.md` | Краткая шпаргалка по API |
| `материалы/пример_api_запроса.py` | Пример запроса к Claude API |
| `материалы/список_api.md` | Список полезных API для бизнеса |

## Темы

- Что такое API и зачем оно бизнесу
- Claude API и OpenAI API
- MCP (Model Context Protocol)
- RAG (Retrieval Augmented Generation)
- Google Sheets API, Telegram API, Webhooks
- Связки сервисов и no-code альтернативы
- Безопасность: токены, .env, ротация ключей
