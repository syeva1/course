# Модуль 06. API и интеграции

Подключаем внешние сервисы через API — AI (Claude, OpenAI), Telegram, Google Sheets. Учимся собирать связки из нескольких сервисов.

## Чему научишься

- Понимать что такое API, ключи, запросы
- Получать API-ключи Claude и OpenAI
- Отправлять уведомления в Telegram через API
- Работать с Google Sheets из кода
- Собирать цепочки: бот → AI → таблица → уведомление

## Файлы модуля

- **презентация.html** — слайды (открой в браузере)
- **инструкция.md** — пошаговый гайд
- **практика.md** — 5 заданий от простого к сложному
- **материалы/** — готовый код, списки API, гайд по MCP

## Порядок прохождения

1. Посмотри презентацию
2. Пройди инструкцию
3. Выполни задания из практики

## Время

~2.5 часа на первое прохождение.

## Материалы

- `пример_api_запроса.py` — первый запрос к Claude API
- `пример_связки.py` — Telegram + AI + Google Sheets
- `список_api.md` — каталог полезных API для бизнеса
- `гайд_MCP.md` — подключение MCP-серверов в Cursor
- `MCP_подключение.md` — конфигурация MCP с примерами JSON
- `готовые_автоматизации/` — 10 готовых скриптов
