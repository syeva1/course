# Инструкция — Модуль 06: API и интеграции

## Шаг 1. Установи библиотеки

```bash
pip install anthropic openai python-dotenv requests gspread
```

## Шаг 2. Создай .env файл

В корне проекта создай файл `.env`:

```
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-proj-...
TELEGRAM_BOT_TOKEN=123456:ABC...
TELEGRAM_CHAT_ID=775745451
ASSEMBLYAI_KEY=a1b2c3d4...
```

**Важно:** добавь `.env` в `.gitignore` до первого коммита!

## Шаг 3. Получи API-ключ Claude

1. Зайди на [console.anthropic.com](https://console.anthropic.com)
2. Зарегистрируйся и подтверди email
3. Перейди в Settings → API Keys → Create Key
4. Скопируй ключ (начинается с `sk-ant-`)
5. Пополни баланс (от $5, хватит на ~500 запросов Sonnet)

## Шаг 4. Получи API-ключ OpenAI

1. Зайди на [platform.openai.com](https://platform.openai.com)
2. Зарегистрируйся
3. Пополни баланс (от $5)
4. API Keys → Create new secret key
5. Скопируй ключ (начинается с `sk-proj-`)

Из РФ: оплата через @PlatiRublemRu_Bot или VPN + иностранная карта.

## Шаг 5. Первый запрос к Claude

Запусти `материалы/пример_api_запроса.py`:

```bash
python материалы/пример_api_запроса.py
```

Если видишь ответ от Claude — всё работает! Измени промпт на что-то полезное для бизнеса.

## Шаг 6. Создай Telegram-бота

1. Открой @BotFather в Telegram
2. Отправь `/newbot`
3. Придумай имя и username
4. Скопируй токен в .env

## Шаг 7. Узнай свой CHAT_ID

1. Напиши что-нибудь своему боту
2. Открой в браузере: `https://api.telegram.org/bot{ТОКЕН}/getUpdates`
3. Найди `"chat":{"id":123456789}` — это твой CHAT_ID
4. Добавь в .env

## Шаг 8. Отправь уведомление

```python
import requests, os
from dotenv import load_dotenv
load_dotenv()

TOKEN = os.getenv("TELEGRAM_BOT_TOKEN")
CHAT_ID = os.getenv("TELEGRAM_CHAT_ID")

url = f"https://api.telegram.org/bot{TOKEN}/sendMessage"
requests.post(url, json={
    "chat_id": CHAT_ID,
    "text": "Привет! Это тестовое уведомление."
})
```

## Шаг 9. Настрой Google Sheets (опционально)

1. Зайди на [console.cloud.google.com](https://console.cloud.google.com)
2. Создай проект → включи Google Sheets API и Google Drive API
3. Создай Service Account → скачай JSON-ключ (credentials.json)
4. Создай Google Таблицу
5. Расшарь таблицу на email сервисного аккаунта (из JSON-файла)

```python
import gspread
gc = gspread.service_account(filename="credentials.json")
sh = gc.open("Название таблицы")
ws = sh.sheet1
ws.append_row(["Тестовая", "строка", "данных"])
```

## Шаг 10. Настрой MCP в Cursor (опционально)

1. В корне проекта создай `.cursor/mcp.json`
2. Добавь конфигурацию MCP-сервера (см. `материалы/гайд_MCP.md`)
3. Перезапусти Cursor (Ctrl+Shift+P → Reload Window)
4. В Agent Mode появятся новые инструменты

## Шаг 11. Собери связку

Теперь у тебя есть все компоненты. Собери цепочку:
- Бот получает сообщение
- AI классифицирует
- Данные в Google Sheets
- Уведомление в Telegram

Используй `материалы/пример_связки.py` как основу.
