# Шпаргалка — API и интеграции

## Термины

| Термин | Что это |
|--------|---------|
| **API** | Интерфейс для общения программ друг с другом |
| **REST API** | Самый популярный тип API (через HTTP-запросы) |
| **JSON** | Формат данных для запросов и ответов |
| **Endpoint** | URL-адрес, куда отправляется запрос |
| **API Key** | Ключ для авторизации (как пароль) |
| **Webhook** | Обратный вызов — сервис сам отправляет данные на твой URL |
| **MCP** | Model Context Protocol — стандарт подключения AI к инструментам |
| **RAG** | Retrieval Augmented Generation — AI + поиск по твоим данным |
| **SDK** | Библиотека для удобной работы с API (не нужно писать запросы вручную) |

## HTTP-методы

| Метод | Действие | Пример |
|-------|----------|--------|
| `GET` | Получить данные | Загрузить список заявок |
| `POST` | Отправить/создать | Создать новую заявку |
| `PUT` | Обновить полностью | Перезаписать данные клиента |
| `PATCH` | Обновить частично | Изменить статус заявки |
| `DELETE` | Удалить | Удалить запись |

## Запрос к Claude API

```python
import anthropic, os
from dotenv import load_dotenv
load_dotenv()

client = anthropic.Anthropic(api_key=os.getenv("ANTHROPIC_API_KEY"))
msg = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=1024,
    system="Ты — помощник.",
    messages=[{"role": "user", "content": "Твой запрос"}]
)
print(msg.content[0].text)
```

## Запрос к OpenAI API

```python
from openai import OpenAI
import os
from dotenv import load_dotenv
load_dotenv()

client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))
resp = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[
        {"role": "system", "content": "Ты — помощник."},
        {"role": "user", "content": "Твой запрос"}
    ]
)
print(resp.choices[0].message.content)
```

## Отправка в Telegram

```python
import requests
TOKEN = os.getenv("TELEGRAM_BOT_TOKEN")
CHAT_ID = os.getenv("TELEGRAM_CHAT_ID")
url = f"https://api.telegram.org/bot{TOKEN}/sendMessage"
requests.post(url, json={"chat_id": CHAT_ID, "text": "Текст", "parse_mode": "HTML"})
```

Файл `.cursor/mcp.json`:
```json

    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {"GITHUB_TOKEN": "ghp_..."}
    }
  }

```
После сохранения: Ctrl+Shift+P → Reload Window.

## Коды ответов HTTP

| Код | Значение | Что делать |
|-----|----------|------------|
| 200 | OK | Всё хорошо |
| 201 | Created | Ресурс создан |
| 400 | Bad Request | Проверь параметры запроса |
| 401 | Unauthorized | Проверь API-ключ |
| 403 | Forbidden | Нет доступа |
| 404 | Not Found | Неверный URL |
| 429 | Too Many Requests | Превышен лимит — подожди |
| 500 | Server Error | Проблема на стороне сервиса |

## Безопасность — чеклист

- [ ] API-ключи в `.env`, не в коде
- [ ] `.env` добавлен в `.gitignore`
- [ ] credentials.json тоже в `.gitignore`
- [ ] Настроены лимиты расходов (OpenAI → Settings → Limits)
- [ ] Мониторинг биллинга включён
- [ ] Ключи ротируются раз в 3-6 месяцев
