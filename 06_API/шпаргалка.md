# Модуль 08 — Шпаргалка

## API — быстрый справочник

### Ключевые термины

| Термин | Что это |
|--------|---------|
| **API** | Интерфейс для общения программ друг с другом |
| **REST API** | Самый популярный тип API (через HTTP-запросы) |
| **JSON** | Формат данных для запросов и ответов |
| **Endpoint** | URL-адрес, куда отправляется запрос |
| **API Key** | Ключ для аутентификации |
| **Webhook** | Обратный вызов — сервер сам отправляет данные |
| **MCP** | Model Context Protocol — стандарт подключения AI к инструментам |
| **RAG** | Retrieval Augmented Generation — AI + поиск по твоим данным |

### HTTP-методы

| Метод | Действие |
|-------|----------|
| `GET` | Получить данные |
| `POST` | Отправить/создать данные |
| `PUT` | Обновить данные |
| `DELETE` | Удалить данные |

### Быстрый запрос к Claude API

```python
import anthropic, os
from dotenv import load_dotenv
load_dotenv()

client = anthropic.Anthropic(api_key=os.getenv("ANTHROPIC_API_KEY"))
msg = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Твой промпт"}]
)
print(msg.content[0].text)
```
