Что это?

MCP — стандарт, который позволяет Cursor подключаться к внешним инструментам: базам данных, API, сервисам. «Удлинитель» для AI — доступ к вещам за пределами кода.

Зачем?

  • Искать в Google/GitHub прямо из чата
  • Читать и писать в базу данных
  • Работать с Figma макетами
  • Запускать тесты в браузере
  • Отправлять сообщения в Telegram/Slack

Как настроить

  • Создай .cursor/mcp.json в корне проекта
  • Перезапусти Cursor (Cmd+Shift+P → Reload Window)
  • Проверь: Agent Mode (Cmd+L) — в списке инструментов появятся новые
