# 📂 Материалы — 06_API

## Что тут лежит

### 📖 Для изучения
HTML-презентации по каждой теме модуля. **Открой в браузере** — удобно листать как слайды.
Это тот же самый текст, что и в папке «Для Cursor», только красиво оформленный.

### 📁 Для Cursor
Markdown-файлы (.md) — **загрузи в Cursor** как контекст.
Перетащи нужный файл прямо в чат агента или положи в папку проекта.
Cursor прочитает файл и будет использовать как базу знаний.

### Как использовать
1. **Изучаешь тему** → открывай файлы из «📖 Для изучения»
2. **Работаешь в Cursor** → загружай файлы из «📁 Для Cursor»
3. **Практика** → открой `06_практика.md` (задания к модулю)
