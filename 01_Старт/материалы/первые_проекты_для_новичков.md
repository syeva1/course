## Проект 1: HTML-отчёт по папке загрузок

У тебя наверняка есть папка загрузок, где скопились сотни файлов. Разбирать вручную — лень. Cursor сделает это за тебя.

### Шаги:
1. Открой Cursor
2. **File → Open Folder** → выбери папку загрузок
3. Открой чат (справа) → режим **Plan**
4. Напиши:

```
Слушай, у меня в папке загрузок полный бардак. Создай HTML-страничку
с отчётом, что тут лежит. Категоризируй всё. Никаких питонов и
скриптов — просто HTML и JavaScript.
```

5. Cursor составит план → одобри → он создаст файл
6. Открой HTML в браузере → готово!

### Дальше — усложняй:
- «Добавь блок с топ-5 самых крупных файлов»
- «Создай папку Вебинар и перенеси туда вот эти два файла» (переключи на Agent)
- Можно даже вставить скриншот с обведёнными файлами!

### Важно:
Фраза «Никаких питонов — просто HTML и JavaScript» — ключевая. Без неё Cursor может предложить Python-скрипт, который сложнее запустить.

## Проект 2: Анализ переписки из Telegram

### Шаги:
1. В десктопном Telegram: три точки → **Экспорт истории чата**
2. Убери галочки с фото и медиа — только текст
3. Выбери период → Экспортировать
4. Создай новую папку → перенеси файл выгрузки
5. Открой эту папку в Cursor
6. Режим **Plan**, напиши:

```
У меня тут выгрузка из телеграм-чата. Сделай HTML-отчёт со
списком авторов и количеством сообщений у каждого.
Без Python — только HTML и JavaScript.
```

7. Одобри план → открой результат в браузере

### Дальше:
- «Выпиши все вопросы из чата и ответы на них»
- «При клике на имя — покажи все его сообщения»
- Вставь картинку-референс → «Переделай дизайн как здесь»

На вебинаре из переписки извлекли **150 вопросов** — 145 с ответами. Вручную это заняло бы часы.

## Что делать, когда не работает

### Тип 1: Ошибка или пустая страница
→ Скриншот ошибки → вставь в чат → «не работает»
Cursor разберётся и исправит.

### Тип 2: Cursor слишком долго думает
→ Останови → переформулируй проще:
```
Ты что-то долго работаешь. Можешь сам проанализировать всю папку,
а отчёт создать и вставить в файл?
```

### Тип 3: Cursor ходит по кругу
→ **Удали чат** → создай новый → начни заново.
Чистая сессия решает проблему накопившихся ошибок.
