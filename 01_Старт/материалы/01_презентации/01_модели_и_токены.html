Что такое модель

Cursor — умный блокнот. Думает модель на сервере: запрос → сервер → ответ → Cursor применяет изменения.

КомпанияМоделиДля чего
AnthropicClaude Sonnet, OpusКод, длинные задачи
OpenAIGPT-5, CodexКод, универсальные задачи
GoogleGeminiАнализ данных, документы
CursorCursor SmallБыстрые простые задачи

Режим Auto — Cursor сам выбирает модель. На старте достаточно.

Токены и контекст

  • Токен ≈ одно слово. Запрос 500 слов + файл 5000 слов = ~5500 токенов на вход; ответ 1000 слов = ~1000 на выход.
  • Контекстное окно — «память» модели. Когда заполняется — модель сжимает историю, детали теряются.
  • Одна задача = один чат. Результаты в файл. Промпты короче.
