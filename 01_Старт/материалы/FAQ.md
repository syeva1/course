# ❓ FAQ — Частые вопросы по Модулю 01

---

### 1. Cursor — это бесплатно?
Есть бесплатный тариф (50 запросов к AI в месяц). Для полноценной работы нужен Pro ($20/мес).

### 2. Нужно ли уметь программировать?
Нет. Cursor — инструмент для тех, кто хочет решать задачи с помощью AI, а не писать код руками.

### 3. Cursor работает на русском?
Да. AI понимает и отвечает на русском. Интерфейс на английском, но это не мешает.

### 4. Чем Cursor отличается от ChatGPT?
ChatGPT — это чат. Cursor — рабочий инструмент. Он видит все файлы проекта, может создавать и редактировать файлы, работать с кодом. ChatGPT — только текст в окне.

### 5. У меня Windows, Cursor не запускается. Что делать?
- Попробуй запустить от имени администратора (правой кнопкой → «Запуск от имени администратора»)
- Временно отключи антивирус
- Проверь, что Windows обновлён

### 6. Файл .cursorrules не виден в Cursor
Файлы, начинающиеся с точки, могут быть скрыты. В Cursor они должны быть видны в Explorer (левая панель). Если нет — проверь, что файл лежит в корне открытой папки.

### 7. AI не читает мой .cursorrules
- Убедись, что файл называется точно `.cursorrules` (без расширения .txt или .md)
- Убедись, что файл в корне проекта (а не в подпапке)
- Попробуй перезапустить Cursor

### 8. AI отвечает на английском
Добавь в .cursorrules строку: `Всегда отвечай на русском языке.`

### 9. AI пишет плохой код / не то, что нужно
Это нормально! Уточни запрос. Чем конкретнее промпт — тем лучше результат. Про это будет целый Модуль 04.

### 10. Как оплатить из России?
См. файл `оплата_из_РФ.md` в этой же папке.

### 11. Могу ли я использовать Cursor на двух компьютерах?
Да. Один аккаунт работает на нескольких устройствах.

### 12. Cursor удалил мой файл!
Всегда делай бэкапы. Cursor может перезаписать файл, если ты нажмёшь Apply. Используй Cmd+Z для отмены.

### 13. Не могу создать файл с точкой в начале на Windows
В проводнике Windows нельзя. Создавай прямо в Cursor: Cmd+N → сохрани как `.cursorrules`.

### 14. Cursor тормозит
- Закрой лишние вкладки
- Перезапусти Cursor
- Проверь интернет (AI работает через облако)

### 15. Чем .cursorrules отличается от системного промпта?
.cursorrules — это файл в проекте, который Cursor автоматически подкладывает как контекст. По сути, это и есть системный промпт, привязанный к проекту.

### 16. Могу ли я иметь разные .cursorrules для разных проектов?
Да! Каждый проект (папка) может иметь свой .cursorrules. Это очень удобно.

### 17. Cursor — это VS Code?
Cursor построен на базе VS Code, но с встроенным AI. Если знаешь VS Code — будет легко. Если нет — тоже не проблема.

### 18. Нужен ли VPN для Cursor?
Обычно нет. Cursor работает из России без VPN. Но если есть проблемы с подключением — попробуй включить VPN.

### 19. Мой первый промпт дал плохой результат
Это нормально! Принцип вайбкодинга: промпт → результат → уточнение → лучше. Никто не получает идеал с первого раза.

### 20. Когда переходить к следующему модулю?
Когда выполнишь все пункты чеклиста в `практика.md`. Не торопись — лучше хорошо разобраться сейчас.
