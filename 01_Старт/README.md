## 🎯 Порядок действий

1. **Открой презентацию** → `презентация.html` (двойной клик в браузере)
2. **Следуй инструкции** → `инструкция.md` (пошагово, со всеми деталями)
3. **Выполни практику** → `практика.md` (проверь себя)
4. **Сохрани шпаргалку** → `шпаргалка.md` (пригодится в следующих модулях)
