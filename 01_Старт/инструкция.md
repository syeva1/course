# 📖 Инструкция: Установка Cursor и первый проект

> **Модуль 01** | Курс АССИСТ+

---

## Шаг 1. Скачать Cursor

1. Открой браузер
2. Перейди на **[cursor.com](https://cursor.com)**
3. Нажми кнопку **Download**
4. Сайт автоматически определит твою систему (Windows / Mac / Linux)
5. Скачай файл

**Что скачается:**
- Windows → файл `.exe` (~130 МБ)
- macOS → файл `.dmg` (~150 МБ)
- Linux → файл `.AppImage`

---

## Шаг 2. Установить

### 🪟 Windows

1. Открой скачанный файл `.exe`
2. Если Windows спрашивает «Разрешить приложению вносить изменения?» — нажми **Да**
3. Если антивирус блокирует — временно отключи его или разреши установку
4. Нажимай **Далее** → **Далее** → **Установить**
5. Дождись завершения
6. Запусти Cursor

⚠️ **ВАЖНО:** Не создавай папки проектов по пути с русскими буквами!
- ❌ `C:\Пользователи\Олеся\Мои проекты\`
- ✅ `C:\Projects\` или `C:\Cursor\`

### 🍎 macOS

1. Открой скачанный файл `.dmg`
2. В появившемся окне перетащи Cursor в папку **Applications**
3. Открой **Applications** → найди **Cursor** → двойной клик
4. Если macOS говорит «Приложение загружено из интернета, открыть?» — нажми **Открыть**
5. Готово!

### 🐧 Linux

1. Сделай файл исполняемым: `chmod +x cursor-*.AppImage`
2. Запусти: `./cursor-*.AppImage`

## Шаг 3. Первый запуск и настройка

При первом запуске Cursor предложит несколько вопросов:

### Импорт настроек
> «Import settings from VS Code?»

- Если ты раньше работал в VS Code → **Yes**
- Если не знаешь что это → **Skip**

### Выбор темы
Тёмная или светлая — выбери какая нравится. Можно поменять потом.

### Keyboard shortcuts
- По умолчанию как в VS Code — просто нажми **Continue**

### Privacy
Cursor спросит про анализ кода для обучения модели:
- Для максимальной конфиденциальности выбери **Privacy Mode**
- Для бизнес-проектов рекомендуем **Privacy Mode**

## Шаг 4. Создать аккаунт

1. Cursor предложит войти — нажми **Sign In**
2. Откроется браузер
3. Выбери способ входа:
   - **Google** (самый простой)
   - **GitHub** (если есть аккаунт)
   - **Email** (регистрация по почте)

5. Вернись в Cursor — ты авторизован

## Шаг 5. Создать первый проект

1. Создай папку на компьютере:
   - Windows: `C:\Projects\МойПроект\`
   - Mac: `~/Projects/МойПроект/`
2. В Cursor: **File → Open Folder** (или **Файл → Открыть папку**)
3. Выбери созданную папку
4. Cursor откроет её как проект

Теперь ты видишь пустую папку в левой панели (Explorer). Всё, что мы создадим — будет здесь.

## Шаг 6. Создать .cursorrules

Это самый важный файл! Он говорит AI кто ты и как с тобой работать.

### Как создать:

1. Открой файл `материалы/шаблон_cursorrules.md`
2. Скопируй всё содержимое
3. В Cursor: нажми **Cmd+N** (Mac) или **Ctrl+N** (Windows) — создастся новый файл
4. Вставь скопированное содержимое
5. **Заполни поля** — замени `[...]` на свои данные:
   - `[Название компании]` → например, «АССИСТ+»
   - `[Ниша]` → например, «HR-услуги»
   - `[Кто ваши клиенты]` → например, «Предприниматели»
   - `[Главная цель]` → например, «Автоматизация рутины»
6. Сохрани файл:
   - **Cmd+S** (Mac) или **Ctrl+S** (Windows)
   - Имя файла: `.cursorrules` (с точкой в начале, без расширения!)
   - Место: корень проекта (главная папка)

### Проверка:

Открой AI Chat (**Cmd+L** или **Ctrl+L**) и напиши:

```
Расскажи, кто я и чем занимаюсь
```

AI должен ответить на основе твоего .cursorrules. Если ответил правильно — всё работает! ✅

## Шаг 7. Первый промпт — создаём страницу

Теперь проверим AI в деле. Открой Chat (**Cmd+L**) и напиши:

```
Создай файл index.html — простую красивую страницу-визитку для моего бизнеса.
Используй информацию из .cursorrules.
Добавь:
- Название компании
- Краткое описание (2-3 предложения)
- Контактную информацию
- Кнопку "Связаться" 
- Современный дизайн с градиентом
```

### Что произойдёт:

1. AI сгенерирует HTML-код
2. Нажми кнопку **Apply** (или **Применить**) — файл создастся в проекте
3. Найди `index.html` в левой панели
4. Кликни правой кнопкой → **Reveal in File Explorer** (Windows) или **Reveal in Finder** (Mac)
5. Открой файл в браузере двойным кликом
6. 🎉 **У тебя есть страница!**

### Как улучшить:

Не понравился результат? Напиши в чат:

```
Сделай дизайн более современным. Используй красный #E63229 как акцентный цвет. 
Добавь секцию с тремя преимуществами компании.
```

AI изменит файл. Нажми **Apply** → обнови страницу в браузере.

**Это и есть вайбкодинг:** описываешь → получаешь → уточняешь → получаешь лучше.
