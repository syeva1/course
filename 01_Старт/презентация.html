Что такое Cursor?

Не редактор кода. Инструмент для управления AI.

Для кого

  • Предприниматели, которые хотят автоматизировать бизнес
  • Ассистенты, которые хотят делать больше за меньше времени
  • Все, кто устал от рутины

Что может

  • Создавать сайты, боты, скрипты — промптами
  • Автоматизировать отчёты, заявки, уведомления
  • Работать с файлами, API, базами данных
  • Понимать контекст всего проекта

Шаг 1. Скачать Cursor

→ cursor.com → Кнопка «Download» → Выбери свою систему
  • 1 Перейди на cursor.com
  • 2 Нажми Download — сайт сам определит Windows / Mac / Linux
  • 3 Скачай файл (.exe для Windows, .dmg для Mac)

ВАЖНО На Windows: не создавай проекты в папках с русскими буквами в пути

Шаг 2. Установка

🪟 Windows

  • 1 Открой скачанный .exe файл
  • 2 Нажимай «Далее» → «Установить»
  • 3 Если антивирус блокирует — разреши установку
  • 4 Запусти Cursor

🍎 macOS

  • 1 Открой скачанный .dmg файл
  • 2 Перетащи Cursor в Applications
  • 3 Запусти из Launchpad
  • 4 Разреши открытие (если macOS спрашивает)

СОВЕТ При первом запуске Cursor предложит импортировать настройки из VS Code — если не знаешь что это, просто пропусти

Шаг 4. Интерфейс — где что

Левая панель

  • Explorer — файлы и папки твоего проекта
  • Search — поиск по всему проекту
  • Source Control — Git (потом пригодится)
  • Extensions — расширения

Главное

  • Центр — редактор файлов
  • Низ — терминал (Ctrl+`)
  • Cmd+L — AI Chat (правая панель)
  • Cmd+K — быстрая команда AI
  • Cmd+I — Composer (мульти-файл)

ЗАПОМНИ Три главные кнопки: Cmd+L (чат) Cmd+K (команда) Cmd+I (composer)

Шаг 5. Первый проект

  • 1 Создай папку на компьютере: МойПроект
    Windows: C:\Projects\МойПроект  |  Mac: ~/Projects/МойПроект
  • 2 В Cursor: File → Open Folder → выбери эту папку
  • 3 Cursor теперь «видит» эту папку как проект
  • 4 Всё, что ты создашь внутри — AI будет понимать как единое целое
МойПроект/
├── .cursorrules     ← цифровой паспорт (следующий шаг)
├── README.md        ← описание проекта
└── ...               ← тут будут твои файлы

Шаг 6. .cursorrules — цифровой паспорт

Файл, который объясняет AI кто ты и как с тобой работать
  • Кто ты — предприниматель, ассистент, маркетолог...
  • Твой бизнес — ниша, клиенты, цели
  • Как отвечать — стиль, язык, структура ответов
  • Правила — не удалять файлы, всегда на русском, и т.д.

КЛЮЧЕВОЕ Без .cursorrules AI — просто чат. С .cursorrules — персональный ассистент, который знает твой бизнес.

Как создать .cursorrules

  • 1 Открой папку материалы/ этого модуля
  • 2 Возьми файл шаблон_cursorrules.md
  • 3 Заполни поля: название бизнеса, ниша, цели
  • 4 Скопируй содержимое
  • 5 В Cursor создай файл .cursorrules в корне проекта
  • 6 Вставь содержимое и сохрани
Проверка: открой Chat (Cmd+L) и спроси:
"Расскажи, кто я и чем занимаюсь"

AI должен ответить на основе твоего .cursorrules ✓

Шаг 7. Первый промпт

Попроси AI сделать что-то реальное

Открой Chat (Cmd+L) и напиши:

Создай файл index.html — простую страницу-визитку для моего бизнеса. Используй информацию из .cursorrules. Добавь: название, описание, контакты, кнопку "Связаться".
  • AI создаст HTML-файл на основе твоего .cursorrules
  • Нажми Apply чтобы сохранить файл
  • Открой index.html в браузере (двойной клик)
  • Поздравляем — первый результат за 2 минуты! 🎉

Шаг 8. Улучшай результат

Не понравилось? Скажи AI, что изменить.

Примеры команд для улучшения:

"Сделай дизайн более современным, добавь градиент"

"Измени цвета на фирменные: красный #E63229 и чёрный"

"Добавь секцию с отзывами клиентов"

"Сделай адаптивным для телефона"

ПРИНЦИП Промпт → Результат → Уточнение → Лучший результат. Это и есть вайбкодинг!
