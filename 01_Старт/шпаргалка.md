## Горячие клавиши Cursor

| Действие | Mac | Windows |
|----------|-----|---------|
| **AI Chat** (главный инструмент) | `Cmd + L` | `Ctrl + L` |
| **Быстрая команда AI** | `Cmd + K` | `Ctrl + K` |
| **Composer** (мульти-файл) | `Cmd + I` | `Ctrl + I` |
| Новый файл | `Cmd + N` | `Ctrl + N` |
| Сохранить | `Cmd + S` | `Ctrl + S` |
| Открыть папку | `Cmd + O` | `Ctrl + O` |
| Терминал | `` Cmd + ` `` | `` Ctrl + ` `` |
| Поиск по файлам | `Cmd + P` | `Ctrl + P` |
| Поиск по проекту | `Cmd + Shift + F` | `Ctrl + Shift + F` |
| Закрыть файл | `Cmd + W` | `Ctrl + W` |

---

## Три главные кнопки

```
Cmd+L  →  💬 Чат с AI (спрашиваешь, AI отвечает и пишет код)
Cmd+K  →  ⚡ Быстрая команда (выделил код → сказал что сделать)
Cmd+I  →  🎼 Composer (работа с несколькими файлами сразу)
```

```

Сначала суть, потом детали.
Мой бизнес: [название], ниша: [ниша].

```

---

## Принцип работы

```
1. Опиши что хочешь  →  промпт в Chat (Cmd+L)
2. Получи результат   →  AI напишет код/текст
3. Примени             →  нажми Apply
4. Проверь             →  открой в браузере / посмотри файл
5. Улучши              →  напиши что изменить
6. Повтори 3-5         →  пока не идеально
```
