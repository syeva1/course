Проект 1: HTML-отчёт по папке загрузок

  • 1 File → Open Folder → папка загрузок
  • 2 Чат справа → режим Plan
  • 3 Напиши промпт (ниже) → одобри план → Cursor создаст файл
  • 4 Открой HTML в браузере
Слушай, у меня в папке загрузок полный бардак. Создай HTML-страничку с отчётом, что тут лежит. Категоризируй всё. Никаких питонов и скриптов — просто HTML и JavaScript.

Проект 2: Анализ переписки из Telegram

  • 1 В Telegram: три точки → Экспорт истории чата (только текст)
  • 2 Создай папку, перенеси файл выгрузки, открой папку в Cursor
  • 3 Режим Plan, напиши промпт (ниже) → одобри → открой в браузере
У меня тут выгрузка из телеграм-чата. Сделай HTML-отчёт со списком авторов и количеством сообщений у каждого. Без Python — только HTML и JavaScript.

Дальше: «Выпиши все вопросы и ответы», «При клике на имя — покажи его сообщения»

Когда не работает и золотое правило

  • Ошибка или пустая страница → скриншот в чат → «не работает»
  • Cursor долго думает → останови → переформулируй проще
  • Ходит по кругу → удали чат, создай новый, начни заново

«Все разговоры про секретные промпты — ерунда. Описывайте, что хотите получить. Не устраивает — объясняйте, что не так. Тупит — говорите ему об этом.» Единственное важное — внятно сформулировать, что нужно.
