Модели и токены

Cursor — блокнот. Думает модель на сервере

  • Claude (Anthropic) — лучший для кода, длинных задач и рефакторинга
  • GPT-5 (OpenAI) — универсальные задачи, хорош для идей
  • Gemini (Google) — анализ данных и длинные документы
  • Токен — единица измерения текста. 1 слово ~ 1.3 токена
  • Контекст — сколько текста модель «помнит». У Claude — до 200K токенов
  • Совет Чем точнее запрос — тем меньше токенов тратишь

Первые проекты

Два проекта за 15 минут — без кода

  • HTML-отчёт по загрузкам — открой папку в Cursor, попроси категоризировать файлы. Результат — красивая HTML-страница
  • Калькулятор для бизнеса — опиши, что считать (например, unit-экономику). AI создаст интерактивную страничку
  • Лайфхак Используй режим Plan, чтобы увидеть план до выполнения
  • Важно Начинай с простых задач — сложность увеличивай постепенно

FAQ

Ответы на частые вопросы

  • Cursor — это бесплатно? — Есть Free-тариф (50 запросов/мес). Для работы нужен Pro ($20/мес)
  • Нужно ли уметь программировать? — Нет. Cursor создан для тех, кто решает задачи через AI
  • Cursor работает на русском? — Да, AI понимает русский. Интерфейс на английском
  • Чем Cursor отличается от ChatGPT? — ChatGPT — чат. Cursor — рабочий инструмент, видит файлы, редактирует код
  • Cursor не запускается на Windows? — Запусти от администратора, отключи антивирус, обнови Windows
  • Какую модель выбрать? — Для старта используй Claude Sonnet — баланс скорости и качества
