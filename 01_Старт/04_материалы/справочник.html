Cursor vs ChatGPT и Windows

  • Чем отличается от ChatGPT? Cursor видит файлы проекта, создаёт и правит код. ChatGPT — только текст в окне.
  • Windows не запускается? Запуск от имени администратора; временно отключи антивирус; проверь обновления Windows.

  • Файл не виден? Файлы с точкой видны в Explorer. Проверь, что файл в корне открытой папки.

Что такое модель

Cursor — умный блокнот. Думает модель на сервере: запрос → сервер → ответ → Cursor применяет изменения.

Токены и контекст

  • Токен ≈ одно слово. Запрос 500 слов + файл 5000 слов = ~5500 токенов на вход; ответ 1000 слов = ~1000 на выход.
  • Контекстное окно — «память» модели. Когда заполняется — модель сжимает историю, детали теряются.
  • Одна задача = один чат. Результаты в файл. Промпты короче.

Первые проекты для новичков

Проект 1: HTML-отчёт по папке загрузок

  • 1 File → Open Folder → папка загрузок
  • 2 Чат справа → режим Plan
  • 3 Напиши промпт (ниже) → одобри план → Cursor создаст файл
  • 4 Открой HTML в браузере
Слушай, у меня в папке загрузок полный бардак. Создай HTML-страничку с отчётом, что тут лежит. Категоризируй всё. Никаких питонов и скриптов — просто HTML и JavaScript.

Проект 1: Усложняй

  • «Добавь блок с топ-5 самых крупных файлов»
  • «Создай папку Вебинар и перенеси туда эти файлы» (режим Agent)
  • Можно вставить скриншот с обведёнными файлами

Важно: фраза «Никаких питонов — просто HTML и JavaScript» ключевая. Без неё Cursor может предложить Python-скрипт.

Когда не работает и золотое правило

  • Ошибка или пустая страница → скриншот в чат → «не работает»
  • Cursor долго думает → останови → переформулируй проще
  • Ходит по кругу → удали чат, создай новый, начни заново

«Все разговоры про секретные промпты — ерунда. Описывайте, что хотите получить. Не устраивает — объясняйте, что не так. Тупит — говорите ему об этом.» Единственное важное — внятно сформулировать, что нужно.

Шаблон .cursorrules

Разделы 1–2: Роль и Обо мне

## 1. РОЛЬ Ты — мой персональный бизнес-ассистент. Цель — решать задачи быстро и эффективно. ## 2. ОБО МНЕ - **Роль:** [Собственник / Ассистент / Маркетолог / ...] - **Имя:** [Твоё имя] - **Стиль общения:** Прямой, конкретный, без воды. - **Приоритет:** Результат важнее процесса. - **Технический уровень:** [Новичок / Средний / Продвинутый]

Разделы 3–4: О бизнесе и Правила

## 3. О БИЗНЕСЕ - **Название:** [Компания] - **Ниша:** [HR / Недвижимость / Маркетинг / ...] - **ЦА:** [Кто клиенты] - **Цель сейчас:** [Автоматизация / Рост продаж / Контент] - **Инструменты:** [Telegram, Google Sheets, CRM, ...] ## 4. ПРАВИЛА РАБОТЫ Ответы: СУТЬ → ДЕТАЛИ → ВАРИАНТЫ. Всегда на русском. Файлы: не удаляй без подтверждения. Код — простой, с комментариями. Планирование: задачи > 3 шагов — сначала план, согласуй.

Раздел 5 и как применить

## 5. БЫСТРЫЕ КОМАНДЫ - /суть — кратко перескажи - /план — составь план - /исправь — найди и исправь ошибку - /объясни — объясни простым языком - /идеи — предложи 5 идей

Создай файл .cursorrules в корне проекта, вставь содержимое. Или перетащи шаблон в чат и напиши: «Примени эти правила к проекту».
