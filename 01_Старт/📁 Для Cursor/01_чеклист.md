# ✅ Чеклист: Всё настроил правильно?

> Пройди по списку после установки. Все пункты должны быть ✅

---

## Установка

- [ ] Cursor скачан с [cursor.com](https://cursor.com)
- [ ] Cursor установлен и запускается без ошибок
- [ ] Аккаунт создан (Google / GitHub / Email)
- [ ] Подписка активна (Free или Pro)

## Проект

- [ ] Создана папка проекта на компьютере
- [ ] Папка открыта в Cursor (File → Open Folder)
- [ ] В левой панели видна структура папки

## .cursorrules

- [ ] Файл `.cursorrules` создан в корне проекта
- [ ] Файл называется ТОЧНО `.cursorrules` (без .txt, .md)
- [ ] Заполнены все поля (имя, бизнес, ниша, цели)
- [ ] AI правильно отвечает на «Кто я и чем занимаюсь?»

## Первый результат

- [ ] Промпт отправлен в Chat (Cmd+L)
- [ ] AI сгенерировал файл
- [ ] Нажата кнопка Apply
- [ ] Файл открывается и работает

## Горячие клавиши (запомнил?)

- [ ] `Cmd+L` / `Ctrl+L` — AI Chat
- [ ] `Cmd+K` / `Ctrl+K` — быстрая команда
- [ ] `Cmd+S` / `Ctrl+S` — сохранить
- [ ] `Cmd+N` / `Ctrl+N` — новый файл
