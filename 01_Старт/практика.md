# Практика — Модуль 01. Старт

## Задание 1. Установка и вход (повтори по инструкции)

Установи Cursor и создай аккаунт. Проверь:

- Cursor запускается без ошибок
- Ты залогинен (видишь свой email в Settings → Account)
- Тема выбрана, интерфейс на английском

Результат: Cursor открыт, в правом нижнем углу нет ошибок, аккаунт активен.

---

## Задание 2. Рабочее пространство (повтори по инструкции)

Создай папку проекта и файл .cursorrules:

1. Создай папку `C:\Projects\МойПервыйПроект\` (Windows) или `~/Projects/МойПервыйПроект/` (Mac)
2. Открой её в Cursor через File → Open Folder
3. Создай файл `.cursorrules` по шаблону из `материалы/шаблон_cursorrules.md`
4. Заполни все поля своими данными

Проверка — открой AI Chat (Cmd+L) и напиши:

```
Расскажи, кто я и чем занимаюсь
```

Результат: AI отвечает на основе твоего .cursorrules — называет твой бизнес, нишу, клиентов.

---

## Задание 3. Страница-визитка (сделай сам)

Открой AI Chat (Cmd+L) и напиши свой промпт для создания страницы-визитки. Не копируй из инструкции — сформулируй сам. В промпте укажи:

- Что создать (страница-визитка)
- Какой контент (название, описание, контакты)
- Какой дизайн хочешь (цвета, стиль)
- Что на ней должно быть (кнопки, секции)

После генерации:

1. Нажми Apply — файл появится в проекте
2. Открой файл в браузере (правый клик → Reveal in Finder/Explorer → двойной клик)
3. Оцени результат

Результат: Страница открывается в браузере, содержит твой контент, выглядит прилично.

---

## Задание 4. Итерация — 5 улучшений подряд (сделай сам)

Улучши страницу из задания 3 ровно пять раз. Каждый раз пиши новый промпт в чат.

Идеи для улучшений:
- Поменять цвета или шрифты
- Добавить секцию с услугами или преимуществами
- Добавить отзывы клиентов
- Сделать кнопку с анимацией при наведении
- Добавить футер с соцсетями
- Сделать адаптивной для телефона

После каждого улучшения обновляй страницу в браузере и смотри что изменилось.

Результат: Страница стала заметно лучше за 5 итераций. Ты видишь, что каждый промпт меняет результат, и чем точнее ты описываешь — тем лучше получается.

## Задание 5. Вторая страница + навигация (творческое)

Создай вторую страницу для своего бизнеса. Например:
- `about.html` — о компании
- `services.html` — услуги и цены
- `portfolio.html` — примеры работ

Потом попроси AI:

```
Добавь навигационное меню на обе страницы, чтобы можно было переходить между ними. Меню должно быть одинаковое на всех страницах.
```

Проверь что переходы работают — кликни по ссылкам в меню.

Результат: У тебя мини-сайт из двух страниц с рабочей навигацией. Ты понимаешь, что AI может работать с несколькими файлами и связывать их между собой.
