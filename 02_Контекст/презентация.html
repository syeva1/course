Символ @ — твой главный инструмент

Один символ меняет всё

  • @ говорит AI: «Посмотри сюда!»
  • Набери @ в чате — появится список источников
  • Каждый @-символ подключает свой тип контекста
  • Привыкай начинать каждый запрос с @
@ → выпадающий список →
@File, @Codebase, @Web, @Doc, @Git...

@Codebase

Весь проект как контекст

  • AI анализирует все файлы проекта
  • Видит связи между компонентами
  • Идеально для: рефакторинг, обзор, глобальные вопросы
  • ⚠️ Не используй для мелких задач — медленнее и дороже
@Codebase Опиши структуру проекта
и предложи улучшения

Структура проекта = контекст

AI читает имена файлов и папок

❌ Плохо

project/
├── file1.html
├── file2.css
├── stuff.js
└── old_copy_v2.html

✅ Хорошо

project/
├── index.html
├── css/styles.css
├── js/main.js
└── .cursorrules

Понятные имена = лучший контекст = лучший результат

Частые ошибки с контекстом

Не повторяй за другими

  • ⚠️ Нет контекста вообще — AI гадает, результат абстрактный
  • ⚠️ @Codebase для мелочей — медленно и дорого
  • ⚠️ Контекст в каждом промпте — настрой .cursorrules вместо этого
  • ⚠️ Хаос в файлах — AI читает имена, хаос = плохой контекст
  • ⚠️ Игнорируешь Notepad — удобный способ без повторений
