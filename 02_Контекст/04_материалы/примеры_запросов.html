@Codebase Напиши текст для главной index.html. Учти описание из .cursorrules. Тон — дружелюбный эксперт.

Исправление и актуальное

Ошибка: «Кнопка не работает» — AI не знает какую. С контекстом: @File index.html Кнопка «Оставить заявку» не открывает форму. Исправь.

Актуальное: «Какие тренды в HR?» — устаревшее. С контекстом: @Web Какие тренды в подборе ассистентов в России в 2025? 5 ключевых с описанием.

Рефакторинг и Code Review

Рефакторинг: @Codebase Проведи рефакторинг: найди дублирующийся CSS, вынеси общие стили, удали неиспользуемые классы.

Code Review: @Git Посмотри последний коммит. Проверь на ошибки, уязвимости и стиль. Предложи улучшения.

Шаблон идеального запроса

@[источник] [конкретная задача]. [Дополнительные условия]. [Формат результата].

Пример: @File styles.css Сделай кнопку «Связаться» заметнее. Цвет #E63229, скруглённые углы. Покажи только изменённый CSS.
