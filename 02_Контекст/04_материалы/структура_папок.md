# Структура папок для Cursor-проекта

> Создай эту структуру в Cursor. AI будет понимать контекст автоматически.

```
МОЙ_ПРОЕКТ/
├── .cursorrules          ← главные правила (скопируй из модуля 01)
├── .cursor/
│   └── rules/
│       ├── код.mdc       ← правила для кода
│       ├── тексты.mdc    ← правила для контента
│       └── данные.mdc    ← правила для работы с данными
├── О_СЕБЕ.md             ← кто ты, чем занимаешься
├── README.md             ← описание проекта
├── tasks/
│   ├── todo.md           ← текущие задачи
│   └── lessons.md        ← уроки и ошибки
├── docs/                 ← документация, заметки
├── src/                  ← исходный код
├── data/                 ← данные, таблицы
└── output/               ← результаты, отчёты
```

## Зачем это нужно
- `.cursorrules` — AI читает при каждом запросе
- `.cursor/rules/*.mdc` — контекстные правила (применяются к конкретным файлам)
- `О_СЕБЕ.md` — перетащи в чат, чтобы AI узнал о тебе
- `tasks/` — AI видит твои задачи и может помогать планировать
