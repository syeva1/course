# 💬 Примеры запросов: с контекстом и без

> Почувствуй разницу. Один и тот же вопрос — разный результат.

---

## Пример 1. Текст для сайта

**❌ Без контекста:**
```
Напиши текст для главной страницы
```
*Результат: Абстрактный шаблонный текст, который подойдёт любой компании.*

**✅ С контекстом:**
```
@Codebase Напиши текст для главной страницы index.html.
Учти описание проекта из .cursorrules.
Тон — как у дружелюбного эксперта.
```
*Результат: Текст под твой бизнес, с правильным тоном и акцентами.*

---

## Пример 2. Исправление ошибки

**❌ Без контекста:**
```
Кнопка не работает, почини
```
*Результат: AI не знает, какая кнопка, в каком файле, что она должна делать.*

**✅ С контекстом:**
```
@File index.html Кнопка «Оставить заявку» не открывает форму.
При клике ничего не происходит. Исправь.
```
*Результат: AI видит файл, находит кнопку, исправляет обработчик.*

---

## Пример 3. Актуальная информация

**❌ Без контекста:**
```
Какие тренды в HR?
```
*Результат: Информация из обучающих данных (может быть устаревшей).*

**✅ С контекстом:**
```
@Web Какие тренды в подборе бизнес-ассистентов в России в 2025 году?
Приведи 5 ключевых трендов с кратким описанием.
```
*Результат: Свежая информация из интернета, структурированная под твой запрос.*

---

## Пример 4. Работа с документацией

**❌ Без контекста:**
```
Как сделать анимацию?
```
*Результат: Общие советы по CSS-анимации.*

**✅ С контекстом:**
```
@Doc Framer Motion Как сделать анимацию появления карточек при скролле?
Нужен эффект fade-in снизу вверх, задержка 0.2с между карточками.
```
*Результат: Готовый код с использованием конкретной библиотеки.*

---

## Пример 5. Рефакторинг

**❌ Без контекста:**
```
Оптимизируй код
```
*Результат: AI не знает, какой код, где он, что оптимизировать.*

**✅ С контекстом:**
```
@Codebase Проведи рефакторинг:
1. Найди дублирующийся CSS
2. Вынеси общие стили в отдельный файл
3. Удали неиспользуемые классы
```
*Результат: Конкретные предложения по конкретным файлам.*

---

## Пример 6. Code Review

**❌ Без контекста:**
```
Проверь мой код
```

**✅ С контекстом:**
```
@Git Посмотри последний коммит.
Проверь на ошибки, уязвимости и стиль кода.
Предложи улучшения.
```
*Результат: Детальный обзор конкретных изменений.*

---

## 🔑 Шаблон идеального запроса

```
@[источник] [конкретная задача].
[Дополнительные условия].
[Формат результата].
```

**Пример:**
```
@File styles.css Сделай кнопку «Связаться» более заметной.
Цвет — фирменный красный (#E63229), скруглённые углы.
Покажи только изменённый CSS, без остального кода.
```
