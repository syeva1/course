---

## Как создать за 1 минуту

1. Скопируй структуру выше
2. Открой Cursor → **Cmd+I** (Composer)
3. Вставь и допиши:

```
Создай эту структуру папок в текущей директории.
Для каждой папки создай README.md с описанием:
- Что здесь хранится
- Когда обращаться к этой папке
- Примеры файлов
```

4. Нажми **Accept** → готово!

---

## Зачем README в каждой папке

README — это **навигация для AI**. Когда Cursor видит README, он понимает:
- Что лежит в этой папке
- Зачем она нужна
- Какие файлы ожидать

Без README → AI не понимает контекст папки.
С README → AI знает, где что искать.

---

## Папка «Распределение» — временная

Когда не знаешь, куда положить файл → кидай в **Распределение/**

Раз в неделю (или попроси AI):
```
Посмотри файлы в Распределение/ и разложи их по правильным папкам.
```

## Персонализация

Не нужны все папки? Убери лишние. Нужны другие? Добавь.

Для персональной структуры используй **режим Plan**:

```
Задай мне вопросы о моей жизни, работе и интересах.
На основе ответов создай идеальную структуру папок для меня.
```

AI спросит о тебе → предложит структуру → ты одобришь → он создаст.
