## Как использовать

1. Скопируй шаблон выше
2. Создай файл `О_СЕБЕ.md` в корне проекта
3. Заполни все поля (пропусти ненужные)
4. Cursor будет автоматически видеть этот файл

## Бонус: пусть AI поможет заполнить

Вставь пустой шаблон в Cursor (Cmd+L) и скажи:

```
Задай мне вопросы по каждому разделу этого шаблона.
На основе моих ответов заполни файл.
```

AI будет спрашивать по порядку — ты отвечаешь — он заполняет. Проще, чем писать самому.
