# Практика — Модуль 02. Контекст

## Задание 1. Контекст vs без контекста (повтори по инструкции)

Открой AI Chat (Cmd+L / Ctrl+L) и задай один и тот же вопрос двумя способами:

Без контекста:
```
Напиши текст для секции "О нас" на сайте
```

С контекстом:
```
@File .cursorrules @Notepad Напиши текст для секции "О нас" на сайте
```

Сохрани оба ответа. Сравни.

Результат: Ты видишь, что второй ответ конкретнее — он учитывает твой бизнес, аудиторию, тон.

---

## Задание 2. Настрой Notepad (повтори по инструкции)

1. Открой Notepad (Cmd+Shift+P → Notepad)
2. Создай заметку и заполни:
   - Что за проект (1-2 предложения)
   - Для кого (целевая аудитория)
   - Какие технологии
   - Какой тон общения
   - Что НЕ использовать (ограничения)
3. В AI Chat напиши:
```
@Notepad Предложи 5 улучшений для моего проекта
```

Результат: Все 5 предложений относятся к твоему конкретному проекту, а не абстрактные советы.

## Задание 3. Файл О_СЕБЕ.md (сделай сам)

Создай файл О_СЕБЕ.md в корне проекта. Не копируй шаблон слово в слово — заполни своими данными. Включи минимум:

- Имя и чем занимаешься
- 3 сильные стороны
- Как общаешься (стиль)
- 2-3 текущие цели
- С кем работаешь

Проверка:
```
@File О_СЕБЕ.md Посоветуй мне 3 навыка, которые стоит развить в ближайший месяц
```

Результат: Советы учитывают твою реальную ситуацию. AI знает кто ты и подстраивается.

---

## Задание 4. Цепочка @-символов (сделай сам)

Выполни 4 запроса, каждый с разным @-символом:

1. `@File index.html` — попроси добавить новую секцию на страницу
2. `@Web` — найди актуальный тренд в своей нише и попроси AI применить его к проекту
3. `@Codebase` — попроси AI найти проблемы или несоответствия в проекте
4. `@Notepad` — задай вопрос, где нужна информация из твоих заметок

Результат: Ты понимаешь когда какой @-символ использовать и чем они отличаются.

## Задание 5. Система папок "Моя жизнь" (творческое)

Создай отдельную папку и открой её в Cursor как новый проект. Внутри:

1. Создай минимум 5 подпапок (например: Работа, Обучение, Здоровье, Финансы, Цели)
2. В каждой подпапке создай хотя бы один файл с реальной информацией
3. Создай папку «Распределение» и положи туда 3-5 заметок (что угодно — мысли, задачи, ссылки)
4. Создай README.md с описанием структуры
5. Создай .cursorrules с правилом: «Ты — мой личный секретарь. Помогаешь организовывать жизнь.»
6. Попроси AI:
```
Разбери все файлы из папки Распределение и предложи, в какие разделы их положить
```

Результат: У тебя рабочая система, где AI помогает сортировать и организовывать информацию.
