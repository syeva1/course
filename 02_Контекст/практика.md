# 🏋️ Практика: Модуль 02

> **Задание:** Научиться давать AI правильный контекст

---

## Задание 1. Сравни запросы с контекстом и без

**Что сделать:**

1. Открой AI Chat (`Cmd+L` / `Ctrl+L`)
2. Задай вопрос БЕЗ контекста:
   ```
   Напиши текст для главной страницы
   ```
3. Посмотри на результат — скорее всего, он будет абстрактным и бесполезным
4. Теперь задай тот же вопрос С контекстом:
   ```
   @Codebase Напиши текст для главной страницы index.html, учитывая информацию из .cursorrules
   ```
5. Сравни результаты

**Критерий:** Ты видишь разницу в качестве и понимаешь, почему контекст важен.

---

## Задание 2. Настрой Notepad

**Что сделать:**

1. Открой Notepad (`Cmd+Shift+P` → «Notepad»)
2. Создай заметку «Контекст проекта» и заполни:
   - Краткое описание проекта (2-3 предложения)
   - Целевая аудитория
   - Стек технологий
   - Тон общения (формальный / дружелюбный / экспертный)
   - 3 ключевых ограничения (например: «Не используй jQuery», «Все тексты на русском»)
3. Задай в AI Chat:
   ```
   @Notepad Предложи 5 улучшений для моего проекта, учитывая контекст
   ```

**Критерий:** AI даёт релевантные предложения, учитывающие специфику именно твоего проекта.

---

## Задание 3. Цепочка @-символов

**Что сделать:**

1. Используй `@File` для конкретного файла:
   ```
   @File index.html Добавь секцию «Отзывы клиентов» после секции «Преимущества»
   ```
2. Используй `@Web` для актуальной информации:
   ```
   @Web Какие лучшие практики дизайна лендингов в 2025 году?
   ```
3. Используй `@Doc` для документации (если подключил):
   ```
   @Doc Tailwind CSS Как сделать адаптивную сетку из 3 колонок?
   ```

**Критерий:** Ты уверенно используешь разные @-символы и понимаешь, когда какой нужен.

## 🌟 Бонус: Создай идеальный контекст

Открой свой проект из Модуля 01 и сделай его «контекстно-идеальным»:

1. Проверь и дополни `.cursorrules`
2. Настрой Notepad с ключевой информацией
3. Приведи в порядок имена файлов и папок
4. Задай AI один короткий запрос (1 строка) и получи качественный результат

**Критерий:** Короткий запрос = отличный результат. Контекст делает свою работу.
