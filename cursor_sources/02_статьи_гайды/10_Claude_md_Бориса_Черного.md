# Claude.md Бориса Черного (создатель Claude Code)

Источник: канал @misha_davai_po_novoi

---

Борис Черный (Boris Cherny) — один из создателей Claude Code в Anthropic. Его Claude.md демонстрирует продвинутые паттерны:

## Ключевые идеи:

### Оркестрация
- Главный агент управляет субагентами
- Каждый субагент — узкоспециализированная задача
- Параллельное выполнение где возможно

### Субагенты
- Отдельные контексты для каждой подзадачи
- Минимальный контекст = точнее результат
- Передача только нужной информации между агентами

### Цикл самосовершенствования
- Агент анализирует свои ошибки
- Обновляет правила на основе опыта
- Документирует паттерны ошибок и решения

### Управление задачами
- Декомпозиция на атомарные шаги
- Каждый шаг верифицируемый
- Чеклист завершения для каждой задачи

## Структура:

```
CLAUDE.md
├── Роль и контекст
├── Правила поведения
├── Инструменты и как их использовать
├── Паттерны ошибок (что НЕ делать)
├── Примеры хороших решений
└── Процесс самоулучшения
```
