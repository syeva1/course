# Что написано в моём Claude.md — Миша Попов

Источник: канал @misha_davai_po_novoi

---

Миша показал свою систему контент-фабрики, работающей через Claude Code / Cursor с 6 AI-агентами:

## Агенты:

1. **Скаут** — ищет тренды и темы для контента
2. **Аналитик** — исследует найденную тему, собирает факты
3. **Писатель** — создаёт черновик поста/статьи
4. **Редактор** — правит текст, улучшает подачу
5. **Дизайнер** — создаёт визуалы / промпты для генерации картинок
6. **Публишер** — форматирует и готовит к публикации

## Принципы:

- Каждый агент — отдельная задача в Claude.md
- Агенты работают последовательно (конвейер)
- Claude.md содержит инструкции для каждого агента
- Результат одного агента — вход для следующего
- Миша контролирует и корректирует на каждом этапе
