# Модуль 04 — Промптинг

> Говори так, чтобы AI понял

## Чему научитесь

- Формулировать чёткие промпты, которые дают нужный результат
- Использовать структуру: роль + контекст + задача + формат
- Применять XML-теги, few-shot и chain-of-thought
- Итерировать промпты вместо того, чтобы бросать после первого ответа
- Создавать базы знаний вместо шаблонов

## Как пользоваться

1. Откройте `презентация.html` в браузере
2. Изучите слайды (стрелки или кнопки навигации)
3. Выполните задания из `практика.md`
4. Держите `шпаргалка.md` под рукой
