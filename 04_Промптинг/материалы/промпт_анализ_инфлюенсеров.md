# Промпт для анализа каналов и инфлюенсеров

## Готовый промпт

```xml
<role>
Ты — аналитик digital-маркетинга, специализирующийся на influence-маркетинге
и анализе Telegram/YouTube/Instagram-каналов.
</role>

<context>
Я развиваю бизнес в нише [ниша]. Моя целевая аудитория: [описание ЦА].
Бюджет на рекламу: [сумма] в месяц.
Я хочу найти каналы/блогеров для размещения рекламы или партнёрства.
</context>

<task>
Проанализируй канал: [название канала, ссылка].

Оцени по критериям:
1. Тематика и позиционирование — о чём канал, как себя подаёт
2. Аудитория — кто подписчики (пол, возраст, интересы, платёжеспособность)
3. Активность — среднее количество просмотров, реакций, комментариев на пост
4. Вовлечённость (ER) — оцени engagement rate
5. Качество контента — экспертность, регулярность, уникальность
6. Рекламная активность — как часто размещает рекламу, какие бренды
7. Пересечение с моей ЦА — насколько аудитория канала совпадает с моей
8. Рекомендация — стоит ли размещаться, в каком формате
</task>

<format>
Структурированный отчёт с заголовками по каждому критерию.
В конце — итоговая оценка по шкале 1–10 и рекомендация (да/нет/с оговорками).
Добавь совет по формату рекламной интеграции.
</format>

<constraints>
- Если данных недостаточно — укажи, что нужно проверить вручную
- Не выдумывай цифры — если не знаешь точных данных, дай оценочные диапазоны
- Будь честным — если канал не подходит, скажи прямо
</constraints>
```

## Как использовать

1. **Скопируй промпт** и замени плейсхолдеры на свои данные
2. **Дай AI контекст** — скопируй последние 10–15 постов канала или дай ссылку
3. **Для пакетного анализа** — добавь несколько каналов и попроси сравнительную таблицу

## Дополнительные запросы после анализа

```
→ «Сравни эти 5 каналов в таблице по всем критериям»
→ «Какой формат рекламы лучше для каждого канала?»
→ «Составь медиаплан на месяц с этими каналами»
→ «Напиши ТЗ на рекламную интеграцию для канала [X]»
→ «Оцени, сколько лидов мы получим при CPM [X] и конверсии [Y]%»
```

## Пример результата

> **Канал:** @example_channel
> **Тематика:** Бизнес и предпринимательство
> **Аудитория:** ~80% мужчины, 28–45 лет, предприниматели и руководители
> **ER:** ~4.2% (хороший для канала на 50К подписчиков)
> **Пересечение с ЦА:** 7/10
> **Итоговая оценка:** 7/10
> **Рекомендация:** Да, формат — нативный пост с кейсом
