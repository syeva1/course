# 20 готовых промптов для бизнеса

## Контент и маркетинг

### 1. Пост для Telegram
```
Ты — SMM-специалист с опытом в B2B.

Напиши пост для Telegram-канала компании [название].
Тема: [тема].
Аудитория: предприниматели, 30–45 лет.
Длина: 150–200 слов.
Тон: экспертный, но дружелюбный.
Добавь 1–2 эмодзи, заверши призывом к действию.
```

### 2. Серия сторис
```
Ты — контент-маркетолог.

Создай серию из 5 сторис для Instagram.
Тема: [тема].
Каждая сторис: 1 короткий тезис + описание визуала.
Формат: нумерованный список.
Финальная сторис — призыв к действию.
```

### 3. Email-рассылка
```
Ты — email-маркетолог.

Напиши письмо для рассылки.
Цель: [вернуть клиента / анонсировать продукт / поделиться пользой].
Аудитория: [описание].
Структура: тема письма + прехедер + тело (до 150 слов) + CTA-кнопка.
Тон: [тёплый / деловой / срочный].
```

### 4. Заголовки для лендинга
```
Ты — копирайтер, специалист по конверсии.

Придумай 10 вариантов заголовка для лендинга.
Продукт: [описание].
Целевая аудитория: [кто].
Главная выгода: [что получает клиент].
Каждый заголовок — до 10 слов.
```

### 5. Контент-план на неделю
```
Ты — контент-стратег.

Составь контент-план на 7 дней для [платформа].
Ниша: [ниша].
Цель: [рост подписчиков / продажи / вовлечение].
Формат: таблица — день, тема, формат поста, ключевая идея.
Чередуй: полезный / развлекательный / продающий контент.
```

## Продажи и клиенты

### 6. Скрипт звонка
```
Ты — тренер по продажам.

Напиши скрипт холодного звонка для [продукт/услуга].
Целевой клиент: [описание].
Структура: приветствие → квалификация → презентация выгод → работа с возражением «дорого» → закрытие на следующий шаг.
Длительность разговора: 3–5 минут.
```

### 7. Коммерческое предложение
```
Ты — бизнес-консультант.

Напиши структуру коммерческого предложения для [услуга].
Клиент: [описание компании клиента].
Включи: проблема клиента, наше решение, этапы работы, результаты (с цифрами), стоимость, гарантии.
Объём: 1 страница.
```

### 8. Ответ на возражение
```
Ты — эксперт по продажам.

Клиент говорит: «[возражение]».
Контекст: мы продаём [продукт], стоимость [цена].

Дай 3 варианта ответа:
1. Рациональный (цифры и факты)
2. Эмоциональный (история, метафора)
3. Вопросом (переключить фокус)
```

### 9. Описание услуги для сайта
```
Ты — UX-копирайтер.

Напиши описание услуги для сайта.
Услуга: [название и суть].
Структура: заголовок + подзаголовок + 3 блока «боль → решение» + CTA.
Объём: 200–300 слов.
Тон: уверенный, конкретный, без воды.
```

### 10. Follow-up письмо
```
Ты — менеджер по продажам.

Напиши follow-up письмо клиенту, который [был на встрече / оставил заявку / замолчал].
Тон: вежливый, не навязчивый.
Цель: [назначить встречу / получить ответ / напомнить о себе].
Длина: 5–7 предложений.
```

## HR и управление

### 11. Текст вакансии
```
Ты — HR-маркетолог.

Напиши текст вакансии для [должность].
Компания: [описание].
Формат: заголовок + кого ищём (3 предложения) + задачи (5 пунктов) + требования (5 пунктов) + мы предлагаем (5 пунктов) + как откликнуться.
Тон: живой, привлекательный — не корпоративный шаблон.
```

### 12. Вопросы для собеседования
```
Ты — HR-директор с 10-летним опытом.

Составь 10 вопросов для собеседования на позицию [должность].
Разбей на категории: хард-скиллы (3), софт-скиллы (3), мотивация (2), кейсы (2).
К каждому вопросу добавь: на что обращать внимание в ответе.
```

### 13. Обратная связь сотруднику
```
Ты — руководитель команды.

Напиши текст обратной связи для сотрудника.
Ситуация: [что произошло].
Принцип: похвала + зона роста + конкретный план.
Тон: поддерживающий, но честный.
Длина: 100–150 слов.
```

### 14. Описание процесса (SOP)
```
Ты — бизнес-аналитик.

Опиши процесс [название процесса] в формате пошаговой инструкции.
Для кого: [новый сотрудник / ассистент / подрядчик].
Формат: нумерованные шаги, каждый — 1–2 предложения.
Добавь: частые ошибки и как их избежать.
```

## Аналитика и стратегия

### 15. Анализ конкурента
```
Ты — маркетинговый аналитик.

Проанализируй конкурента: [название или ссылка].
Структура анализа:
1. Позиционирование (в 2 предложениях)
2. Сильные стороны (3 пункта)
3. Слабые стороны (3 пункта)
4. Что мы можем у них взять
5. Чем мы лучше
```

### 16. SWOT-анализ
```
Ты — бизнес-стратег.

Проведи SWOT-анализ для [компания/продукт].
Контекст: [описание ситуации].
Формат: таблица 2×2 — Strengths, Weaknesses, Opportunities, Threats.
По 4–5 пунктов в каждой ячейке. Кратко, конкретно.
```

### 17. Резюме встречи
```
Ты — бизнес-ассистент.

Вот заметки со встречи:
[вставь заметки]

Оформи в структуру:
1. Участники
2. Ключевые решения (список)
3. Задачи (кто — что — дедлайн)
4. Открытые вопросы
```

### 18. Бриф для подрядчика
```
Ты — проджект-менеджер.

Составь бриф для [дизайнера / разработчика / копирайтера].
Проект: [описание].
Включи: цель, аудитория, референсы, требования, дедлайн, бюджет, критерии приёмки.
Формат: структурированный документ с заголовками.
```

### 19. Декомпозиция цели
```
Ты — бизнес-коуч.

Цель: [формулировка цели].
Срок: [период].

Разбей на:
1. Ключевые этапы (3–5)
2. Для каждого этапа: конкретные задачи, метрики, ресурсы
3. Первый шаг, который можно сделать сегодня
```

### 20. Идеи для продукта
```
Ты — продуктовый менеджер.

Компания: [описание].
Текущие продукты: [список].
Аудитория: [кто].

Предложи 5 идей для нового продукта/услуги.
Для каждой идеи: название, суть в 2 предложениях, 
потенциальная выручка (низкая/средняя/высокая), 
сложность реализации (1–5).
```
