# Модуль 03. Инструменты Cursor + GitHub

Осваиваем все ключевые инструменты Cursor — от автодополнения по Tab до мультифайлового Composer. Подключаем GitHub для хранения проектов.

## Чему научишься

- Использовать Tab, Cmd+K, Cmd+L, Cmd+I — и понимать когда какой
- Работать в Agent Mode и Plan Mode
- Создать аккаунт GitHub и подключить его к Cursor
- Делать commit и push из Cursor
- Устанавливать расширения

## Файлы модуля

- **презентация.html** — слайды (открой в браузере)
- **инструкция.md** — пошаговый гайд по каждому инструменту
- **практика.md** — 5 заданий от простого к сложному
- **материалы/** — команды терминала, горячие клавиши

## Порядок прохождения

1. Посмотри презентацию
2. Пройди инструкцию, пробуя каждый инструмент на своём проекте
3. Выполни все 5 заданий из практики

## Время

~2 часа на первое прохождение.

## Материалы

- `команды_терминала.md` — базовые команды для работы в терминале Cursor
- `как_работать_с_агентами_cursor.md` — официальные практики работы с агентами
- `worktree_параллельные_агенты.md` — параллельная работа нескольких агентов
- `LSP_вместо_grep.md` — быстрая навигация по проекту
