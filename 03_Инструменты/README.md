# Модуль 03 — Инструменты Cursor + GitHub

## О чём этот модуль

Вы освоите **все ключевые инструменты Cursor** — от автодополнения по Tab до мультифайлового Composer. Научитесь работать с **GitHub**: создадите аккаунт, подключите его к Cursor и сделаете первый push. Разберётесь в расширениях, плагинах и LSP.

## Что внутри

| Файл | Описание |
|------|----------|
| `презентация.html` | Слайды модуля (17 шт.) |
| `инструкция.md` | Пошаговый гайд по каждому инструменту |
| `практика.md` | 3 задания + чеклист |
| `шпаргалка.md` | Горячие клавиши + Git-команды |
| `материалы/список_расширений.md` | Топ-15 расширений с описанием |
| `материалы/github_шпаргалка.md` | Git-команды для новичка |

## Результат модуля

- Уверенно используете Tab, Cmd+K, Cmd+L, Cmd+I
- Понимаете, когда какой инструмент применять
- Имеете аккаунт на GitHub и умеете делать push/pull
- Установили нужные расширения
- Знаете, что такое LSP, MCP и плагины
