## Задание 1. Четыре инструмента на одном файле (повтори по инструкции)

Открой index.html из Модуля 01 и выполни:

1. **Tab**: Начни писать новый `<div>` — прими подсказку через Tab
2. **Cmd+K**: Выдели заголовок → Cmd+K → «Сделай заголовок крупнее и добавь тень»
3. **Cmd+L**: Открой Chat → «Какие секции обычно есть на лендинге HR-агентства?»
4. **Cmd+I**: Открой Composer → «Добавь секцию FAQ с 5 вопросами и ответами про подбор ассистентов»

Результат: Ты попробовал все 4 инструмента. Видишь чем они отличаются: Tab для мелочей, Cmd+K для точечных правок, Cmd+L для вопросов, Cmd+I для больших изменений.

## Задание 2. Plan Mode (повтори по инструкции)

1. Открой Composer (Cmd+I)
2. Переключи в режим Plan (выпадающий список внизу)
3. Напиши:
```
Мне нужна страница "Услуги" для моего бизнеса. 
Секции: заголовок, 4 карточки с услугами (название, описание, цена), 
кнопка "Заказать" под каждой карточкой, футер.
Стиль как у index.html.
Составь план.
```
4. Прочитай план. Если что-то не нравится — напиши комментарий.
5. Когда план устраивает — одобри выполнение.

Результат: Ты создал новую страницу через Plan Mode. Понимаешь разницу между Agent и Plan — Plan сначала показывает план, потом делает.

## Задание 3. GitHub — публикация проекта (сделай сам)

Если ещё не сделал:
1. Создай аккаунт на github.com
2. В Cursor: Source Control → Initialize Repository
3. Войди в GitHub через Cursor
4. Сделай commit с сообщением «Мой первый проект — модули 01-03»
5. Нажми Publish Branch → выбери Public

Если уже сделал — добавь страницу из задания 2:
1. В Source Control нажми + напротив новых файлов
2. Напиши сообщение: «Добавил страницу услуг»
3. Commit → Sync Changes

Проверка: открой github.com/твой-username — проект должен быть виден.

Результат: Твой проект на GitHub. Ты умеешь commit и push.

## Задание 4. «AI напишет промпт за тебя» (сделай сам)

1. Открой Chat (Cmd+L)
2. Напиши: «Я хочу добавить на сайт интерактивный калькулятор стоимости услуг. Напиши подробный промпт для Composer, который создаст это максимально качественно.»
3. AI выдаст детальный промпт
4. Скопируй его
5. Открой Composer (Cmd+I), вставь промпт, выполни

Сравни результат с тем, что получилось бы если бы ты просто написал «Сделай калькулятор».

Результат: Ты видишь что промпт от AI даёт лучший результат, чем твой короткий запрос.

## Задание 5. Полный цикл — новая страница с нуля (творческое)

Создай новую страницу для своего проекта — любую на выбор:
- Портфолио / примеры работ
- Блог с 3 статьями
- Страница команды с фото-заглушками
- Форма заявки с валидацией

Требования:
1. Сначала составь план через Plan Mode
2. Выполни план пошагово
3. Используй Cmd+K для точечных правок
4. Используй Chat для вопросов если что-то непонятно
5. Сделай commit с осмысленным сообщением
6. Push на GitHub

Результат: Полный цикл: план → создание → правки → сохранение → публикация. Ты освоил рабочий процесс.
