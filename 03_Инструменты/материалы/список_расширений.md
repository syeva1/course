# Топ-15 расширений для Cursor

## Обязательные

### 1. Russian Language Pack
Русский интерфейс для Cursor/VS Code. Меню, настройки, подсказки — всё на русском.

### 2. Prettier — Code Formatter
Автоформатирование кода при сохранении. HTML, CSS, JS, JSON — всё красиво и единообразно.

### 3. GitLens
Расширенная работа с Git. Показывает, кто и когда менял каждую строку. История файла, сравнение версий.

## Рекомендуемые

### 4. Live Server
Запускает локальный сервер с автообновлением. Изменили HTML — браузер обновился сам.

### 5. Auto Rename Tag
Переименовали открывающий тег — закрывающий обновится автоматически.

### 6. Path Intellisense
Автодополнение путей к файлам. Пишете `src="` — видите список файлов.

### 7. Color Highlight
Подсвечивает цвета прямо в коде. `#E63229` будет показан красным фоном.

### 8. Bracket Pair Color DeLighter
Раскрашивает парные скобки в разные цвета. Легче видеть вложенность.

### 9. Error Lens
Показывает ошибки и предупреждения прямо в строке кода, а не только в панели.

### 10. indent-rainbow
Раскрашивает уровни отступов разными цветами. Легче видеть структуру.

## Для продвинутых

### 11. ESLint
Проверяет JavaScript/TypeScript код на ошибки и стиль. Стандарт индустрии.

### 12. Tailwind CSS IntelliSense
Автодополнение классов Tailwind CSS. Подсказки, превью цветов.

### 13. Docker
Управление Docker-контейнерами прямо из редактора.

### 14. REST Client
Отправка HTTP-запросов прямо из `.http` файлов. Удобнее Postman для простых задач.

### 15. Thunder Client
Полноценный HTTP-клиент с интерфейсом, как у Postman. Всё внутри редактора.

---

## Как установить

1. `Cmd+Shift+X` (Win: `Ctrl+Shift+X`)
2. Ввести название в поиск
3. Нажать **Install**
