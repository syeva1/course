# Горячие клавиши Cursor

> Распечатай или держи открытым. Самые нужные сверху.

## AI-функции
| Действие | Mac | Windows |
|----------|-----|---------|
| Открыть AI-чат (Agent) | `Cmd+Shift+I` | `Ctrl+Shift+I` |
| Inline Edit (выдели код → правь) | `Cmd+K` | `Ctrl+K` |
| Принять предложение | `Tab` | `Tab` |
| Отклонить предложение | `Esc` | `Esc` |
| Добавить файл в контекст | `@файл` в чате | `@файл` в чате |

## Навигация
| Действие | Mac | Windows |
|----------|-----|---------|
| Палитра команд | `Cmd+Shift+P` | `Ctrl+Shift+P` |
| Быстро открыть файл | `Cmd+P` | `Ctrl+P` |
| Перейти к строке | `Ctrl+G` | `Ctrl+G` |
| Поиск по проекту | `Cmd+Shift+F` | `Ctrl+Shift+F` |
| Переключить терминал | `` Ctrl+` `` | `` Ctrl+` `` |

## Редактирование
| Действие | Mac | Windows |
|----------|-----|---------|
| Множественный курсор | `Cmd+D` | `Ctrl+D` |
| Переместить строку | `Alt+↑/↓` | `Alt+↑/↓` |
| Дублировать строку | `Cmd+Shift+D` | `Ctrl+Shift+D` |
| Комментировать | `Cmd+/` | `Ctrl+/` |
| Форматировать файл | `Cmd+Shift+F` | `Ctrl+Shift+F` |
