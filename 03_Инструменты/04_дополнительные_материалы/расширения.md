# Расширения для Cursor — копируй и ставь

> Открой терминал в Cursor (Ctrl+`) и вставь команды:

## Обязательные

```bash
# Русский язык интерфейса
cursor --install-extension MS-CEINTL.vscode-language-pack-ru

# Подсветка скобок (встроена, но проверь)
# Settings → Editor: Bracket Pair Colorization → ON

# Prettier — автоформатирование кода
cursor --install-extension esbenp.prettier-vscode

# GitLens — история изменений
cursor --install-extension eamodio.gitlens
```

## Для Python

```bash
# Python
cursor --install-extension ms-python.python

# Pylance — умный автокомплит
cursor --install-extension ms-python.vscode-pylance
```

## Для веб-разработки

```bash
# Live Server — превью HTML в браузере
cursor --install-extension ritwickdey.LiveServer

# Auto Rename Tag — парное переименование тегов
cursor --install-extension formulahendry.auto-rename-tag

# CSS Peek — переход к CSS по клику
cursor --install-extension pranaygp.vscode-css-peek
```

## Полезные

```bash
# Markdown Preview Enhanced
cursor --install-extension shd101wyy.markdown-preview-enhanced

# Todo Tree — все TODO в проекте
cursor --install-extension Gruntfuggly.todo-tree

# Error Lens — ошибки прямо в коде
cursor --install-extension usernamehw.errorlens
```
