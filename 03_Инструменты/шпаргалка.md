## Git-команды (терминал)

```bash
# Инициализация
git init                          # Создать репозиторий
git clone <url>                   # Клонировать проект

# Повседневная работа
git status                        # Что изменилось
git add .                         # Добавить все файлы
git add <файл>                    # Добавить конкретный файл
git commit -m "сообщение"         # Сохранить изменения
git push                          # Отправить на GitHub
git pull                          # Получить изменения

# Просмотр
git log --oneline                 # История коммитов
git diff                          # Посмотреть изменения

# Ветки
git branch                        # Список веток
git checkout -b <имя>             # Создать новую ветку
git checkout <имя>                # Переключиться на ветку
git merge <имя>                   # Слить ветку
```

## Git через Cursor (без терминала)

1. **Source Control** — иконка ветки в боковой панели
2. **Stage** — нажмите `+` у файла
3. **Commit** — введите сообщение → `✓`
4. **Push** — нажмите `…` → Push (или Publish Branch)
5. **Pull** — нажмите `…` → Pull
