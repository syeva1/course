# Практика — Модуль 03

## Задание 1: AI-инструменты Cursor

1. Создайте файл `index.html`
2. Начните писать структуру HTML — используйте **Tab** для автодополнения
3. Выделите `<title>` и через **Cmd+K** попросите: «Сделай заголовок "Мой первый проект"»
4. Откройте **Cmd+L** и спросите: «Что делает тег meta viewport?»
5. Откройте **Cmd+I** и попросите: «Добавь секцию hero с заголовком и кнопкой»

**Результат:** HTML-файл, созданный с помощью всех четырёх инструментов.

---

## Задание 2: GitHub

1. Зарегистрируйтесь на [github.com](https://github.com) (если ещё нет аккаунта)
2. В Cursor откройте Source Control → Initialize Repository
3. Войдите в GitHub через Cursor
4. Сделайте commit с сообщением «Первый коммит»
5. Нажмите Publish Branch → выберите Public
6. Откройте github.com — убедитесь, что проект там

**Результат:** Ваш проект опубликован на GitHub.

---

## Задание 3: Расширения

1. Откройте Extensions (**Cmd+Shift+X**)
2. Установите:
   - Russian Language Pack
   - Prettier
   - GitLens
3. Перезагрузите Cursor
4. Откройте любой файл — убедитесь, что Prettier форматирует код при сохранении

**Результат:** Три расширения установлены и работают.
